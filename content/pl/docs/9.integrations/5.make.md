[Make.com](https://www.make.com) to platforma integracyjna, która pomaga łączyć różne aplikacje internetowe i automatyzować przepływy pracy. Blue jest **oficjalnie zintegrowany** z Make.com, co umożliwia bezproblemową automatyzację w ponad 2000 aplikacji—bez potrzeby kodowania!

![Przegląd modułów Blue Make.com](/docs/blue_make_modules_overview.png "Overview of available Blue modules in Make.com")

## Przykłady użycia

Oto kilka potężnych sposobów na rozszerzenie możliwości Blue za pomocą integracji z Make.com:
Integracja Blue z Make.com umożliwia potężną automatyzację w różnych funkcjach biznesowych. W zakresie doświadczeń klientów możesz połączyć Blue z narzędziami takimi jak Intercom, HubSpot i Zendesk, aby tworzyć kompleksowe rekordy klientów, synchronizować interakcje i utrzymywać komunikację w czasie rzeczywistym za pośrednictwem Slacka i Microsoft Teams. W zarządzaniu projektami Blue integruje się z Jira, GitHub, Trello i Monday.com, aby synchronizować zadania, śledzić postępy i automatyzować powiadomienia.

W marketingu i sprzedaży Blue łączy się z Mailchimp, Stripe i Calendly, aby śledzić kampanie, zarządzać płatnościami i automatyzować przepływy pracy związane z spotkaniami. Zarządzanie dokumentami jest usprawnione dzięki integracjom z Google Drive, Dropbox, OneDrive i SharePoint, podczas gdy narzędzia komunikacyjne, takie jak Slack, Gmail i WhatsApp, są zjednoczone w centralnym systemie zarządzania zadaniami Blue. Możliwości analizy danych są wzmacniane dzięki połączeniom z Google Sheets, Power BI, Airtable i Tableau, a rozwiązania specyficzne dla branży są dostępne dla usług finansowych, e-commerce i zarządzania zespołami zdalnymi.

## Połącz i uwierzytelnij swoje konto Blue

Aby bezpiecznie połączyć Blue z Make.com:

1. Zaloguj się na swoje konto Blue i otwórz swój profil → przejdź do zakładki **API**
2. Kliknij **Wygeneruj token**, podaj nazwę (i opcjonalnie datę wygaśnięcia)
3. Skopiuj oba:
   - **Token ID** (`x‑bloo‑token‑id`)
   - **Secret Key** (`x‑bloo‑token‑secret`)

   - **Token ID**
   - **Secret Key**

## Referencja parametrów modułu

### Obserwuj zdarzenia
- Zdarzenia (opcjonalnie): Wiele typów zdarzeń do wyboru
- ID firmy (opcjonalnie): Wiele ID lub slugów firm
- ID projektu (opcjonalnie): Wiele ID lub slugów projektów

### Utwórz rekord
- ID firmy: ID lub slug firmy
- ID projektu: ID lub slug projektu
- Lista: ID listy
- Nazwa rekordu: Nazwa rekordu
- Domyślne pole: Wartość domyślna
- Pojedyncza data: Termin
- Zakres dat: Daty rozpoczęcia i terminy
- Przydzieleni: Wiele użytkowników do wyboru
- Tagi: Wiele tagów do wyboru
- Opis: Pole tekstowe, obsługuje HTML
- Listy kontrolne: Tablica tytułów list kontrolnych

**Obsługiwane niestandardowe pola:**
| Typ pola | Opis |
|----------|------|
| Tekst jednoliniowy | Pole tekstowe jednoliniowe z opcjonalną walidacją i limitem znaków |
| Tekst wieloliniowy | Pole tekstowe wieloliniowe z obsługą tekstu sformatowanego i opcjami formatowania |
| Wybór jednolity | Lista rozwijana z możliwością wyboru jednej opcji i niestandardowymi opcjami |
| Wybór wielokrotny | Lista rozwijana z możliwością wyboru wielu opcji i niestandardowymi opcjami |
| Lokalizacja | Szerokość i długość geograficzna z widokiem mapy i wyszukiwaniem adresów |
| Checkbox | Przełącznik prawda/fałsz z opcjonalną wartością domyślną i niestandardowymi etykietami |
| Kraj | Selekcja dwu-literowego kodu kraju ISO z nazwami krajów |
| Waluta | Pole numeryczne z symbolem waluty i obsługą kursów wymiany (np. USD200, 200USD, KHR4000) |
| Data | Wybór daty z opcjonalnym komponentem czasu i obsługą strefy czasowej |
| Telefon | Pole numeru telefonu obsługujące format E.164, międzynarodowy lub kod kraju + numer z walidacją |
| Email | Pole e-mail z walidacją formatu i weryfikacją domeny |
| Ocena w gwiazdkach | Numeryczna ocena od 1-5 z wizualnymi gwiazdkami i niestandardowymi etykietami |
| Procent | Pole numeryczne z symbolem procentu (0-100) i precyzją dziesiętną |
| URL | Pole adresu internetowego z walidacją formatu i podglądem |
| Liczba | Pole numeryczne z opcjonalnymi miejscami dziesiętnymi i wyświetlaniem jednostki |
| Formuła | Pole obliczeniowe oparte na innych polach z operacjami matematycznymi |
| Unikalny ID | Automatycznie generowany identyfikator z niestandardowym prefiksem i formatem |
| Referencja | Link do innego rekordu z opcjami wyszukiwania i filtrowania |
| Lookup | Dynamiczne pole, które pobiera dane z odniesionych rekordów z opcjami mapowania |
| Czas trwania | Czas trwania w godzinach i minutach z niestandardowym formatem wyświetlania |

### Zaktualizuj rekord
- ID firmy: ID lub slug firmy
- ID projektu: ID lub slug projektu
- Lista: ID listy
- ID rekordu: ID rekordu do zaktualizowania
- Nazwa rekordu: Nazwa rekordu
- Domyślne pole: Wartość domyślna
- Pojedyncza data: Termin
- Zakres dat: Daty rozpoczęcia i terminy
- Przydzieleni: Wiele użytkowników do wyboru
- Tagi: Wiele tagów do wyboru
- Opis: Pole tekstowe, obsługuje HTML
- Listy kontrolne: Tablica tytułów list kontrolnych

**Obsługiwane niestandardowe pola:** (Takie same jak "Utwórz rekord")

### Wyszukaj rekordy
- ID firmy: ID lub slug firmy
- ID projektu: ID lub slug projektu
- Lista: ID listy
- Wyszukiwanie/Zapytanie: Tekst wyszukiwania z zaawansowanymi opcjami filtrowania

### Pobierz rekord
- ID firmy: ID lub slug firmy
- ID projektu: ID lub slug projektu
- Lista: ID listy
- ID rekordu: ID rekordu do pobrania ze wszystkimi powiązanymi danymi

### Zaproś użytkownika
- ID firmy: ID lub slug firmy
- ID projektu: ID lub slug projektu
- Adres e-mail: Adres e-mail użytkownika z walidacją
- ID roli: ID roli (domyślna lub niestandardowa) z uprawnieniami

### Wyświetl niestandardowe pola
- ID firmy: ID lub slug firmy
- ID projektu: ID lub slug projektu
- Limit: Maksymalna liczba pól do zwrócenia z paginacją

### Wyświetl listy
- ID firmy: ID lub slug firmy
- ID projektu: ID lub slug projektu
- Limit: Maksymalna liczba list do zwrócenia z opcjami sortowania

### Wyświetl wszystkie projekty
- ID firmy: Wiele ID lub slugów firm
- Limit: Maksymalna liczba projektów do zwrócenia z filtrowaniem

### Wyświetl tagi
- ID firmy: ID lub slug firmy
- ID projektu: ID lub slug projektu
- Limit: Maksymalna liczba tagów do zwrócenia z statystykami użycia

### Wykonaj GraphQL
- Nagłówki: Tablica par klucz-wartość do autoryzacji i niestandardowych nagłówków
- Metoda: GET lub POST z opcjami żądania
- Zapytanie: Zapytanie lub mutacja GraphQL z obsługą zmiennych
- Nazwa operacji: Nazwa operacji do śledzenia
- Źródło danych zmiennych: Surowe, Formularz lub Kolekcja z kontrolą typów
- Zmienne: Zawartość zmiennych GraphQL z walidacją
