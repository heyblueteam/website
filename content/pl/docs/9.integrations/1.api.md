Blue ma API (Interfejs Programowania Aplikacji), więc możesz budować własne integracje, jeśli zajdzie taka potrzeba.

**Blue ma 100% pokrycia API**. Oznacza to, że wszystko, co możesz zrobić w [dowolnej z naszych aplikacji](/docs/start-guide/download-apps), możesz zrobić programowo. Mamy potężne API [GraphQL](https://graphql.org/), które ma pełne środowisko testowe, w którym możesz testować swoje zapytania i jest w pełni udokumentowane.
