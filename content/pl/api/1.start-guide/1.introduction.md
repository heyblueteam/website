API Blue zapewnia pełny dostęp do platformy zarządzania procesami Blue poprzez nowoczesny interfejs [GraphQL](https://graphql.org/). Zaufany przez **ponad 17 000 klientów** przetwarzających **miliony żądań API rocznie**, nasze API napędza krytyczne dla misji przepływy pracy z **[99,99% dostępnością](/platform/status)**.

## Dlaczego API Blue?

- **100% pokrycia funkcji** - Wszystko, co możesz zrobić w interfejsie użytkownika Blue, możesz zrobić za pomocą API
- **Gotowe dla przedsiębiorstw** - Przetestowane w skali z kompleksowym ograniczaniem przepustowości i bezpieczeństwem
- **Przyjazne dla deweloperów** - GraphQL dostarcza dokładnie te dane, których potrzebujesz, nic więcej
- **Aktualizacje w czasie rzeczywistym** - Subskrypcje WebSocket dla natychmiastowej synchronizacji danych

## Szybki start

Gotowy, aby wykonać swoje pierwsze wywołanie API? Oto jak proste to jest:

```graphql
query MyProjects {
  projectList(filter: { companyIds: ["your-company-id"] }) {
    items {
      id
      name
      updatedAt
    }
  }
}
```

**[Wypróbuj to w naszym GraphQL Playground →](https://api.blue.cc/graphql)**

## Popularne przypadki użycia

### Zarządzanie danymi
- Używaj Blue jako elastycznej bazy danych dla niestandardowych aplikacji
- Synchronizuj dane między Blue a istniejącymi systemami
- Twórz niestandardowe pulpity nawigacyjne i narzędzia raportowe

### Automatyzacja przepływu pracy
- Twórz rekordy z zewnętrznych wyzwalaczy (formularze, webhooki, e-maile)
- Aktualizuj wiele projektów na podstawie logiki biznesowej
- Automatyzuj przypisywanie zadań i powiadomienia

### Rozwój integracji
- Połącz Blue z zewnętrznymi CRM, ERP i specjalistycznymi narzędziami
- Twórz niestandardowe integracje dla swojej branży
- Rozszerzaj możliwości Blue o własne funkcje
