---
title: Utwórz firmę
description: Jak utworzyć nową firmę za pomocą API Blue.
---

Aby utworzyć nową firmę, możesz użyć następującej mutacji:

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

Zauważ, że tworzenie nowych firm może wpłynąć na rozliczenia Twojego konta Blue, ponieważ każda firma jest rozliczana oddzielnie.

Oto opcje dla mutacji `createCompany` oraz ich opisy:

| Opcja | Typ | Opis |
|-------|-----|------|
| name | String | Nazwa nowej firmy. To pole jest wymagane. |  |
| slug | String | Przyjazna dla URL wersja nazwy firmy. Jest używana w URL firmy i powinna być unikalna. To pole jest wymagane i będzie również działać jako unikalny identyfikator dla firmy. |
