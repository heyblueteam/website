The Checkbox custom field provides a checkbox that can be used for a variety of purposes, including approvals or [to trigger an automation](/public/docs/automations/introduction)

![Checkbox 2](/public/docs/customField/checkBox/Checkbox_1.png "Checkbox 1")

Using Checkbox custom fields for approvals

Approvals are a specifically good use case for the Checkbox custom field because of Blue's audit capabilities. Each time the checkbox is ticked or unticked, Blue logs the action in a permanent manner:

![Checkbox 2](/public/docs/customField/checkBox/Checkbox_2.png "Checkbox 2")

Blue logs the person, action, and time for every change.
