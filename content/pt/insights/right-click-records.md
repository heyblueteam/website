A atualização de hoje traz uma pequena, mas atenta melhoria para nossa [plataforma de gerenciamento de projetos](/).

Agora você pode clicar com o botão direito em qualquer registro para abrir um menu de contexto, permitindo que você faça edições rápidas sem precisar abrir completamente o registro. Este recurso é perfeito para fazer alterações rápidas, como atualizar tags, atribuir tarefas ou modificar outros detalhes do registro em questão de segundos.

![](/insights/right-click-record.png)

## Lista de Opções

Aqui estão as opções disponíveis quando você clica com o botão direito em um registro:

- **Abrir em Nova Aba:** Abre o registro em uma nova aba do navegador.
- **Renomear:** Permite que você mude o nome do registro diretamente do menu de contexto.
- **Marcar como Completo:** Marque rapidamente o registro como concluído.
- **Mover:** Mova o registro para outra lista de projetos.
- **Copiar Tarefa:** Crie um duplicado do registro atual.
- **Copiar URL:** Copia a URL do registro para sua área de transferência para fácil compartilhamento.
- **Mover para o Topo:** Mova o registro para o topo da lista.
- **Data de Vencimento:** Defina ou altere a data de vencimento do registro.
- **Atribuir:** Atribua o registro a um membro da equipe.
- **Tag:** Adicione ou altere tags associadas ao registro.
- **Definir Cor:** Altere o rótulo de cor do registro para melhor organização visual.
- **Excluir:** Remova permanentemente o registro.


Quando você quiser editar um registro, agora pode clicar com o botão direito para ter um menu em vez de ter que abrir o registro para fazer edições. Isso torna edições simples, como mudar uma tag ou atribuir alguém, muito mais rápidas.

## Seleção Múltipla

Mas isso não é tudo!

Essa nova funcionalidade é totalmente compatível com nosso recurso de seleção múltipla. Basta CMD+Click (no Mac) ou CTRL+Click (no Windows) para selecionar vários registros simultaneamente. Em seguida, clique com o botão direito para abrir o menu de contexto e realizar ações em massa. Essa melhoria acelera significativamente processos como atribuir várias tarefas a um membro da equipe ou etiquetar vários registros de uma só vez.
