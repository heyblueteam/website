[Make.com](https://www.make.com) é uma plataforma de integração que ajuda a conectar diferentes aplicativos da web e automatizar fluxos de trabalho. O Blue está **oficialmente integrado** com o Make.com, permitindo uma automação perfeita em mais de 2.000 aplicativos—sem necessidade de codificação!

![Visão Geral dos Módulos do Blue Make.com](/docs/blue_make_modules_overview.png "Overview of available Blue modules in Make.com")

## Casos de Uso

Aqui estão algumas maneiras poderosas de expandir as capacidades do Blue com integrações do Make.com:
A integração do Blue com o Make.com permite uma automação poderosa em várias funções de negócios. Para a experiência do cliente, você pode conectar o Blue a ferramentas como Intercom, HubSpot e Zendesk para criar registros abrangentes de clientes, sincronizar interações e manter comunicação em tempo real através do Slack e Microsoft Teams. Na gestão de projetos, o Blue se integra ao Jira, GitHub, Trello e Monday.com para sincronizar tarefas, acompanhar o progresso e automatizar notificações.

Para marketing e vendas, o Blue se conecta ao Mailchimp, Stripe e Calendly para acompanhar campanhas, gerenciar pagamentos e automatizar fluxos de trabalho de reuniões. A gestão de documentos é simplificada através de integrações com Google Drive, Dropbox, OneDrive e SharePoint, enquanto ferramentas de comunicação como Slack, Gmail e WhatsApp são unificadas através do sistema central de gerenciamento de tarefas do Blue. As capacidades de análise de dados são aprimoradas através de conexões com Google Sheets, Power BI, Airtable e Tableau, e soluções específicas do setor estão disponíveis para serviços financeiros, e-commerce e gerenciamento de equipes remotas.

## Conectar e Autenticar Sua Conta do Blue

Para conectar o Blue ao Make.com de forma segura:

1. Faça login na sua conta do Blue e abra seu perfil → vá para a aba **API**
2. Clique em **Gerar um Token**, forneça um nome (e uma expiração opcional)
3. Copie ambos:
   - **Token ID** (`x‑bloo‑token‑id`)
   - **Chave Secreta** (`x‑bloo‑token‑secret`)

::callout
 ---
icon: lightbulb
---
**Nota Importante** <br/>
A chave secreta é exibida apenas uma vez—guarde-a com segurança!
::

5. No Make.com, ao criar uma nova conexão, insira:
   - **Token ID**
   - **Chave Secreta**

## Referência de Parâmetros do Módulo

### Assistir Eventos
- Eventos (opcional): Tipos de eventos de seleção múltipla
- Company ID (opcional): IDs ou slugs de empresas de seleção múltipla
- Project ID (opcional): IDs ou slugs de projetos de seleção múltipla

### Criar um Registro
- Company ID: ID ou slug da empresa
- Project ID: ID ou slug do projeto
- List: ID da lista
- Record Name: Nome do registro
- Default Field: Valor padrão
- Single Date: Data de vencimento
- Date Range: Datas de início e vencimento
- Assignees: Usuários de seleção múltipla
- Tags: Tags de seleção múltipla
- Description: Campo de texto, suporta HTML
- Checklists: Array de títulos de checklist

**Campos Personalizados Suportados:**
| Tipo de Campo | Descrição |
|---------------|-----------|
| Texto de Linha Única | Campo de texto de linha única com validação opcional e limite de caracteres |
| Texto de Múltiplas Linhas | Campo de texto de múltiplas linhas com suporte a texto rico e opções de formatação |
| Seleção Única | Dropdown com seleção de uma única opção e opções personalizadas |
| Seleção Múltipla | Dropdown com seleção de múltiplas opções e opções personalizadas |
| Localização | Latitude e longitude com visualização de mapa e busca de endereço |
| Caixa de Seleção | Alternância verdadeiro/falso com valor padrão opcional e rótulos personalizados |
| País | Seletor de código de país ISO de duas letras com nomes de países |
| Moeda | Campo numérico com símbolo de moeda e suporte a taxa de câmbio (ex: USD200, 200USD, KHR4000) |
| Data | Seletor de data com componente de tempo opcional e suporte a fuso horário |
| Telefone | Campo de número de telefone suportando E.164, formato internacional ou código do país + número com validação |
| Email | Campo de email com validação de formato e verificação de domínio |
| Avaliação por Estrelas | Avaliação numérica de 1-5 com estrelas visuais e rótulos personalizados |
| Percentual | Campo numérico com símbolo de porcentagem (0-100) e precisão decimal |
| URL | Campo de endereço da web com validação de formato e visualização |
| Número | Campo numérico com casas decimais opcionais e exibição de unidade |
| Fórmula | Campo calculado com base em outros campos com operações matemáticas |
| ID Único | Identificador gerado automaticamente com prefixo e formato personalizados |
| Referência | Link para outro registro com opções de busca e filtro |
| Lookup | Campo dinâmico que puxa dados de registros referenciados com opções de mapeamento |
| Duração do Tempo | Período em horas e minutos com formato de exibição personalizado |

### Listar Campos Personalizados
- Company ID: ID ou slug da empresa
- Project ID: ID ou slug do projeto

### Listar Listas
- Company ID: ID ou slug da empresa
- Project ID: ID ou slug do projeto
- Limit: Número máximo de listas a serem retornadas com opções de ordenação

### Listar Tags
- Company ID: ID ou slug da empresa
- Project ID: ID ou slug do projeto
- Limit: Número máximo de tags a serem retornadas com estatísticas de uso

### Executar GraphQL
- Headers: Array de pares chave-valor para autenticação e cabeçalhos personalizados
- Method: GET ou POST com opções de solicitação
- Query: Consulta ou mutação GraphQL com suporte a variáveis
- Operation Name: Nome da operação para rastreamento
- Variables Data Source: Raw, Form, ou Collection com verificação de tipo
- Variables: Conteúdo das variáveis GraphQL com validação
