Blue tem uma API (Interface de Programação de Aplicações), para que você possa construir suas próprias integrações, se necessário.

**Blue tem 100% de cobertura de API**. Isso significa que qualquer coisa que você possa fazer em [qualquer um de nossos aplicativos](/docs/start-guide/download-apps), você pode fazer programaticamente. Temos uma poderosa API [GraphQL](https://graphql.org/) que possui um playground completo onde você pode testar suas consultas e está totalmente documentada.
