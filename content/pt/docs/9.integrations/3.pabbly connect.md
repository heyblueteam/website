title: Pabbly Connect
description: Blue está integrado com o Pabbly Connect para receber dados de milhares de outras aplicações.

[Pabbly Connect](https://payments.pabbly.com/api/affurl/RVYZ07kQyUZ0Z1HUKZ1m/zp2GoChM1JcZ0dQBo?target=nMfdbf0I90K3UdJn) permite que você conecte o Blue com muitos aplicativos populares como Google Calendar, Stripe, MailChimp e mais. Isso possibilita a automação de fluxos de trabalho ao enviar dados entre ferramentas.

<youtube url="https://www.youtube.com/watch?v=IRxYAe5snzY" />

Existem possibilidades infinitas para integrar o Blue usando o Pabbly Connect. Aqui estão os passos principais:

 ## Acesse seu Painel do Pabbly

1. Faça login no seu [Painel do Pabbly](https://connect.pabbly.com/workflows)
2. Clique no botão "+ Criar Fluxo de Trabalho".
3. Nomeie seu fluxo de trabalho.
4. Selecione o Blue como o aplicativo de destino.

## Conecte-se à sua Conta do Blue

1. Gere um [token de API](/docs/integrations/api) no seu perfil do Blue.
2. No Pabbly, insira seu:
- ID do Token
- Chave Secreta
- URL da Empresa (ex. `https://app.blue.cc/company/mycompany`)
- ID do Projeto (ex. `myproject`)

::callout
 ---

icon: lightbulb
---

*Copie sua Chave Secreta imediatamente, pois ela não será exibida novamente.* 
::

## Crie uma Integração de Teste
1. Selecione um aplicativo de gatilho e evento (ex. novo evento do Google Calendar).
2. Envie uma solicitação de teste.
3. Verifique o Blue para garantir que os dados de teste foram recebidos no seu projeto especificado.
