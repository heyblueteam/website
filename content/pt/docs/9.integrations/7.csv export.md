## Exportando dados

1. No projeto que você deseja exportar, clique no ícone de configurações (engrenagem)
2. Selecione "Exportar"
3. Clique em "Exportar" para iniciar o processo de exportação nos Servidores do Blue.
4. Uma vez concluído, você receberá um e-mail com um link para baixar o arquivo CSV exportado

::callout
 ---

icon: lightbulb
---

**Quanto tempo leva para receber o e-mail com os dados?** <br/> <br/>
Normalmente, isso chegará em alguns segundos, mas pode depender de quantos outros processos de exportação estão em execução de outros clientes e do tamanho dos seus dados. Para conjuntos de dados que têm mais de 100.000 itens, isso pode levar alguns minutos.
::

![Exportação CSV 1](/docs/integration/CSVexport/1.png "CSV Export 1")

Observe que a funcionalidade de exportação exporta apenas os dados dos seus registros. Arquivos na aba Arquivos devem ser baixados manualmente em massa.

## Exportar com Filtros Ativos

Se você já aplicou filtros ao seu projeto, pode facilmente exportar apenas os registros filtrados usando a opção "Exportar com Filtros Ativos". Dessa forma, você não precisará se preocupar em filtrar manualmente os dados exportados, pois ele incluirá automaticamente apenas os registros que correspondem às suas configurações de filtro atuais.

Veja como você pode exportar com filtros ativos:

1. Aplique os filtros ao seu projeto usando as opções de filtro disponíveis, como responsáveis, tags ou filtros avançados.
2. Uma vez que você tenha a visualização filtrada que deseja exportar, vá para as configurações do projeto e clique em Importar & Exportar.
3. No menu de opções de exportação, selecione "Exportar com Filtros Ativos."
4. Uma vez que sua exportação estiver pronta, você receberá os dados em seu e-mail.

## Principais Benefícios

- **Portabilidade de dados**: Baixe projetos do Blue quando necessário para backup ou análise externa
- [Acionar via API](/docs/integrations/api): Você pode usar sistemas de terceiros para acionar automaticamente exportações do Blue regularmente para atender às suas necessidades de backup de dados.
- **Acessibilidade para partes interessadas**: Compartilhe CSVs exportados facilmente com clientes, gerentes e outros fora do Blue.

A exportação oferece acessibilidade e flexibilidade de dados para atender a várias necessidades potenciais de integração e relatórios. Combinada com [importação](/docs/integrations/csv-import), proporciona controle total sobre seus dados do Blue.
