Zapier é uma plataforma de integração que ajuda a conectar diferentes aplicativos web e automatizar fluxos de trabalho.

**Casos de Uso**

Aqui estão algumas das maneiras mais populares que o Blue é integrado com outros aplicativos via Zapier:

**_Sincronizar Registros do Blue com Google Sheets_**

À medida que os registros do Blue são criados ou atualizados, adicione-os automaticamente como novas linhas em uma planilha do Google para relatórios centralizados.

**_Enviar Confirmação de Pedido a partir de registros do Blue_**

Quando você seleciona um determinado campo personalizado no Blue, envie automaticamente um e-mail para confirmar o pedido com seu cliente.

**_Adicionar Comentários do Blue ao Slack_**

Receba notificações em tempo real do Blue postadas nos canais corretos do Slack para que sua equipe esteja ciente de atualizações importantes nos registros.

## Principais Benefícios

**Automação Entre Ferramentas**: Zaps criados no Zapier podem enviar e receber dados automaticamente entre o Blue e milhares de outros aplicativos. Isso elimina o trabalho manual de mover dados entre sistemas.

**Integrações em Tempo Real**: Usando os gatilhos de webhook do Blue, os Zaps reagem instantaneamente a eventos no Blue, como um novo registro, comentário adicionado, mudança de status, etc. Isso permite sincronização em tempo real e automação de processos.

**Fácil de Construir e Manter**: Os Zaps oferecem uma maneira amigável e sem código para integrar ferramentas. Nenhum trabalho de desenvolvimento é necessário. E eles continuam funcionando sem manutenção contínua.

1. Faça login na sua conta do Blue nas configurações de perfil
2. Vá para as configurações de perfil e clique em "Webhooks"
3. Clique em "Adicionar Webhook"
4. Dê um nome ao webhook e insira a URL do webhook do Zapier
5. Escolha quais eventos devem acionar o webhook
6. Clique em "Criar Webhook"

## Blue para Zapier

1. Faça login na sua conta do Zapier
2. Clique em Fazer um Zap  
3. Para o primeiro passo, pesquise e selecione o [aplicativo de gatilho "Webhooks by Zapier"](https://zapier.com/apps/webhook)
4. Configure-o para aceitar payloads da sua URL de webhook do Blue
5. Para o segundo passo, pesquise e selecione o aplicativo de destino para enviar dados
6. Mapeie os campos do payload JSON para os campos de entrada do aplicativo de destino
7. Ative o Zap

Agora, quando os eventos especificados ocorrerem no Blue, o payload do webhook será enviado para o Zapier, que automaticamente passará os dados para o seu aplicativo integrado de acordo com as configurações do Zap.

## Zapier para Blue

Para enviar dados de outros aplicativos para o Blue, siga estes passos:

**1. Obter Credenciais da API do Blue**
- Vá para Configurações do Blue > Tokens da API
- Crie um novo token ou use credenciais existentes:
  - ID do Token
  - Segredo do Token
  - ID da Empresa
  - ID do Projeto

**2. Criar Zap no Zapier**
1. Configure seu aplicativo de gatilho (por exemplo, Google Sheets, Slack, etc.)
2. Adicione "Webhooks by Zapier" como o aplicativo de Ação
3. Escolha o tipo de evento "Solicitação Personalizada"

**3. Configurar Webhook**
- Método: POST
- URL: `https://api.blue.cc/graphql` (ou endpoint beta, se necessário)
- Passagem de Dados: FALSO
- Descompactar: NÃO

**4. Configurar Cabeçalhos:**
```json
{
  "Content-Type": "application/json",
  "X-Bloo-Token-ID": "Your_Token_ID",
  "X-Bloo-Token-Secret": "Your_Token_Secret", 
  "X-Bloo-Company-ID": "Your_Company_ID",
  "X-Bloo-Project-ID": "Your_Project_ID"
}
```

**5. Configurar Consulta (Exemplo de Mutação):**
```json
{
  "query": "mutation CreateTodo($input: CreateTodoInput!) {\n  createTodo(input: $input) {\n    id\n    title\n    customFields {\n      value\n    }\n  }\n}",
  "variables": {
    "input": {
      "todoListId": "your-todo-list-id",
      "title": "New task from Zapier",
      "customFields": [
        {
          "customFieldId": "priority-field-id",
          "value": "High"
        },
        {
          "customFieldId": "due-date-field-id",
          "value": "2024-03-01"
        }
      ]
    }
  }
}
```

**6. Testar e Ativar**
- Use o Playground da API do Blue para testar mutações primeiro
- Mapeie valores dinâmicos dos aplicativos de gatilho usando os campos de dados do Zapier
- Ative o Zap após testes bem-sucedidos
