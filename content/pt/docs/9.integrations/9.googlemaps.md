O Blue permite que você use o Google Maps para a visualização do mapa em vez do OpenStreetMap padrão. O Google Maps frequentemente possui mapas mais detalhados de certas áreas e dados de localização mais precisos. No entanto, isso requer uma chave de API do Google Maps que deve ser inserida nas configurações da empresa pelo proprietário da empresa.

Quando você muda para o Google Maps, isso afeta todas as visualizações de mapa no Blue, incluindo o mapa da empresa, a [visualização do mapa em projetos](/docs/views/map) e o [campo personalizado de localização](/docs/custom-fields/location-map).

Para usar o Google Maps no Blue, as seguintes APIs precisam ser ativadas no Google Cloud Console:
- [Maps Tiles API](https://console.cloud.google.com/apis/library/tile.googleapis.com): Isso é necessário para renderizar os blocos do mapa.
- [Places API](https://console.cloud.google.com/apis/library/places.googleapis.com): Isso é necessário para pesquisar lugares e obter suas coordenadas.
- [Geocoding API](https://console.cloud.google.com/apis/library/geocoding-backend.googleapis.com): Isso é necessário para obter as coordenadas de um lugar.

Você precisará se inscrever para uma [conta do Google Cloud](https://console.cloud.google.com/) e criar um projeto para obter as chaves da API seguindo as instruções [aqui](https://developers.google.com/maps/documentation/embed/get-api-key).

## Restringindo o Uso da Chave da API

Por motivos de segurança, é crucial restringir sua chave da API do Google Maps para funcionar apenas com domínios autorizados. Isso impede o uso não autorizado de sua chave da API se ela for exposta. Você deve restringir a chave da API para funcionar apenas com:

- app.blue.cc - O aplicativo principal do Blue
- forms.blue.cc - Para formulários públicos
- Seu domínio personalizado se você estiver usando [marca branca](/docs/start-guide/white-label)

Para restringir sua chave da API:

1. Vá para a [página de Credenciais do Google Cloud Console](https://console.cloud.google.com/apis/credentials)
2. Encontre sua chave da API e clique em "Editar" (ícone de lápis)
3. Em "Restrições de aplicativo", selecione "Referenciadores HTTP (sites)"
4. Adicione os seguintes referenciadores:
   - `*.app.blue.cc/*`
   - `*.forms.blue.cc/*`
   - `*.yourdomain.com/*` (se estiver usando marca branca)
5. Clique em "Salvar"
