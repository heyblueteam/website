## Acessando a Pesquisa

A nova barra de pesquisa está localizada na barra de navegação superior para fácil acesso. No entanto, também adicionamos atalhos de teclado para tornar sua experiência de pesquisa ainda mais rápida:

1. **No Mac**: Pressione **cmd+K**
2. **No Windows**: Pressione **ctrl+K**

Esses atalhos permitem que você acesse rapidamente a função de pesquisa de qualquer lugar no Blue, sem precisar mover o cursor.

![Pesquisa](/docs/Search.png "Search")

## Experiência de Pesquisa Expandida

Para uma experiência de pesquisa mais imersiva, você pode pressionar a tecla Tab após usar os atalhos de teclado. Isso expandirá a pesquisa para um modo de tela cheia, dando mais espaço para visualização.

## Pesquisa Abrangente

A Função de Pesquisa pode pesquisar em:
- Registros
- Comentários dentro de registros
- Documentos
- Arquivos

Isso significa que, se você está procurando um registro de cliente específico, uma decisão chave feita em um comentário de projeto, um documento importante ou um arquivo crucial, a Pesquisa pode encontrá-lo para você em segundos.

## Navegação Rápida entre Registros

Uma das características marcantes da Pesquisa é sua capacidade de atuar como um interruptor rápido para mover entre registros. Ao digitar o nome ou qualquer parte de um registro, você pode pular instantaneamente para esse registro, independentemente de qual projeto ou lista ele esteja.
