---
title: Ativando e Desativando Recursos
description: Recursos no Blue podem ser ativados ou desativados para um projeto.
---

Os administradores do projeto podem ativar ou desativar recursos para um projeto para todos os usuários. Isso é útil se você deseja limitar os recursos disponíveis para um projeto. Por exemplo, nem todos os projetos precisam ter o Wiki ativado. Os administradores também podem reorganizar a ordem das abas no projeto.

Limitar recursos em um projeto também pode ser feito em nível de grupo de usuários usando [funções de usuário personalizadas](/docs/user-management/roles/custom-user-roles).
