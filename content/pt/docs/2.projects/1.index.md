## Como os Projetos Funcionam

Cada projeto no Blue é totalmente independente. Apenas usuários que são especificamente convidados para o projeto podem ver os dados e colaborar. Você pode ter um número ilimitado de projetos dentro de uma conta Blue.

Existem duas formas de pensar sobre projetos no Blue:

1. **Projetos**: Um projeto geralmente tem um início e fim claros, focando em metas ou eventos específicos. Exemplos incluem o lançamento de um evento, rebranding ou design de um website. Esses projetos são orientados a objetivos e têm metas e cronogramas definidos. Uma vez concluídos, são frequentemente [arquivados](/docs/projects/archiving-projects).
2. **Processos**: O Blue também suporta processos contínuos que não têm uma data de término fixa. Estes podem incluir tarefas como atendimento ao cliente, operações de helpdesk, acompanhamento de vendas e recrutamento. Esses projetos exigem gerenciamento e organização regulares, mas são de natureza contínua.

## Configurações do Projeto

[Administradores de Projeto](/docs/user-management/roles/project-administrator) verão um ícone de configurações ao lado do nome do projeto na barra superior.

Existem várias configurações disponíveis:

- **Editar configurações do projeto**: Você pode alterar o nome do projeto, escrever uma descrição do projeto, adicionar uma foto ou ícone do projeto e também alterar o nome de [Registros](/docs/records/introduction) para outra coisa (ex: Tarefa, Venda)
- **Definir Ícone e Cor**: Você pode escolher um ícone e cor para o projeto; isso será mostrado na barra lateral para todos os usuários que estão no projeto.
- **Gerenciar**: Isso abre um submenu onde você pode editar os recursos disponíveis e [campos personalizados](/docs/custom-fields/introduction) no projeto.
- [Converter em Modelo](/docs/projects/templates): Isso permite que o projeto seja convertido em um modelo reutilizável.
- [Copiar](/docs/projects/copying-projects): Permite duplicar o projeto
- [Arquivar](/docs/projects/archiving-projects): Arquiva o projeto para referência futura, transformando-o em um projeto somente leitura.
- [Importar](/docs/integrations/csv-import) **&** [Exportar](/docs/integrations/csv-export): Permite importar e exportar registros do projeto
- **Conexões OAuth**: Permite conectar seu projeto a um serviço externo como Github ou Quickbooks.
- **Excluir**: Permite a exclusão permanente do projeto.
