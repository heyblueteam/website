O campo Texto de Linha Única permite adicionar strings de texto não formatado aos seus registros no Blue. Não existe limite de caracteres, então você pode inserir texto de qualquer comprimento neste campo. No entanto, o texto é restrito a uma única linha e não suporta opções de formatação como negrito, itálico ou links.

![Single Line Text](/docs/SingleLineText.png "Single Line Text")

Uma forma flexível de capturar dados de texto estruturados

O campo de texto é altamente flexível, permitindo a entrada de letras e números, incluindo caracteres especiais. Não há restrições sobre quais caracteres podem ser inseridos. O sistema não aplica validações ou máscaras ao texto inserido pelo usuário.

O campo Texto de Linha Única é excelente para capturar todos os tipos de informação, incluindo nomes, descrições, números de modelo, SKUs, endereços, notas, IDs e muito mais. Você pode usar este campo flexível para armazenar nomes de clientes, fornecedores ou contatos, indicar códigos de inventário e identificadores, fazer anotações gerais sobre registros ou armazenar IDs únicos para relacionamentos e consultas de banco de dados. O campo de texto de linha única permite inserir eficientemente qualquer informação importante baseada em texto relacionada aos seus registros no Blue.

Você pode aproveitar as [capacidades avançadas de filtragem de registros](/pt/docs/views/filters) do Blue para encontrar e filtrar registros contendo texto específico dentro do campo Texto de Linha Única para localizar registros correspondentes.

Com seu design simples de linha única, capacidade de texto ilimitada e integração com filtragem, o campo Texto de Linha Única oferece uma experiência otimizada para capturar dados não estruturados. Ele fornece flexibilidade para armazenar qualquer informação conforme necessário por registro.
