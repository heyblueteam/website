Campos personalizados de fórmulas permitem que você crie cálculos baseados nos seus dados existentes no Blue. Isso é ótimo para calcular margens de lucro, totais, taxas de impostos e comissões.


![Formula 1](/docs/Formula_1.png "Formula 1")

Fórmulas podem usar dados dos seguintes campos personalizados dentro do mesmo registro:

- [Moeda](/pt/docs/custom-fields/currency)
- [Classificação por Estrelas](/pt/docs/custom-fields/star-rating)
- [Número](/pt/docs/custom-fields/number)
- [Porcentagem](/pt/docs/custom-fields/percent)

Simplesmente digite uma chave `{` e o sistema oferecerá uma lista suspensa de campos que você pode referenciar.

## Operações Suportadas

Você pode então executar operações nesses valores usando funções:

- **AVERAGE**: Calcular a média de um conjunto de números.
- **AVERAGEA**: Determinar a média, tratando texto como zero.
- **COUNT**: Contar o número de entradas em um campo numérico.
- **COUNTA**: Contar o número de valores não vazios.
- **MAX**: Encontrar o valor máximo em um conjunto.
- **MIN**: Identificar o valor mínimo em um conjunto.
- **MINUS**: Subtrair um valor de outro.
- **MULTIPLY**: Multiplicar dois valores.
- **SUM**: Somar uma série de números

Você pode usar essas operações passando o nome do campo personalizado para a função. Por exemplo, se você tivesse um campo personalizado chamado `Custo`, poderia usar `AVERAGE(Custo)` para calcular o custo médio de um conjunto de registros.

Para passar múltiplos campos personalizados para uma função, você pode usar a vírgula `,` para separar cada campo. Por exemplo, se você tivesse campos personalizados chamados `Custo` e `Lucro`, poderia usar `AVERAGE(Custo, Lucro)` para calcular o custo médio e lucro de um conjunto de registros.

Você também pode usar os seguintes operadores matemáticos:

| Operador | Descrição |
|:--------:|:-----------:|
| **+**    | Adição    |
| **-**    | Subtração |
| **\***    | Multiplicação |
| **/**    | Divisão    |
| **%**    | Porcentagem  |
| **^**    | Exponenciação |

Não há restrições sobre a complexidade ou número de operações em suas fórmulas. Elas validarão corretamente desde que a sintaxe seja válida. Fórmulas apenas referenciam campos dentro do mesmo registro, não dados externos.

## Configurar Exibição
O passo final para criar um campo personalizado de fórmula é decidir como exibir os dados.

Campos de fórmula oferecem três opções de exibição:

1. **Número**: Esta opção mostra um valor numérico simples. Você também pode escolher quantas casas decimais exibir, dando controle sobre o nível de detalhe em seus dados.
2. **Moeda**: Use esta opção para mostrar valores em formato monetário. É ótima para rastrear orçamentos, receita, custos e outros dados financeiros. A moeda será formatada de acordo com padrões locais e suporta várias moedas globais, com [suporte para numerosas moedas globais](/pt/docs/custom-fields/currency).
3. **Porcentagem**: Este formato exibe valores como porcentagens, que é útil para mostrar taxas de conclusão, métricas de crescimento ou outras proporções. Ele converte automaticamente o valor calculado em porcentagem, facilitando o entendimento de proporções.
