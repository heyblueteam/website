O campo personalizado URL/Link permite armazenar links e endereços web dentro dos seus registros no Blue. Quando uma URL é inserida neste campo, o sistema validará se é um link estruturado adequadamente.

![URL-Link 1](/docs/URLlink_1.png "URL-Link 1")

O campo personalizado URL/Link no Blue

Cada registro pode conter um link, armazenado em um único campo personalizado URL/Link. O link é clicável, permitindo navegar diretamente para a página web ou recurso clicando nele. Por conveniência, ele abrirá automaticamente em uma nova aba do navegador.

![URL-Link 2](/docs/URLlink_2.png "URL-Link 2")

Você pode abrir URLs em uma nova aba com um clique

Este campo é extremamente útil para vincular documentos baseados na nuvem como Google Docs, arquivos Word, planilhas Excel, etc. Armazenar esses links permite acesso com um clique aos arquivos relevantes associados a cada registro.

O campo URL/Link é ótimo para casos de uso adicionais como:

- Armazenar sites de clientes ou parceiros
- Capturar links associados aos registros
- Criar referências para páginas web

Com sua interface simplificada e recursos de link integrados, o campo personalizado URL/Link permite enriquecer registros com recursos web contextuais e acessá-los instantaneamente com um clique.
