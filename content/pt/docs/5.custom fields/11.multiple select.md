O campo personalizado de Seleção Múltipla funciona de forma idêntica ao [campo personalizado de Seleção Única](/pt/docs/custom-fields/single-select), com uma distinção principal — a capacidade de fazer múltiplas seleções por registro.

O campo personalizado de Seleção Múltipla permite criar uma lista suspensa de opções que os usuários podem escolher ao inserir dados nos registros. Você pode definir até 10.000 opções com rótulos e cores.

![MultipleSelect 1](/docs/MultipleSelect_1.png "MultipleSelect 1")

Escolhendo cores para opções dentro de um campo personalizado de Seleção Múltipla

Permitir múltiplas seleções é ideal para capturar:

- Taxonomias - Classificar registros em múltiplas categorias (ou seja, marcar registros com todos os rótulos relevantes)
- Dados Compostos - Combinar opções para descrever registros complexos (ou seja, selecionar todas as propriedades aplicáveis do produto)
- Trilhas de Auditoria - Rastrear mudanças ao longo do tempo (ou seja, registrar atualizações de status de um registro)

::callout
---

icon: lightbulb
---

Adicionar opções uma por uma pode se tornar tedioso para listas suspensas de seleção. Felizmente, as opções podem ser adicionadas em massa copiando e colando de uma lista com marcadores/separada por linhas ou coluna de planilha. Basta copiar a lista de escolhas e colá-las no diálogo de criação de opções no Blue. Isso criará automaticamente uma opção para cada linha ou marcador, simplificando a configuração.
::

A flexibilidade das múltiplas seleções expande como as listas suspensas personalizáveis podem estruturar dados do projeto.

Uma interface de pesquisa-enquanto-digita torna fácil encontrar a seleção ideal, mesmo com milhares de opções.

![MultipleSelect 2](/docs/MultipleSelect_2.png "MultipleSelect 2")

Após a criação, os administradores do projeto ainda podem editar a lista de opções conforme necessário, incluindo reordenação, adições ou remoções.

O campo de Seleção Múltipla oferece uma lista suspensa personalizada projetada para seleção rápida de opções predefinidas extensas. Definir opções e rótulos personalizados alinha o campo intimamente com seus fluxos de trabalho e terminologia. Essas listas suspensas se integram firmemente em todo o Blue. [Administradores de Projeto](/pt/docs/user-management/roles/project-administrator) podem restringir o acesso de visualização e edição por [função de usuário](/pt/docs/user-management/roles/custom-user-roles), [formulários](/pt/docs/projects/forms) externos podem aproveitar Seleções Múltiplas para entrada padronizada, e [painéis](/pt/docs/dashboards) podem filtrar por valores de Seleção. Seja coletando dados categóricos ou padronizando terminologia, o campo de Seleção Múltipla fornece uma experiência otimizada para gerenciar listas suspensas personalizadas alinhadas com suas necessidades de negócio.
