O Quadro é a visualização padrão no Blue, fornecendo um quadro de projeto flexível no estilo kanban. As listas representam diferentes estágios ou etapas em um processo, permitindo que os usuários mapeiem fluxos de trabalho visualmente.

A Visualização de Quadro é comumente usada para:

- Gerenciamento de projetos — Mapeamento de estágios de projeto desde o backlog até a conclusão
- Funis de vendas — Rastreamento de estágios de negócios desde leads até fechamento
- Processos de contratação — Visualização de estágios de candidatos desde recrutamento até contratação


## Listas como Etapas do Processo

As listas no quadro representam etapas principais de um processo. Configurações típicas envolvem estágios progredindo da esquerda para a direita, dos estágios iniciais aos finais. Os nomes das listas indicam claramente o estágio específico.

Os registros podem ser arrastados entre listas para avançá-los através de diferentes etapas do processo.

## Cartões de Registro

Os cartões de registro fornecem detalhes importantes como título, responsável(eis), data de vencimento, tags e descrição. Os cartões podem ser arrastados para reordená-los dentro de uma lista.

Os usuários podem criar registros rapidamente clicando em "+ Adicionar Registro" no topo ou na parte inferior de qualquer lista.

::callout
---

icon: lightbulb
---

Você também pode criar múltiplos registros simultaneamente copiando e colando uma lista de marcadores ou uma coluna de células do Excel ou Google Sheets.
::

Os usuários podem clicar com o botão direito em um cartão de registro para acessar opções adicionais:

- **Abrir em Nova Aba** - Abre os detalhes completos do registro em uma nova aba do navegador
- **Renomear** - Edita o nome do registro
- **Marcar como Concluído** - Marca o status do registro como concluído
- **Atribuir** - Atribui o registro a um usuário
- **Tag** - Adiciona tags ao registro
- **Mover** - Move o registro para uma lista diferente
- **Definir Cor do Cartão** - Altera a cor do cartão de registro
- **Copiar** - Duplica o registro
- **Copiar URL** - Copia um link para o registro
- **Mover para o Topo** - Move o registro para o topo de sua lista
- **Excluir** - Exclui o registro

![Board 2](/docs/Board_2.png "Board 2")


Clicar com o botão direito nos cartões de registro fornece acesso rápido às opções de gerenciamento, permitindo que os usuários modifiquem registros diretamente da Visualização de Quadro.

## Ações em Lote

As listas oferecem ações em lote como marcar todos os registros com tags, atribuir registros, marcar como concluído/incompleto, ou mover registros entre listas. Isso permite gerenciar grupos de registros de forma eficiente. [Veja nossa página dedicada sobre listas para mais informações](/docs/records/lists)


## Filtros

Os filtros permitem focar em subconjuntos de registros com base em critérios como responsável, tag, data de vencimento, etc. Os filtros se aplicam apenas aos cartões de registro visíveis, não às listas inteiras. [Veja nossa página dedicada sobre filtros para mais informações](/docs/views/filters)
