Os painéis no Blue são poderosos porque permitem combinar dados em tempo real de múltiplos projetos simultaneamente. Isso permite que você construa painéis ao vivo que fornecem informações significativas para a tomada de decisões.

## Opções de Gráficos

O Blue vem com três opções para gráficos:

1. [Cartões Estatísticos:](/docs/dashboards/stat-cards) Ideais para métricas-chave de relance, os cartões estatísticos exibem pontos de dados singulares em tempo real, como receita total do projeto, problemas ativos atuais ou porcentagem geral de conclusão do projeto. Eles fornecem uma visão imediata de dados críticos com uma interface simples e clara.
2. [Gráficos de Pizza:](/docs/dashboards/pie-charts) Melhores para visualizar a composição dos dados, os gráficos de pizza no Blue ajudam você a entender a proporção de vários elementos dentro de um conjunto de dados. Por exemplo, você poderia usar um gráfico de pizza para mostrar a distribuição de tarefas entre diferentes departamentos ou a divisão percentual da receita de diferentes tipos de clientes.
3. [Gráficos de Barras:](/docs/dashboards/bar-graphs) Eficazes para comparar quantidades, os gráficos de barras permitem que você contraste diferentes grupos lado a lado. Use gráficos de barras para comparar números de vendas em diferentes trimestres, o número de tarefas que cada membro da equipe completa ou a frequência de certas consultas de clientes ao longo do tempo.

## Compartilhamento
Os painéis no Blue não são apenas para visualização pessoal; eles podem ser compartilhados com membros da equipe dentro da plataforma. Você pode conceder permissões de edição ou visualização. Qualquer pessoa com acesso a um painel pode compartilhá-lo com outros usuários.

::callout
 ---

icon: lightbulb
---

 As permissões de compartilhamento são escalonadas de acordo com as permissões do usuário; por exemplo, usuários com permissões de visualização podem convidar outros para visualizar o painel, garantindo que os direitos de acesso sejam mantidos e respeitados.
::

Atualmente, os painéis do Blue são projetados para colaboração interna e não podem ser compartilhados externamente com usuários não Blue via links públicos. No entanto, isso está planejado para um lançamento futuro. Se você tiver mais ideias para painéis, pode sempre [participar do nosso fórum comunitário](https://ask.blue.cc/) ou entrar em contato diretamente conosco em [support@blue.cc](support@blue.cc)

## Filtros

Os painéis do Blue fornecem capacidades de filtragem sofisticadas, permitindo segmentação precisa de dados por projeto, lista, tag e período de tempo. Listas e tags que compartilham o mesmo nome são inteligentemente consolidadas entre projetos, simplificando o processo de filtragem de dados que abrange várias iniciativas. Esse design inteligente permite uma análise cruzada de projetos sem costura, garantindo insights abrangentes e relevantes.

## Interface Interativa e Personalizável

- **Funcionalidade de Arrastar e Soltar**: Personalizar o layout do seu painel é intuitivo com o recurso de arrastar e soltar do Blue. Você pode reposicionar rapidamente gráficos dentro de um painel para atender às suas preferências de visualização ou para destacar as informações mais críticas.
- **Flexibilidade de Exibição**: O Blue entende que a visualização de dados não é uma solução única para todos. É por isso que ele fornece várias opções de exibição para todos os gráficos e representações de dados nos painéis. Se você precisa de um gráfico de pizza para detalhar a distribuição de tarefas ou um gráfico de linha para acompanhar o progresso ao longo do tempo, você tem a flexibilidade de escolher o formato que melhor transmite seus dados.

## Excluindo Painéis

Para excluir um painel, basta clicar com o botão direito no painel na lista e pressionar excluir. Observe que você deve ser o proprietário do painel — a pessoa que criou o painel — para poder excluí-lo.
