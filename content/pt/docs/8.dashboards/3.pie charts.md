Gráficos de pizza no Blue fornecem um gráfico circular intuitivo para visualizar dados proporcionais em registros categorizados. Esses gráficos ilustram magnitudes e relações relativas dentro de conjuntos de dados usando fatias coloridas e segmentadas.

## Criando Gráfico de Pizza

### Definir Título

Ao criar um gráfico de pizza, a primeira coisa a fazer é definir um título para sua pizza e, em seguida, puxar dados para o valor de suas fatias. Fatias individuais podem conter subgrupos empilhados, permitindo uma análise em múltiplas camadas em um único gráfico. Por exemplo, empilhe divisões regionais e de canal dentro de cada fatia de produto.

![Gráfico de Pizza 1](/docs/dashboard/pieChart/PieChart_1.png "Pie Chart 1")

### Criar Valores ou Múltiplos Valores.

Você pode criar várias fatias por gráfico que formarão seu círculo. Cada fatia deve puxar dados de campos personalizados de outros projetos para fazer sentido. Se você estiver selecionando dados do [campo personalizado "Moeda"](/docs/custom-fields/currency), o restante das fatias deve puxar os dados do mesmo conjunto. Isso permite que você compare os mesmos dados entre projetos em um único gráfico de pizza.

Os valores no Gráfico de Pizza podem ser derivados de um projeto específico, permitindo que você faça o seguinte:

- Analisar a composição de vendas entre regiões, canais, mercados, etc.
- Dividir a distribuição de tarefas ou problemas por categoria, prioridade, responsável, etc.
- Compreender a progressão das etapas do fluxo de trabalho e gargalos
- Comparar receitas ou índices de lucro entre produtos, campanhas, setores, etc.
- Acompanhar métricas de diversidade e inclusão entre equipes e níveis

![Gráfico de Pizza 2](/docs/dashboard/pieChart/PieChart_2.png "Pie Chart 2")

Para garantir que os valores em seu Gráfico de Pizza sejam o mais relevantes e direcionados possível, o Blue oferece opções robustas de filtragem — [qualquer filtro](/docs/views/filters) em qualquer visualização de projeto está disponível para uso!

### Adicionar Lógica

No Blue, a funcionalidade de lógica dentro dos cartões estatísticos é o motor para calcular e apresentar insights baseados em dados. A lógica permite que você aproveite o poder dos dados por meio de fórmulas e operações personalizadas, transformando dados brutos do projeto em informações acionáveis.

Comece referenciando valores específicos dentro do seu projeto digitando '{'. Essa ação trará uma lista de valores disponíveis que você pode incluir em sua fórmula de lógica.

![Gráfico de Pizza 3](/docs/dashboard/pieChart/PieChart_3.png "Pie Chart 3")

Você pode aprimorar sua lógica com várias funções digitando '='. O Blue fornece um conjunto de funções, como:

- **MÉDIA**: Calcular a média de um conjunto de números.
- **MÉDIAA**: Determinar a média, tratando texto como zero.
- **CONTAR**: Contar o número de entradas em um campo numérico.
- **CONTARA**: Contar o número de valores não vazios.
- **MÁXIMO**: Encontrar o valor máximo em um conjunto.
- **MÍNIMO**: Identificar o valor mínimo em um conjunto.
- **MENOS**: Subtrair um valor de outro.
- **MULTIPLICAR**: Multiplicar dois valores.
- **SOMA**: Somar uma série de números.
