## Visão Geral

As etiquetas são uma maneira poderosa de categorizar e organizar registros no Blue. O sistema de etiquetas fornece funcionalidade abrangente para criar, gerenciar e filtrar etiquetas, com suporte para operações em massa e filtragem avançada.

## Listar Etiquetas

Recupere etiquetas de projetos com opções flexíveis de filtragem e ordenação.

### Exemplo Básico

```graphql
query ListTags {
  tagList(
    filter: { 
      projectIds: ["project-123"] 
      excludeArchivedProjects: false 
    }
    first: 50
    orderBy: title_ASC
  ) {
    items {
      id
      uid
      title
      color
      project {
        id
        name
      }
      createdAt
      updatedAt
    }
    pageInfo {
      totalPages
      totalItems
      page
      perPage
      hasNextPage
      hasPreviousPage
    }
    totalCount
  }
}
```

### Exemplo de Filtragem Avançada

```graphql
query FilterTags {
  tagList(
    filter: {
      projectIds: ["project-123", "project-456"]
      search: "urgent"
      colors: ["#ff0000", "#ffaa00"]
      excludeArchivedProjects: true
    }
    orderBy: createdAt_DESC
    first: 20
  ) {
    items {
      id
      title
      color
      todos {
        id
        title
      }
    }
    totalCount
  }
}
```

## Criar Etiquetas

Crie novas etiquetas dentro de um projeto.

```graphql
mutation CreateTag {
  createTag(
    input: {
      title: "High Priority"
      color: "#ff0000"
    }
  ) {
    id
    uid
    title
    color
    project {
      id
      name
    }
    createdAt
  }
}
```

## Atualizar Etiquetas

Modifique as propriedades das etiquetas existentes.

```graphql
mutation UpdateTag {
  editTag(
    input: {
      id: "tag-123"
      title: "Critical Priority"
      color: "#cc0000"
    }
  ) {
    id
    title
    color
    updatedAt
  }
}
```

## Excluir Etiquetas

Remova etiquetas do sistema. Isso também removerá a etiqueta de todos os registros associados.

```graphql
mutation DeleteTag {
  deleteTag(id: "tag-123")
}
```

## Etiquetar Registros

Associe etiquetas a registros (todos) usando etiquetas existentes ou crie novas.

### Usando Etiquetas Existentes

```graphql
mutation TagRecord {
  setTodoTags(
    input: {
      todoId: "todo-123"
      tagIds: ["tag-456", "tag-789"]
    }
  )
}
```

### Criar Etiquetas Durante a Etiquetagem

Você pode criar novas etiquetas instantaneamente fornecendo títulos de etiquetas:

```graphql
mutation TagRecordWithNewTags {
  setTodoTags(
    input: {
      todoId: "todo-123"
      tagIds: ["existing-tag-id"]
      tagTitles: ["New Tag", "Another New Tag"]
    }
  )
}
```

## Sugestões de Etiquetas por IA

Obtenha sugestões de etiquetas impulsionadas por IA para um registro:

```graphql
mutation GetTagSuggestions {

      todoId: "todo-123"
      suggestionsCount: 5
    }
  ) {
    suggestions {
      title

```

## Parâmetros de Entrada

### TagListFilter

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `projectIds` | [String!] | Não | Filtrar etiquetas por IDs ou slugs de projetos |  |
| `excludeArchivedProjects` | Boolean | Não | Excluir etiquetas de projetos arquivados |  |
| `search` | String | Não | Pesquisar títulos de etiquetas (sem distinção entre maiúsculas e minúsculas) |  |
| `titles` | [String!] | Não | Filtrar por títulos de etiquetas específicos |  |
| `colors` | [String!] | Não | Filtrar por cores específicas (formato hex) |  |
| `tagIds` | [String!] | Não | Filtrar por IDs de etiquetas específicas |  |

### CreateTagInput

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `title` | String | Não | Título da etiqueta (gerado automaticamente se não fornecido) |  |
| `color` | String! | ✅ Sim | Cor da etiqueta em formato hex (por exemplo, "#ff0000") |  |

### EditTagInput

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `id` | String! | ✅ Sim | ID da etiqueta a ser atualizada |  |
| `title` | String | Não | Novo título da etiqueta |  |
| `color` | String | Não | Nova cor da etiqueta em formato hex |  |

### SetTodoTagsInput

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `todoId` | String! | ✅ Sim | ID do registro a ser etiquetado |  |
| `tagIds` | [String!] | Não | IDs de etiquetas existentes a serem aplicadas |  |
| `tagTitles` | [String!] | Não | Títulos de novas etiquetas a serem criadas e aplicadas |  |

### TagOrderByInput Valores

| Valor | Descrição |
|-------|-----------|
| `id_ASC` / `id_DESC` | Ordenar por ID da etiqueta |  |
| `uid_ASC` / `uid_DESC` | Ordenar por identificador único |  |
| `title_ASC` / `title_DESC` | Ordenar alfabeticamente por título |  |
| `color_ASC` / `color_DESC` | Ordenar por valor de cor |  |
| `createdAt_ASC` / `createdAt_DESC` | Ordenar por data de criação |  |
| `updatedAt_ASC` / `updatedAt_DESC` | Ordenar por data da última atualização |  |

## Campos de Resposta

### Tipo de Etiqueta

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `id` | ID! | Identificador único para a etiqueta |  |
| `uid` | String! | Identificador único amigável ao usuário |  |
| `title` | String! | Título/nome da etiqueta |  |
| `color` | String! | Cor da etiqueta em formato hex |  |
| `project` | Project! | Projeto ao qual esta etiqueta pertence |  |
| `todos` | [Todo!]! | Registros associados a esta etiqueta |  |
| `createdAt` | DateTime! | Quando a etiqueta foi criada |  |
| `updatedAt` | DateTime! | Quando a etiqueta foi atualizada pela última vez |  |

### Resposta de Paginação de Etiquetas

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `items` | [Tag!]! | Array de registros de etiquetas |  |
| `pageInfo` | PageInfo! | Informações de paginação |  |
| `totalCount` | Int! | Número total de etiquetas que correspondem aos filtros |  |

### Campos de Informações da Página

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `totalPages` | Int | Número total de páginas |  |
| `totalItems` | Int | Número total de itens em todas as páginas |  |
| `page` | Int | Número da página atual |  |
| `perPage` | Int | Número de itens por página |  |
| `hasNextPage` | Boolean! | Se há uma próxima página |  |
| `hasPreviousPage` | Boolean! | Se há uma página anterior |  |

| Ação | Função Necessária |

| Edit tags | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |

### Visibilidade das Etiquetas

As etiquetas são visíveis para todos os usuários que têm acesso ao projeto onde a etiqueta foi criada. O filtro baseado em função pode ser aplicado para certas operações.

## Melhores Práticas

1. **Use Cores Consistentes**: Estabeleça um sistema de codificação de cores para sua equipe
2. **Títulos Descritivos**: Use nomes de etiquetas claros e descritivos que os membros da equipe entenderão
3. **Operações em Lote**: Use `setTodoTags` para aplicar várias etiquetas de uma vez
4. **Pesquisar e Filtrar**: Use a funcionalidade de pesquisa para encontrar etiquetas existentes antes de criar novas
5. **Gerenciamento de Etiquetas**: Revise e limpe regularmente etiquetas não utilizadas
6. **Convenções de Cores**: Considere usar cores padrão para tipos comuns de etiquetas (vermelho para urgente, verde para concluído, etc.)

## Recursos Avançados

### Padrões de Criação de Etiquetas

- Novas etiquetas criadas via `setTodoTags` com `tagTitles` padrão para a cor `#4a9fff`
- Etiquetas sem cores especificadas retornam para `#00a0d2`
- Títulos de etiquetas são automaticamente removidos de espaços em branco

### Integração de Automação

As etiquetas podem ser:
- Aplicadas automaticamente por regras de automação
- Usadas como gatilhos para fluxos de trabalho de automação
- Modificadas através de ações de automação

### Atualizações em Tempo Real

Mudanças nas etiquetas são transmitidas via assinaturas GraphQL, permitindo atualizações em tempo real em clientes conectados.

## Notas Importantes

- Excluir uma etiqueta a remove de todos os registros associados
- As etiquetas são limitadas ao projeto e não podem ser compartilhadas entre projetos
- Os valores de cor devem ser códigos hexadecimais válidos (por exemplo, "#ff0000", "#00aa00")
- Os títulos das etiquetas são sensíveis a maiúsculas e minúsculas
- Um máximo de 100 etiquetas pode ser aplicado a um único registro
- As etiquetas suportam pesquisa de texto completo em títulos
