A API Blue fornece acesso completo à plataforma de gerenciamento de processos da Blue através de uma interface moderna [GraphQL](https://graphql.org/). Confiada por **mais de 17.000 clientes** processando **bilhões de solicitações de API anualmente**, nossa API alimenta fluxos de trabalho críticos com **[99,99% de disponibilidade](/platform/status)**.

## Por que a API Blue?

- **Cobertura de Recursos 100%** - Tudo que você pode fazer na interface da Blue, você pode fazer via API
- **Pronto para Empresas** - Testado em larga escala com limitação de taxa e segurança abrangentes
- **Amigável para Desenvolvedores** - GraphQL fornece exatamente os dados que você precisa, nada mais
- **Atualizações em Tempo Real** - Assinaturas WebSocket para sincronização instantânea de dados

## Início Rápido

Pronto para fazer sua primeira chamada de API? Veja como é simples:

```graphql
query MyProjects {
  projectList(filter: { companyIds: ["your-company-id"] }) {
    items {
      id
      name
      updatedAt
    }
  }
}
```

**[Experimente em nosso Playground GraphQL →](https://api.blue.cc/graphql)**

## Casos de Uso Populares

### Gerenciamento de Dados
- Use a Blue como um banco de dados flexível para aplicativos personalizados
- Sincronize dados entre a Blue e seus sistemas existentes
- Crie painéis personalizados e ferramentas de relatórios

### Automação de Fluxo de Trabalho
- Crie registros a partir de gatilhos externos (formulários, webhooks, e-mails)
- Atualize múltiplos projetos com base na lógica de negócios
- Automatize atribuições de tarefas e notificações

### Desenvolvimento de Integração
- Conecte a Blue com CRMs, ERPs e ferramentas especializadas externas
- Crie integrações personalizadas para sua indústria
- Amplie as capacidades da Blue com seus próprios recursos
