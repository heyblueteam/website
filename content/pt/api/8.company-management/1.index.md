---
title: Criar uma Empresa
description: Como criar uma nova empresa usando a API Blue.
---

Para criar uma nova empresa, você pode usar a seguinte mutação:

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

Observe que criar novas empresas pode afetar a cobrança da sua conta Blue, pois cada empresa é cobrada separadamente.

Aqui estão as opções para a mutação `createCompany` e suas descrições:

| Opção | Tipo | Descrição |
|-------|------|-----------|
| name | String | O nome da nova empresa. Este é um campo obrigatório. |  |
| slug | String | Uma versão amigável para URL do nome da empresa. Isso é usado na URL da empresa e deve ser único. Este é um campo obrigatório e também servirá como o identificador único para a empresa. |
