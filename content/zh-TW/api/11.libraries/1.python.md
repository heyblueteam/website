我們提供了一個 [Python 函式庫](https://pypi.org/project/bluepm/)，可以用來與 Blue API 互動。這個函式庫對 Blue API 提供 100% 的覆蓋，並在任何現代 IDE 中具備類型安全和代碼補全功能。

您可以通過運行以下命令來安裝：

```bash
pip install bluepm
```

然後，您可以在代碼中導入並使用該函式庫：

```python [python]
from bluepm import BlueAPIClient
```

您可以將身份驗證詳細信息設置為環境變量。

要使用環境變量，在運行腳本之前設置以下內容：

- BLUE_TOKEN_ID: 您的 Blue API 令牌 ID
- BLUE_SECRET_ID: 您的 Blue API 秘密 ID
- BLUE_COMPANY_ID: 您的 Blue 公司 ID（可選）
- BLUE_PROJECT_ID: 您的 Blue 項目 ID（可選）

或者，將它們直接傳遞給 BlueAPIClient 構造函數：

```python [python]
client = BlueAPIClient(
    token_id="your_token_id",
    secret_id="your_secret_id",
    company_id="your_company_id" #optional
    project_id = "your_project_id" #optional
)
```

完成這些步驟後，您可以使用該函式庫與 Blue API 互動。我們有一些輔助工具來簡化常見任務，但整個 API 都是公開的，以便您進行所需的任何互動。

例如，要獲取您有權訪問的所有項目的列表，您可以使用以下代碼：

```python [python]
projects = client.projects.list(company_id)
```

要獲取某個項目的列表，您可以使用以下代碼：

```python [python]
todo_lists = client.get_todo_lists(project_id)
```
