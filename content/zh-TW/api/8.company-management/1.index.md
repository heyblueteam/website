---
title: 創建公司
description: 如何使用 Blue API 創建新公司。
---

要創建新公司，您可以使用以下突變：

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

請注意，創建新公司可能會影響您 Blue 帳戶的計費，因為每家公司都是單獨計費的。

以下是 `createCompany` 突變的選項及其描述：

| 選項 | 類型 | 描述 |
|--------|------|-------------|
| name | String | 新公司的名稱。這是一個必填欄位。 |  |
| slug | 字串 | 公司名稱的 URL 友好版本。這用於公司的 URL，並且應該是唯一的。這是一個必填欄位，並且將作為公司的唯一標識符。 |
