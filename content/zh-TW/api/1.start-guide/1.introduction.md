Blue API 通過現代的 [GraphQL](https://graphql.org/) 接口提供對 Blue 的流程管理平台的完整訪問。受到 **17,000+ 客戶** 的信任，每年處理 **數十億個 API 請求**，我們的 API 為關鍵任務工作流程提供支持，並擁有 **[99.99% 的正常運行時間](/platform/status)**。

## 為什麼選擇 Blue API？

- **100% 功能覆蓋** - 您在 Blue 的 UI 中可以做的所有事情，都可以通過 API 完成
- **企業級準備** - 在大規模環境中經過實戰考驗，具備全面的速率限制和安全性
- **開發者友好** - GraphQL 提供您所需的準確數據，沒有多餘
- **實時更新** - WebSocket 訂閱實現即時數據同步

## 快速入門

準備好進行您的第一次 API 調用了嗎？這是多麼簡單：

```graphql
query MyProjects {
  projectList(filter: { companyIds: ["your-company-id"] }) {
    items {
      id
      name
      updatedAt
    }
  }
}
```

**[在我們的 GraphQL Playground 中試試 →](https://api.blue.cc/graphql)**

## 常見用例

### 數據管理
- 將 Blue 作為自定義應用程序的靈活數據庫
- 在 Blue 和您現有系統之間同步數據
- 建立自定義儀表板和報告工具

### 工作流程自動化
- 從外部觸發器（表單、網絡釘、電子郵件）創建記錄
- 根據業務邏輯更新多個項目
- 自動分配任務和通知

### 集成開發
- 將 Blue 與外部 CRM、ERP 和專業工具連接
- 為您的行業構建自定義集成
- 使用您自己的功能擴展 Blue 的能力
