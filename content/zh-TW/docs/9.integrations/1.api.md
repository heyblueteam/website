Blue 擁有一個 API（應用程式介面），因此如果需要，您可以建立自己的整合。

**Blue 擁有 100% 的 API 覆蓋**。這意味著您在 [我們的任何應用程式](/docs/start-guide/download-apps) 中可以執行的任何操作，您都可以以程式方式執行。我們擁有一個強大的 [GraphQL](https://graphql.org/) API，擁有完整的遊樂場，您可以在其中測試查詢，並且有完整的文檔。

::callout
