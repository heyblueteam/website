Blue 中的記錄具有靈活性，可用於多種用途，例如追蹤客戶互動、銷售、專案里程碑或庫存。這種適應性讓您可以自訂 Blue 以滿足您的業務需求，使其成為組織重要資訊的寶貴工具。

## 重新命名記錄

您可以輕鬆地在 Blue 中重新命名記錄以符合您的需求。只需前往專案設定，將名稱更改為您喜歡的任何名稱，例如任務或分配。此更改將套用於整個專案。

以下是您可以用於記錄的一些名稱範例：

- 任務 / 任務
- 分配 / 分配
- 步驟 / 步驟
- 問題 / 問題
- 工單 / 工單
- 潛在客戶 / 潛在客戶
- 交易 / 交易
- 案例 / 案例
- 商機 / 商機
- 客戶 / 客戶
- 產品 / 產品
- 訂單 / 訂單

請記住要包含您所選名稱的單數和複數形式。這有助於系統根據上下文顯示正確的術語。

自訂記錄名稱讓您可以使用適合您的產業或個人偏好的術語，使平台對您的團隊來說更簡單、更直觀。

記錄可以代表各種實體，例如：

- 客戶互動
