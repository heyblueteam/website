## 概述

封面圖片讓您能夠透過直接在記錄卡片上顯示圖片預覽，為您的記錄增添視覺化情境。此功能會自動從記錄內的各種來源擷取圖片並以縮圖形式顯示，讓您能夠輕鬆識別並整理視覺化內容。

無論您是在管理設計專案、內容日曆、產品目錄，或任何涉及視覺資產的工作流程，封面圖片都能幫助您快速瀏覽記錄並找到您需要的內容。此功能包含即時進度追蹤和彈性顯示選項，以符合您的特定工作流程需求。

## 主要功能

- **自動圖片擷取**：Blue 會自動從您選擇的來源尋找並顯示圖片
- **多重圖片來源**：從記錄描述、留言或自訂檔案欄位中擷取圖片
- **顯示調適選項**：選擇圖片以「覆蓋」或「包含」方式顯示
- **智慧選取**：設定是否使用來源中的第一張或最後一張圖片
- **即時更新**：當記錄內容變更時，封面圖片會自動更新
- **即時進度追蹤**：透過即時進度訂閱監控批次處理

## 運作方式

### 啟用封面圖片

封面圖片由專案管理員在專案層級進行設定：

1. 開啟您的專案並導航至**專案設定**
2. 選擇**記錄設定**
3. 找到**圖片封面選項**區段
4. 切換**啟用封面圖片**以啟動此功能

### 設定選項

#### 圖片來源
選擇 Blue 應在何處尋找圖片：
- **描述**：從記錄描述中擷取嵌入的圖片（預設）
- **留言**：從記錄留言中提取第一張/最後一張圖片
- **自訂欄位**：使用特定檔案類型自訂欄位中的檔案（支援任何檔案類型）

#### 圖片選取
當有多張圖片可用時，選擇要顯示的圖片：
- **第一張圖片**：使用找到的第一張圖片（預設）
- **最後一張圖片**：使用最近的圖片

#### 顯示調適
控制圖片在預覽區域中的顯示方式：
- **覆蓋**：圖片填滿整個預覽區域，可能會裁切邊緣以維持長寬比（預設）
- **包含**：在預覽區域內顯示完整圖片，可能會有空白區域

## 使用案例

**設計專案** - 直接在任務卡片上顯示模型和設計迭代，讓您能夠輕鬆查看每個設計任務的當前狀態，無需開啟個別記錄。

**內容規劃** - 對於內容日曆和編輯工作流程，顯示精選圖片或縮圖，讓您能夠一眼就看到內容管線的視覺化呈現。

**產品管理** - 在庫存記錄或功能需求上顯示產品照片，幫助團隊在討論期間快速識別項目。

**房地產** - 在房源記錄上顯示房產照片，讓經紀人能夠以視覺化方式快速瀏覽房產。

**活動規劃** - 在規劃任務上顯示場地照片、設計概念或活動佈局，以實現更好的視覺化組織。

## 設定封面圖片

### 步驟一：選擇您的圖片來源

考慮圖片在您的工作流程中自然存在的位置：
- 如果圖片是任務描述或簡報的一部分，請使用**描述**
- 如果圖片是透過團隊協作新增的，請使用**留言**
- 如果您有專門的圖片欄位，請使用**自訂欄位**

### 步驟二：設定顯示設定

1. 根據您的圖片類型選擇偏好的**圖片調適**：
   - 對於可接受裁切的照片和設計，使用**覆蓋**
   - 對於需要完整顯示的圖表或螢幕截圖，使用**包含**

2. 根據您的工作流程選擇**圖片選取**：
   - 對於主要圖片最初就新增的工作流程，使用**第一張**
   - 對於最近圖片最相關的工作流程，使用**最後一張**

### 步驟三：處理現有記錄

當您首次啟用封面圖片或變更設定時，Blue 會自動處理您的記錄：
1. 即時進度更新顯示處理狀態
2. 記錄以批次方式處理（每批 50 筆記錄），以獲得最佳效能
3. 您可以在處理於背景進行時繼續工作
4. 處理完成時清單會自動重新整理
5. 儲存設定變更時會自動觸發處理

## 技術細節

### 支援格式
- 使用描述或留言來源時：
  - JPEG/JPG 圖片
  - PNG 圖片（包含透明度）
  - GIF 圖片（靜態顯示）
- 使用自訂欄位來源時：
  - 可顯示任何檔案類型（圖片、PDF、文件）
  - 非圖片檔案顯示適當的檔案類型圖示
- 所有檔案必須上傳至 Blue（不支援外部 URL）

### 限制
- 每個專案最多 6,000 筆記錄可進行批次封面產生
- 只有檔案類型自訂欄位可作為來源使用
- 封面圖片僅供檢視，無法直接編輯

### 自動更新
在以下情況下會自動重新產生封面：
- 編輯記錄描述（如果使用描述來源）
- 新增包含圖片的留言（如果使用留言來源）
- 在自訂欄位中更新檔案（如果使用自訂欄位來源）
- 變更封面設定
- 修改顯示調適選項

### API 存取
封面圖片完全整合 GraphQL API：
- 透過 `setTodoCoverConfig` 突變進行設定
- 透過 Todo 類型的 `cover` 欄位存取封面資料
- 使用 `subscribeToCoverGenerationProgress` 訂閱監控進度
- 縮圖 URL 在 `/uploads/{uid}/thumbnail/cover` 自動產生


## 與其他功能的整合

封面圖片與其他 Blue 功能無縫協作：
- **[表單](/zh-TW/docs/projects/forms)**：提交的圖片可自動成為封面
- **[API](/zh-TW/docs/integrations/api)**：完整的 GraphQL API 支援設定和資料存取
- **[自訂欄位](/zh-TW/docs/projects/custom-fields)**：任何檔案類型欄位都可作為封面來源
