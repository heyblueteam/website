## Почему стоит использовать OTP и магические ссылки?

OTP и магические ссылки предоставляют две основные выгоды:

**Повышенная безопасность**

OTP и магические ссылки обеспечивают дополнительный уровень защиты, устраняя необходимость ввода реального пароля. Ваш пароль не может быть украден или подслушан при входе. Также многие пользователи имеют привычку использовать одни и те же пароли для различных сервисов. Проблема в том, что когда один сервис подвергается утечке данных, это создает проблемы безопасности во всех других сервисах.

**Удобство**

Необходимость запоминать длинные и сложные пароли может быть неудобной. OTP и магические ссылки позволяют быстро и просто получить доступ без необходимости искать пароли.

## Как это работает?

**Одноразовые пароли**

Когда вход с использованием OTP включен, Blue запросит у вас уникальный 4-значный код вместо вашего пароля. Этот код отправляется на вашу электронную почту и действителен только для одной попытки входа.

![Magic Link 1](/docs/MagicLink_1.png "Magic Link 1")

**Магические ссылки для входа**

Магические ссылки — это уникальные URL для входа, отправляемые вам по электронной почте. Щелчок по ссылке мгновенно войдет в вашу учетную запись Blue без необходимости ввода пароля. Ссылка истекает после одного использования.
