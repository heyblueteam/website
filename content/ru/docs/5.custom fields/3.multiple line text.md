Поле "Многострочный текст" позволяет пользователям вводить длинный многоабзацный текстовый контент в записи Blue. Оно предоставляет:

- Без ограничений по символам, неограниченная длина текста
- Поддержка форматирования, такого как жирный, курсив, гиперссылки
- Переносы строк, интервалы, выравнивание текста

![Многострочный текст](/docs/MultipleLineText.png "Многострочный текст")

**Случаи использования**

Это поле идеально подходит для ситуаций, таких как:

- Описания - спецификации продуктов, детали задач, области проектов
- Заметки и инструкции
- Сообщения - обновления, объяснения
- Контент - статьи, блоги, руководства
- Резюме и отчеты
- Содержимое электронных писем
- Отзывы и рецензии

По сути, любой тип длинного текстового контента может быть введен.
