Настраиваемое поле "Множественный выбор" функционирует идентично [настраиваемому полю "Одиночный выбор"](/ru/docs/custom-fields/single-select), с одним ключевым отличием — возможностью делать несколько выборов для каждой записи.

Настраиваемое поле "Множественный выбор" позволяет создать выпадающий список опций, из которых пользователи могут выбирать при вводе данных в записи. Вы можете определить до 10 000 опций с метками и цветами.

![MultipleSelect 1](/docs/MultipleSelect_1.png "MultipleSelect 1")

Выбор цветов для опций в настраиваемом поле "Множественный выбор"

Возможность множественного выбора идеально подходит для захвата:

- Таксономий - Классификация записей по нескольким категориям (например, пометка записей всеми релевантными метками)
- Составных данных - Комбинирование опций для описания сложных записей (например, выбор всех применимых свойств продукта)
- Аудиторских следов - Отслеживание изменений во времени (например, логирование обновлений статуса записи)

::callout
---

icon: lightbulb
---

Добавление опций одну за другой может стать утомительным для выпадающих списков выбора. К счастью, опции можно добавлять массово, копируя и вставляя из маркированного/разделенного строками списка или столбца электронной таблицы. Просто скопируйте список вариантов и вставьте их в диалог создания опций в Blue. Это автоматически создаст опцию для каждой строки или маркера, упростив настройку.
::

Гибкость множественного выбора расширяет возможности настраиваемых выпадающих списков в структурировании данных проекта.

Интерфейс поиска при вводе делает поиск идеального выбора простым, даже среди тысяч опций.

![MultipleSelect 2](/docs/MultipleSelect_2.png "MultipleSelect 2")

После создания администраторы проекта могут по-прежнему редактировать список опций по мере необходимости, включая переупорядочивание, добавления или удаления.

Поле "Множественный выбор" обеспечивает настроенный выпадающий список, предназначенный для быстрого выбора из обширных предопределенных опций. Определение настроенных опций и меток тесно выравнивает поле с вашими рабочими процессами и терминологией. Эти выпадающие списки тесно интегрируются в Blue. [Администраторы проекта](/ru/docs/user-management/roles/project-administrator) могут ограничить доступ к просмотру и редактированию по [роли пользователя](/ru/docs/user-management/roles/custom-user-roles), внешние [формы](/ru/docs/projects/forms) могут использовать "Множественный выбор" для стандартизированного ввода, а [панели управления](/ru/docs/dashboards) могут фильтровать по значениям выбора. Будь то сбор категориальных данных или стандартизация терминологии, поле "Множественный выбор" обеспечивает оптимизированный опыт для управления настроенными выпадающими списками, выровненными с потребностями вашего бизнеса.
