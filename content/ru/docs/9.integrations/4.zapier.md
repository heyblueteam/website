Zapier — это платформа интеграции, которая помогает соединять различные веб-приложения и автоматизировать рабочие процессы.

**Сценарии использования**

Вот некоторые из самых популярных способов интеграции Blue с другими приложениями через Zapier:

**_Синхронизация записей Blue с Google Sheets_**

По мере создания или обновления записей Blue автоматически добавляйте их в качестве новых строк в таблицу Google для централизованной отчетности.

**_Отправка подтверждения заказа из записей Blue_**

Когда вы выбираете определенное настраиваемое поле в Blue, автоматически отправляйте электронное письмо для подтверждения заказа вашему клиенту.

**_Добавление комментариев Blue в Slack_**

Получайте уведомления Blue в реальном времени, публикуемые в нужных каналах Slack, чтобы ваша команда была в курсе важных обновлений записей.

## Ключевые преимущества

**Автоматизация между инструментами**: Zaps, созданные в Zapier, могут автоматически отправлять и получать данные между Blue и тысячами других приложений. Это устраняет ручную работу по перемещению данных между системами.

**Интеграции в реальном времени**: Используя триггеры вебхуков Blue, Zaps мгновенно реагируют на события в Blue, такие как новая запись, добавленный комментарий, изменение статуса и т. д. Это позволяет осуществлять синхронизацию в реальном времени и автоматизацию процессов.

**Простота создания и поддержки**: Zaps предоставляют удобный способ интеграции инструментов без кода. Не требуется разработка. И они продолжают работать без постоянного обслуживания.

1. Войдите в свою учетную запись Blue в настройках профиля

3. Нажмите "Добавить вебхук"
4. Дайте вебхуку имя и введите URL вебхука Zapier
5. Выберите, какие события должны запускать вебхук
6. Нажмите "Создать вебхук"

## Blue в Zapier

1. Войдите в свою учетную запись Zapier
2. Нажмите "Создать Zap"  
3. На первом шаге найдите и выберите [приложение-триггер "Webhooks by Zapier"](https://zapier.com/apps/webhook)
4. Настройте его для приема полезных нагрузок из вашего URL вебхука Blue
5. На втором шаге найдите и выберите целевое приложение для отправки данных
6. Сопоставьте поля полезной нагрузки JSON с входными полями целевого приложения
7. Включите Zap


Теперь, когда указанные события происходят в Blue, полезная нагрузка вебхука будет отправлена в Zapier, который автоматически передает данные в ваше интегрированное приложение согласно настройкам Zap.

## Zapier в Blue

Чтобы отправить данные из других приложений в Blue, выполните следующие шаги:

**1. Получите учетные данные API Blue**
- Перейдите в Настройки Blue > Токены API
- Создайте новый токен или используйте существующие учетные данные:
  - Идентификатор токена
  - Секрет токена
  - Идентификатор компании
  - Идентификатор проекта

**2. Создайте Zap в Zapier**
1. Настройте ваше приложение-триггер (например, Google Sheets, Slack и т. д.)
2. Добавьте "Webhooks by Zapier" в качестве приложения действия
3. Выберите тип события "Пользовательский запрос"

**3. Настройте вебхук**
- Метод: POST
- URL: `https://api.blue.cc/graphql` (или бета-эндпоинт, если необходимо)
- Передача данных: ЛОЖЬ
- Развернуть: НЕТ

**4. Установите заголовки:**
```json
{
  "Content-Type": "application/json",
  "X-Bloo-Token-ID": "Your_Token_ID",
  "X-Bloo-Token-Secret": "Your_Token_Secret", 
  "X-Bloo-Company-ID": "Your_Company_ID",
  "X-Bloo-Project-ID": "Your_Project_ID"
}
```

**5. Настройте запрос (пример мутации):**
```json
{
  "query": "mutation CreateTodo($input: CreateTodoInput!) {\n  createTodo(input: $input) {\n    id\n    title\n    customFields {\n      value\n    }\n  }\n}",
  "variables": {
    "input": {
      "todoListId": "your-todo-list-id",
      "title": "New task from Zapier",
      "customFields": [
        {
          "customFieldId": "priority-field-id",
          "value": "High"
        },
        {
          "customFieldId": "due-date-field-id",
          "value": "2024-03-01"
        }
      ]
    }
  }
}
```

**6. Тестирование и активация**
- Сначала протестируйте мутации с помощью API Playground Blue
- Сопоставьте динамические значения из триггерных приложений, используя поля данных Zapier
- Включите Zap после успешного тестирования
