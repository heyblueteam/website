Blue позволяет использовать Google Карты для отображения карты вместо стандартной OpenStreetMap. Google Карты часто имеют более детализированные карты определенных районов и более точные данные о местоположении. Однако для этого требуется ключ API Google Карты, который должен быть введен в настройках компании владельцем компании.

Когда вы переключаетесь на Google Карты, это влияет на все представления карты в Blue, включая карту компании, [представление карты в проектах](/docs/views/map) и [настраиваемое поле местоположения](/docs/custom-fields/location-map).

Чтобы использовать Google Карты в Blue, необходимо включить следующие API в Google Cloud Console:
- [API тайлов карт](https://console.cloud.google.com/apis/library/tile.googleapis.com): Это необходимо для отображения тайлов карты.
- [API мест](https://console.cloud.google.com/apis/library/places.googleapis.com): Это необходимо для поиска мест и получения их координат.
- [API геокодирования](https://console.cloud.google.com/apis/library/geocoding-backend.googleapis.com): Это необходимо для получения координат места.

Вам нужно будет зарегистрироваться для получения [учетной записи Google Cloud](https://console.cloud.google.com/) и создать проект, чтобы получить ключи API, следуя инструкциям [здесь](https://developers.google.com/maps/documentation/embed/get-api-key).

## Ограничение использования ключа API

В целях безопасности важно ограничить использование вашего ключа API Google Карты только для авторизованных доменов. Это предотвращает несанкционированное использование вашего ключа API, если он будет раскрыт. Вы должны ограничить ключ API для работы только с:

- app.blue.cc - Основное приложение Blue
- forms.blue.cc - Для публичных форм
- Вашим собственным доменом, если вы используете [white label](/docs/start-guide/white-label)

Чтобы ограничить ваш ключ API:

1. Перейдите на страницу [Учетные данные Google Cloud Console](https://console.cloud.google.com/apis/credentials)
2. Найдите ваш ключ API и нажмите "Редактировать" (значок карандаша)
3. В разделе "Ограничения приложения" выберите "HTTP-рефереры (веб-сайты)"
4. Добавьте следующие рефереры:
   - `*.app.blue.cc/*`
   - `*.forms.blue.cc/*`
   - `*.yourdomain.com/*` (если используется white label)
5. Нажмите "Сохранить"

**Важное примечание о расходах на Google Карты**<br><br>

Google будет взимать с вас плату за использование API Google Карты отдельно от вашей подписки на Blue. Все расходы, затраты и сборы, связанные с вашей учетной записью Google Карты и использованием API, являются исключительной ответственностью администратора организации. Blue не несет ответственности за любые расходы, возникшие в результате использования вами API Google Карты, включая, но не ограничиваясь, интеграциями с третьими сторонами, использованием API или затратами на хранение.
