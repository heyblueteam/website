Столбчатые графики предоставляют полезный способ визуально анализировать и сравнивать значения по категориям данных. Высота каждого столбца представляет величину значений в этой категории.

## Создание столбчатого графика

### Установить заголовок

Начните с установки описательного заголовка для вашего столбчатого графика. Четко укажите, какие данные отображаются.

![Столбчатый график 1](/docs/dashboard/barGraph/BarGraph_1.png "Bar Graph 1")

## Выбор типа создания столбцов

Решите, использовать ли ручное создание столбцов или автоматическое:

![Столбчатый график 2](/docs/dashboard/barGraph/BarGraph_2.png "Bar Graph 2")

## Ручное

Полностью настраивайте каждый столбец, включая источник данных, метку, цвет и логику. Позволяет достичь точности, но требует больше усилий.

### Выберите поле категории

Затем выберите поле, содержащее категории, которые будут представлены в виде столбцов. Общие варианты включают регион, продуктовую линию, статус приоритета и т.д. Это поле определяет количество столбцов.

### Выберите поле значения

Затем выберите числовое поле для определения высоты столбцов. Типичные варианты включают доход, распределение бюджета, количество записей и т.д. Значения из этого поля будут определять длину столбца.

![Столбчатый график 1](/docs/dashboard/barGraph/BarGraph_3.png "Bar Graph 3")

### Фильтр (по желанию)

Примените фильтры, чтобы сузить анализ до подмножества данных, если это необходимо. Можно добавить любые существующие фильтры представления проекта.

### Настройка визуальных элементов и анализ

Настройте визуальные параметры, такие как цвета, интервалы и метки. Затем извлеките инсайты, сравнивая производительность по категориям.

![Столбчатый график 4](/docs/dashboard/barGraph/BarGraph_4.png "Bar Graph 4")

## Автоматическое

Упрощенный процесс, при котором заголовок, категории и значения достаточно для генерации полного столбчатого графика всего за несколько кликов.

Вы также можете настроить автоматические столбцы в зависимости от ваших потребностей и добавить любые необходимые фильтры.

Столбчатые диаграммы преобразуют данные проекта в интуитивно понятные визуальные анализы, сосредотачиваясь на сравнении значений метрик по категориям в Blue.
