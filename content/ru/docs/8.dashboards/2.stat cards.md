Статистические карточки отображают рассчитанное в реальном времени значение из одного или нескольких проектов. Это отлично подходит для отслеживания ключевых статистических данных.

## Создание статистических карточек

### Установите заголовок

При создании статистической карточки первым делом необходимо установить заголовок. Название ваших статистических карточек — это важный шаг для ясности и удобства использования. Заголовок должен отражать ключевую метрику или инсайт, который представляет статистическая карточка, например, "Ежемесячные продажи", "Открытые задачи" или "Еженедельные активные пользователи".

![Статистическая карточка 2](/docs/dashboard/statCard/StatCard_2.png "Stat Card 2")

Хорошо определенный заголовок помогает пользователям быстро понять контекст и актуальность данных.

## Создание значений

Вы можете создать несколько значений для каждой статистической карточки, каждое из которых представляет собой измерение или метрику, важную для вашего проекта. Эта функция множественных значений позволяет создать комплексную панель инструментов, отражающую производительность проекта в реальном времени.

Значения на статистических карточках могут быть получены из конкретного проекта, что позволяет вам делать следующее:
- **Подсчет записей**: Рассчитать общее количество записей в проекте, предоставляя вам быстрый обзор объема задач или записей.
- **Расчеты по пользовательским полям**: Выполнять расчеты на основе данных, содержащихся в пользовательских полях. Например, вы можете суммировать значения из поля "Цена проекта", чтобы оценить общие бюджетные показатели.

Чтобы гарантировать, что значения на ваших статистических карточках будут максимально актуальными и целевыми, Blue предлагает мощные параметры фильтрации — [любой фильтр](/docs/views/filters) в любом представлении проекта доступен для использования!

## Добавить логику

В Blue функциональность логики в статистических карточках является движущей силой для расчета и представления основанных на данных инсайтов. Логика позволяет вам использовать мощь данных через настраиваемые формулы и операции, преобразуя сырые данные проекта в полезную информацию.

Начните с обращения к конкретным значениям в вашем проекте, набрав '{'. Это действие вызовет список доступных значений, которые вы можете включить в вашу логическую формулу.

Вы можете улучшить свою логику с помощью различных функций, набрав '='. Blue предоставляет набор функций, таких как:

- **AVERAGE**: Рассчитать среднее значение набора чисел.
- **AVERAGEA**: Определить среднее значение, рассматривая текст как ноль.
- **COUNT**: Подсчитать количество записей в числовом поле.
- **COUNTA**: Подсчитать количество непустых значений.
- **MAX**: Найти максимальное значение в наборе.
- **MIN**: Определить минимальное значение в наборе.
- **MINUS**: Вычесть одно значение из другого.
- **MULTIPLY**: Умножить два значения.
- **SUM**: Сложить ряд чисел.

Независимо от того, хотите ли вы определить максимальные продажи, рассчитать среднюю продолжительность задач или суммировать бюджетные allocations, логика в Blue делает это простым.

## Настроить отображение

Последний шаг в создании статистической карточки — это решение о том, как отображать данные.

Статистические карточки имеют три варианта отображения:

1. **Число**: Выберите простой числовой формат, с дополнительной функциональностью для установки уровня точности. Это означает, что вы можете указать, сколько знаков после запятой будет показано, что позволяет достичь точности, соответствующей уровню детализации, необходимому для вашего анализа данных.
2. **Валюта**: Выберите этот вариант, чтобы представить значения в денежном эквиваленте. Он идеально подходит для финансовых метрик, таких как отслеживание бюджета, доходы, расходы или другие финансовые статистики. Формат валюты применит соответствующее местное форматирование и символы валюты.
3. **Процент**: Используйте этот формат, чтобы выразить значения в процентах, идеально подходящий для отображения коэффициентов завершения, метрик роста или других пропорциональных данных. Этот вариант автоматически преобразует рассчитанное значение в процентный формат, обеспечивая мгновенное понимание соотношений и долей.
