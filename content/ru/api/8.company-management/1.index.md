---
title: Создать компанию
description: Как создать новую компанию с помощью Blue API.
---

Чтобы создать новую компанию, вы можете использовать следующую мутацию:

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

Обратите внимание, что создание новых компаний может повлиять на выставление счетов вашему аккаунту Blue, так как каждая компания выставляется на отдельный счет.

Вот параметры для мутации `createCompany` и их описания:

| Параметр | Тип | Описание |
|----------|-----|----------|
| name | String | Название новой компании. Это обязательное поле. |  |
| slug | Строка | Версия названия компании, удобная для URL. Она используется в URL компании и должна быть уникальной. Это обязательное поле и также будет служить уникальным идентификатором для компании. |
