---
title: Delete Project
description: How to delete projects in Blue.
---

To delete a project, you can use the following mutation:

```graphql
mutation {
  deleteProject(id: "{project-id}") {
    success
  }
}
``` 


::callout
---
icon: circle-inf---
**Project deletion is not reversible.** Once a project is deleted, all associated data, including forms, responses, and files, will be permanently removed from the system. Please ensure you have backed up any important information before proceeding with the deletion.
::
