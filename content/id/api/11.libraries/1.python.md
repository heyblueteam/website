Kami menyediakan [pustaka Python](https://pypi.org/project/bluepm/) yang dapat digunakan untuk berinteraksi dengan Blue API. Ini memiliki cakupan 100% dari Blue API dengan keamanan tipe dan penyelesaian kode di IDE modern mana pun.

Anda dapat menginstalnya dengan menjalankan:

```bash
pip install bluepm
```

Anda kemudian dapat mengimpor dan menggunakan pustaka ini dalam kode Anda:

```python [python]
from bluepm import BlueAPIClient
```

Anda dapat mengatur detail otentikasi Anda baik sebagai variabel lingkungan

Untuk menggunakan variabel lingkungan, atur yang berikut sebelum menjalankan skrip Anda:

- BLUE_TOKEN_ID: ID token API Blue Anda
- BLUE_SECRET_ID: ID rahasia API Blue Anda
- BLUE_COMPANY_ID: ID perusahaan Blue Anda (opsional)
- BLUE_PROJECT_ID: ID proyek Blue Anda (opsional)

Atau, langsung berikan kepada konstruktor BlueAPIClient:

```python [python]
client = BlueAPIClient(
    token_id="your_token_id",
    secret_id="your_secret_id",
    company_id="your_company_id" #optional
    project_id = "your_project_id" #optional
)
```

Setelah Anda melakukan ini, Anda dapat menggunakan pustaka untuk berinteraksi dengan Blue API. Kami memiliki beberapa pembantu untuk menyederhanakan tugas-tugas umum, tetapi seluruh API tersedia untuk memungkinkan interaksi apa pun yang Anda butuhkan.

Sebagai contoh, untuk mendapatkan daftar semua proyek yang dapat Anda akses, Anda dapat menggunakan kode berikut:

```python [python]
projects = client.projects.list(company_id)
```

Untuk mendapatkan daftar daftar untuk sebuah proyek, Anda dapat menggunakan kode berikut:

```python [python]
todo_lists = client.get_todo_lists(project_id)
```
