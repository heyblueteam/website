---
title: Buat Perusahaan
description: Cara membuat perusahaan baru menggunakan Blue API.
---

Untuk membuat perusahaan baru, Anda dapat menggunakan mutasi berikut:

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

Perlu dicatat bahwa membuat perusahaan baru dapat mempengaruhi penagihan akun Blue Anda, karena setiap perusahaan ditagih secara terpisah.

Berikut adalah opsi untuk mutasi `createCompany` dan deskripsinya:

| Opsi | Tipe | Deskripsi |
|--------|------|-------------|
| name | String | Nama perusahaan baru. Ini adalah bidang yang wajib diisi. |  |
| slug | String | Versi ramah URL dari nama perusahaan. Ini digunakan dalam URL perusahaan dan harus unik. Ini adalah bidang yang wajib diisi dan juga akan berfungsi sebagai pengidentifikasi unik untuk perusahaan. |
