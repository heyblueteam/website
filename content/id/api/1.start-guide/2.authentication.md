## Header

Sebelum menjalankan kueri apa pun, sangat penting untuk mengatur header untuk permintaan API Anda. Header ini tetap konsisten di semua permintaan, hanya memerlukan pengaturan satu kali. Mereka berfungsi untuk mengautentikasi setiap permintaan ke API.

Header yang diperlukan adalah Token-ID dan Secret-ID, yang masing-masing diwakili sebagai `x-bloo-token-id` dan `x-bloo-token-secret`.

Selain itu, permintaan tertentu mungkin memerlukan penyertaan Company-ID `x-bloo-company-id` dan Project-ID `x-bloo-project-id` untuk operasi yang lebih spesifik.

Anda dapat belajar cara membuat token dan rahasia dalam video berikut:

<youtube url="https://www.youtube.com/watch?v=C-q_AqdFUzE" />

## Token ID
Token ID adalah pengidentifikasi unik Anda untuk akses API.

Untuk mulai menggunakan API kami, klik pada profil Anda di sudut kanan atas, dan di bawah menu profil, Anda akan menemukan tab API.

![API 2](/docs/API_2.png "API 2")

Di tab API, klik tombol "Generate a Token":

![API 3](/docs/API_3.png "API 3")

Berikan nama untuk token dan atur tanggal kedaluwarsa jika diinginkan. Jika Anda mengatur tanggal kedaluwarsa, token API akan otomatis berhenti berfungsi setelah tanggal tersebut.

![API 4](/docs/API_4.png "API 4")

Setelah dihasilkan, Anda akan ditampilkan Token ID dan Secret ID. Token ID akan diawali dengan `pat_` untuk identifikasi yang mudah.

## Secret ID

Secret ID berfungsi sebagai kata sandi, memastikan komunikasi yang aman dengan API Blue.

Ingat, Secret ID hanya ditampilkan sekali untuk alasan keamanan, jadi simpan dengan aman untuk mempertahankan akses ke data Anda melalui API. Untuk keamanan maksimum, Blue menyimpan rahasia Anda menggunakan hashing bcrypt, yang berarti bahkan tim kami tidak dapat mengambil rahasia teks biasa Anda setelah dibuat.

![API 5](/docs/API_5.png "API 5")

::callout
 ---

icon: lightbulb

**Peringatan Keamanan** <br/> <br/>
Pastikan Anda menjaga Token ID dan Secret ID Anda tetap aman, karena siapa pun yang memiliki akses ke kredensial ini dapat mengakses data Anda di Blue.
::

## Company & Project IDs

Berikut adalah video yang menunjukkan cara menemukan company ID dan project ID Anda di Blue:

<youtube url="https://www.youtube.com/watch?v=zLEvs6zqGTc" />

Perusahaan adalah entitas tingkat atas di Blue. Company ID digunakan untuk mengidentifikasi perusahaan di Blue. Mereka digunakan dalam API untuk menentukan perusahaan mana yang akan berinteraksi. ID ini sama dengan slug perusahaan di URL saat Anda mengunjungi perusahaan di Blue:

```
app.blue.cc/company/{company-id}/
```

Untuk kueri dan mutasi tertentu, Anda perlu menyediakan Project ID. Ini adalah ID dari proyek yang ingin Anda interaksi. ID ini sama dengan slug proyek di URL saat Anda mengunjungi proyek di Blue:

```
app.blue.cc/company/{company-id}/project/{project-id}/
```
