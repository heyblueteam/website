Chat memungkinkan anggota untuk berkomunikasi satu sama lain dalam platform. Ini adalah cara untuk mendiskusikan topik, berbagi pembaruan, mengajukan pertanyaan, dan berkolaborasi.

::callout
---

icon: lightbulb
---

Perhatikan bahwa Blue mendukung chat berbasis topik dalam proyek, namun tidak mendukung pesan langsung ke individu.
::

## Membuat Chat

Untuk memulai chat baru:
1. Klik ikon Chat di sidebar kiri.
2. Klik tombol "New Chat" di bagian atas.
3. Klik "Create Chat" ketika siap.

Anda dapat membuat beberapa chat untuk mendiskusikan topik yang berbeda.

![Chat 1](/docs/Chat_1.png "Chat 1")

Gunakan tombol "+" untuk membuat chat baru dan atur nama topik Anda

## Berkomunikasi dalam Chat

Setelah berada dalam chat, anggota dapat:
- Mengirim pesan teks
- Mengunggah gambar dengan menyeret dan menjatuhkan file
- @mention anggota lain untuk memberi notifikasi
- Menyematkan video YouTube dengan menempelkan URL video

![Chat 2](/docs/Chat_2.png "Chat 2")

Unggah gambar dan jaga diskusi tetap dalam konteks

Chat memungkinkan diskusi yang lancar dan berkelanjutan tanpa thread email yang panjang.

## Fitur Chat

- Cari pesan chat
- Edit pesan sebelumnya
- Hapus pesan

Menggunakan chat Blue membuat percakapan penting dan file yang dibagikan tetap terorganisir di satu tempat. Dengan notifikasi terintegrasi dan mention, mereka memfasilitasi komunikasi tim yang mulus.
