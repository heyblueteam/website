Bidang kustom URL/Link memungkinkan Anda untuk menyimpan tautan dan alamat web dalam catatan Anda di Blue. Ketika URL dimasukkan ke dalam bidang ini, sistem akan memvalidasi bahwa itu adalah tautan yang terstruktur dengan benar.

![URL-Link 1](/docs/URLlink_1.png "URL-Link 1")

Bidang kustom URL/Link di Blue

Setiap catatan dapat berisi satu tautan, disimpan dalam satu bidang kustom URL/Link. Tautan tersebut dapat diklik, memungkinkan Anda untuk menavigasi langsung ke halaman web atau sumber daya dengan mengkliknya. Untuk kemudahan, tautan akan secara otomatis terbuka di tab browser baru.

![URL-Link 2](/docs/URLlink_2.png "URL-Link 2")

Anda dapat membuka URL di tab baru dengan satu klik

Bidang ini sangat berguna untuk menautkan ke dokumen berbasis cloud seperti Google Docs, file Word, spreadsheet Excel, dll. Menyimpan tautan ini memungkinkan akses satu klik ke file relevan yang terkait dengan setiap catatan.

Bidang URL/Link sangat baik untuk kasus penggunaan tambahan seperti:

- Menyimpan situs web pelanggan atau mitra
- Menangkap tautan yang terkait dengan catatan
- Membuat referensi ke halaman web

Dengan antarmuka yang efisien dan kemampuan tautan bawaan, bidang kustom URL/Link memungkinkan Anda untuk memperkaya catatan dengan sumber daya web kontekstual dan mengaksesnya secara instan dengan satu klik.
