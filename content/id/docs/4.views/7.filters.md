Filter terletak di antara tombol otomasi dan tag di bagian kanan atas papan. Filter tidak memengaruhi apa yang dilihat pengguna lain - filter hanya memengaruhi tampilan Anda.

## Filter Dasar

Di panel filter, Anda akan melihat beberapa kategori filter cepat:

- **Tag**: Pilih satu atau lebih tag untuk hanya menampilkan record dengan tag tersebut.
- **Penerima Tugas**: Pilih satu atau lebih penerima tugas untuk hanya menampilkan record yang ditugaskan kepada mereka.
- **Tanggal Jatuh Tempo**: Filter berdasarkan rentang tanggal jatuh tempo seperti jatuh tempo hari ini, terlambat, dll.
- **Record**: Sembunyikan Record yang sudah selesai, belum selesai

## Filter Lanjutan

Untuk mengakses filter lanjutan, navigasikan ke tampilan papan proyek Anda. Klik ikon filter di kanan atas (terlihat seperti corong). Ini akan membuka panel filter dan memungkinkan Anda mengatur filter berdasarkan field kustom.

Klik ikon "+" untuk menambahkan kriteria filter. Anda dapat memilih field kustom, operator (Sama dengan, Tidak Sama dengan, Mengandung, dll.) dan nilai untuk difilter.

Gunakan tombol AND/OR untuk menggabungkan beberapa kriteria filter dalam satu filter.

![Filters 2](/docs/Filters_2.png "Filters 2")

## Menumpuk Record yang Difilter

Setelah Anda menerapkan filter menggunakan Filter Cepat atau Filter Lanjutan, filter akan langsung menyaring record yang ditampilkan di tampilan proyek Anda. Anda dapat menambahkan beberapa filter untuk mempersempit hasil lebih lanjut.

Filter yang sedang aktif akan ditampilkan di bagian atas, sehingga Anda mengetahui filter apa yang diterapkan.

## Persistensi Filter

Filter apa pun yang Anda atur akan tetap aktif meskipun Anda menavigasi ke tampilan lain dan kembali ke tampilan papan nanti. Jadi Anda tidak perlu menerapkan ulang filter setiap saat.
