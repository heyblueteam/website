Zapier adalah platform integrasi yang membantu menghubungkan berbagai aplikasi web dan mengotomatiskan alur kerja.

**Kasus Penggunaan**

Berikut adalah beberapa cara paling populer Blue terintegrasi dengan aplikasi lain melalui Zapier:

**_Sinkronkan Rekaman Blue ke Google Sheets_**

Saat rekaman Blue dibuat atau diperbarui, secara otomatis tambahkan mereka sebagai baris baru di Google Sheet untuk pelaporan terpusat.

**_Kirim Konfirmasi Pesanan dari Rekaman Blue_**

Saat Anda memilih bidang kustom tertentu di Blue, secara otomatis kirim email untuk mengonfirmasi pesanan kepada pelanggan Anda.

**_Tambahkan Komentar Blue ke Slack_**

Dapatkan notifikasi Blue secara real-time yang diposting ke saluran Slack yang tepat sehingga tim Anda menyadari pembaruan rekaman yang penting.

## Manfaat Utama

**Automasi Antara Alat**: Zaps yang dibuat di Zapier dapat secara otomatis mengirim dan menerima data antara Blue dan ribuan aplikasi lainnya. Ini menghilangkan pekerjaan manual dalam memindahkan data antara sistem.

**Integrasi Real-Time**: Menggunakan pemicu webhook Blue, Zaps bereaksi secara instan terhadap peristiwa di Blue seperti rekaman baru, komentar ditambahkan, perubahan status, dll. Ini memungkinkan sinkronisasi real-time dan otomatisasi proses.

**Mudah Dibangun dan Dipelihara**: Zaps menyediakan cara yang ramah pengguna, tanpa kode untuk mengintegrasikan alat. Tidak diperlukan pekerjaan pengembangan. Dan mereka terus berfungsi tanpa pemeliharaan yang berkelanjutan.

1. Masuk ke akun Blue Anda di pengaturan profil
2. Pergi ke pengaturan profil dan klik "Webhook"
3. Klik "Tambahkan Webhook"
4. Beri nama webhook dan masukkan URL webhook Zapier
5. Pilih peristiwa mana yang harus memicu webhook
6. Klik "Buat Webhook"

## Blue ke Zapier

1. Masuk ke akun Zapier Anda
2. Klik Buat Zap  
3. Untuk langkah pertama, cari dan pilih aplikasi pemicu ["Webhooks by Zapier"](https://zapier.com/apps/webhook)
4. Konfigurasikan untuk menerima payload dari URL webhook Blue Anda
5. Untuk langkah kedua, cari dan pilih aplikasi tujuan untuk mengirim data
6. Peta bidang payload JSON ke bidang input aplikasi tujuan
7. Aktifkan Zap

Sekarang ketika peristiwa yang ditentukan terjadi di Blue, payload webhook akan dikirim ke Zapier yang secara otomatis meneruskan data ke aplikasi terintegrasi Anda sesuai dengan pengaturan Zap.

## Zapier ke Blue

Untuk mengirim data dari aplikasi lain ke Blue, ikuti langkah-langkah berikut:

**1. Dapatkan Kredensial API Blue**
- Pergi ke Pengaturan Blue > Token API
- Buat token baru atau gunakan kredensial yang ada:
  - ID Token
  - Rahasia Token
  - ID Perusahaan
  - ID Proyek

**2. Buat Zap di Zapier**
1. Atur aplikasi pemicu Anda (misalnya, Google Sheets, Slack, dll.)
2. Tambahkan "Webhooks by Zapier" sebagai aplikasi Tindakan
3. Pilih jenis peristiwa "Permintaan Kustom"

**3. Konfigurasikan Webhook**
- Metode: POST
- URL: `https://api.blue.cc/graphql` (atau endpoint beta jika diperlukan)
- Data Pass-Through: FALSE
- Unflatten: NO

**4. Atur Header:**
```json
{
  "Content-Type": "application/json",
  "X-Bloo-Token-ID": "Your_Token_ID",
  "X-Bloo-Token-Secret": "Your_Token_Secret", 
  "X-Bloo-Company-ID": "Your_Company_ID",
  "X-Bloo-Project-ID": "Your_Project_ID"
}
```

**5. Konfigurasikan Query (Contoh Mutasi):**
```json
{
  "query": "mutation CreateTodo($input: CreateTodoInput!) {\n  createTodo(input: $input) {\n    id\n    title\n    customFields {\n      value\n    }\n  }\n}",
  "variables": {
    "input": {
      "todoListId": "your-todo-list-id",
      "title": "New task from Zapier",
      "customFields": [
        {
          "customFieldId": "priority-field-id",
          "value": "High"
        },
        {
          "customFieldId": "due-date-field-id",
          "value": "2024-03-01"
        }
      ]
    }
  }
}
```

**6. Uji & Aktifkan**
- Gunakan API Playground Blue untuk menguji mutasi terlebih dahulu
- Peta nilai dinamis dari aplikasi pemicu menggunakan bidang data Zapier
- Aktifkan Zap setelah pengujian berhasil
