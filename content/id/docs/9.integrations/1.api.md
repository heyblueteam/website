Blue memiliki API (Antarmuka Pemrograman Aplikasi), sehingga Anda dapat membangun integrasi Anda sendiri jika diperlukan.

**Blue memiliki cakupan API 100%**. Ini berarti bahwa apa pun yang dapat Anda lakukan di [salah satu aplikasi kami](/docs/start-guide/download-apps), Anda dapat melakukannya secara programatik. Kami memiliki API [GraphQL](https://graphql.org/) yang kuat dengan playground lengkap di mana Anda dapat menguji kueri Anda dan sepenuhnya didokumentasikan.
