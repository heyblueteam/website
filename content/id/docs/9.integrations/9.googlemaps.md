Blue memungkinkan Anda menggunakan Google Maps untuk tampilan peta alih-alih OpenStreetMap default. Google Maps sering kali memiliki peta yang lebih detail dari area tertentu dan data lokasi yang lebih akurat. Namun, ini memerlukan kunci API Google Maps yang harus dimasukkan di pengaturan perusahaan oleh pemilik perusahaan.

Saat Anda beralih ke Google Maps, ini mempengaruhi semua tampilan peta di Blue termasuk peta perusahaan, [tampilan peta di proyek](/docs/views/map), dan [field kustom lokasi](/docs/custom-fields/location-map).

Untuk menggunakan Google Maps di Blue, API berikut harus diaktifkan di Google Cloud Console:
- [Maps Tiles API](https://console.cloud.google.com/apis/library/tile.googleapis.com): Ini diperlukan untuk merender ubin peta.
- [Places API](https://console.cloud.google.com/apis/library/places.googleapis.com): Ini diperlukan untuk mencari tempat dan mendapatkan koordinatnya.
- [Geocoding API](https://console.cloud.google.com/apis/library/geocoding-backend.googleapis.com): Ini diperlukan untuk mendapatkan koordinat suatu tempat.

Anda perlu mendaftar untuk [akun Google Cloud](https://console.cloud.google.com/) dan membuat proyek untuk mendapatkan kunci API mengikuti instruksi [di sini](https://developers.google.com/maps/documentation/embed/get-api-key).

## Pembatasan Penggunaan Kunci API

Untuk tujuan keamanan, sangat penting untuk membatasi kunci API Google Maps Anda agar hanya berfungsi dengan domain yang diizinkan. Ini mencegah penggunaan kunci API Anda yang tidak sah jika terpapar. Anda harus membatasi kunci API agar hanya berfungsi dengan:

- app.blue.cc - Aplikasi Blue utama
- forms.blue.cc - Untuk formulir publik
- Domain kustom Anda jika Anda menggunakan [white label](/docs/start-guide/white-label)

Untuk membatasi kunci API Anda:

1. Pergi ke [halaman Kredensial Google Cloud Console](https://console.cloud.google.com/apis/credentials)
2. Temukan kunci API Anda dan klik "Edit" (ikon pensil)
3. Di bawah "Pembatasan aplikasi", pilih "HTTP referrers (situs web)"
4. Tambahkan referer berikut:
   - `*.app.blue.cc/*`
   - `*.forms.blue.cc/*`
   - `*.yourdomain.com/*` (jika menggunakan white label)
5. Klik "Simpan"

**Catatan Penting tentang Biaya Google Maps**<br><br>

Google akan mengenakan biaya untuk penggunaan API Google Maps secara terpisah dari langganan Blue Anda. Semua biaya, pengeluaran, dan biaya yang terkait dengan akun Google Maps Anda dan penggunaan API adalah tanggung jawab penuh administrator organisasi. Blue tidak bertanggung jawab atas biaya yang timbul melalui penggunaan API Google Maps Anda, termasuk tetapi tidak terbatas pada integrasi pihak ketiga, penggunaan API, atau biaya penyimpanan.
