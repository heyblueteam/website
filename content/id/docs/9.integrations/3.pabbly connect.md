[Pabbly Connect](https://payments.pabbly.com/api/affurl/RVYZ07kQyUZ0Z1HUKZ1m/zp2GoChM1JcZ0dQBo?target=nMfdbf0I90K3UdJn) memungkinkan Anda menghubungkan Blue dengan banyak aplikasi populer seperti Google Calendar, Stripe, MailChimp, dan lainnya. Ini memungkinkan otomatisasi alur kerja dengan mengirimkan data antara alat.

<youtube url="https://www.youtube.com/watch?v=IRxYAe5snzY" />

Ada kemungkinan tak terbatas untuk mengintegrasikan Blue menggunakan Pabbly Connect. Berikut adalah langkah-langkah kunci:

 ## Akses Dasbor Pabbly Anda

1. Masuk ke [Dasbor Pabbly](https://connect.pabbly.com/workflows)
2. Klik tombol "+ Buat Alur Kerja".
3. Beri nama alur kerja Anda.
4. Pilih Blue sebagai aplikasi tujuan.

## Hubungkan ke Akun Blue Anda

1. Hasilkan [token API](/docs/integrations/api) di profil Blue Anda.
2. Di Pabbly, masukkan:
- ID Token
- Kunci Rahasia
- URL Perusahaan (mis. `https://app.blue.cc/company/mycompany`)
- ID Proyek (mis. `myproject`)

::callout
 ---

icon: lightbulb
---

*Salin Kunci Rahasia Anda segera, karena tidak akan ditampilkan lagi.*
::

## Buat Integrasi Uji
1. Pilih aplikasi pemicu dan acara (mis. acara Google Calendar baru).
2. Kirim permintaan uji.
3. Periksa Blue untuk memastikan data uji diterima ke proyek yang Anda tentukan.
