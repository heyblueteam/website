Administrator Proyek adalah peran dengan hak istimewa tertinggi dalam proyek Blue yang mirip dengan Pemilik Proyek. Admin Proyek memiliki kontrol penuh atas proyek dan pengaturannya.

**Izin**

Admin Proyek dapat:

- Mengedit semua pengaturan proyek
- Menambah/menghapus anggota proyek
- Menetapkan [peran](/docs/user-management/roles) dan [izin](/docs/user-management/roles/custom-user-roles) pengguna
- [Mengimpor](/docs/integrations/csv-import) dan [mengekspor](/docs/integrations/csv-export) data proyek
- Menghapus proyek
- Membuat [bidang kustom](/docs/custom-fields/introduction)
- Membuat peran kustom
- Mengatur [Automasi](/docs/automations/introduction)

Pada dasarnya, mereka memiliki izin tak terbatas dalam proyek.

**Kasus Penggunaan**

Pengguna tipikal yang ditugaskan sebagai Admin Proyek:

- Manajer proyek
- Pemimpin tim
- Manajer TI
- Pemilik bisnis

Siapa pun yang perlu mengonfigurasi alur kerja sepenuhnya atau mengelola anggota.

**Menetapkan Peran**

Saat mengundang anggota, peran Admin Proyek dapat ditetapkan.

Anggota yang ada juga dapat dipromosikan menjadi Admin Proyek. Namun, harus selalu ada setidaknya satu Admin Proyek dalam sebuah proyek.

**Menghapus Peran**

Untuk menghapus izin Admin Proyek dari seorang anggota, mereka harus terlebih dahulu diturunkan ke peran kustom yang lebih terbatas atau peran standar.
