[Pabbly Connect](https://payments.pabbly.com/api/affurl/RVYZ07kQyUZ0Z1HUKZ1m/zp2GoChM1JcZ0dQBo?target=nMfdbf0I90K3UdJn) អនុញ្ញាតឱ្យអ្នកភ្ជាប់ Blue ជាមួយកម្មវិធីពេញនិយមជាច្រើនដូចជា Google Calendar, Stripe, MailChimp, និងផ្សេងទៀត។ នេះធ្វើឱ្យអាចអូតូម៉ាតវ៉ាក់ដូសដោយការផ្ញើទិន្នន័យរវាងឧបករណ៍។

<youtube url="https://www.youtube.com/watch?v=IRxYAe5snzY" />

មានឱកាសអស់សীমសម្រាប់ការភ្ជាប់ Blue ដោយប្រើ Pabbly Connect។ នេះគឺជាជំហានសំខាន់ៗ៖

 ## ចូលទៅកាន់ផ្ទាំងគ្រប់គ្រង Pabbly របស់អ្នក

1. ចូលទៅកាន់ [ផ្ទាំងគ្រប់គ្រង Pabbly](https://connect.pabbly.com/workflows)
2. ចុចប៊ូតុង "+ បង្កើតវ៉ាក់ដូស"។
3. ឈ្មោះវ៉ាក់ដូសរបស់អ្នក។
4. ជ្រើស Blue ជាកម្មវិធីគោលដៅ។

## ភ្ជាប់ទៅគណនី Blue របស់អ្នក

1. បង្កើត [API token](/docs/integrations/api) នៅក្នុងប្រវត្តិរូប Blue របស់អ្នក។
2. នៅក្នុង Pabbly, បញ្ចូល៖
- Token ID
- Secret Key
- Company URL (ឧ. `https://app.blue.cc/company/mycompany`)
- Project ID (ឧ. `myproject`)

## បង្កើតការភ្ជាប់សាកល្បង
1. ជ្រើសកម្មវិធីជំរុញ និងព្រឹត្តិការណ៍ (ឧ. ព្រឹត្តិការណ៍ថ្មីនៅ Google Calendar)។
2. ផ្ញើសំណើសាកល្បង។
3. ពិនិត្យ Blue ដើម្បីធានាថាទិន្នន័យសាកល្បងត្រូវបានទទួលនៅក្នុងគម្រោងដែលអ្នកបានកំណត់។
