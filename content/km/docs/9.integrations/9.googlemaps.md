Blue អនុញ្ញាតឱ្យអ្នកប្រើផែនទី Google សម្រាប់ការមើលផែនទី ជំនួសការបង្ហាញផែនទី OpenStreetMap ដើម។ ផែនទី Google មានផែនទីលម្អិតជាងនេះសម្រាប់តំបន់ជាក់លាក់មួយចំនួន និងទិន្នន័យទីតាំងដែលត្រឹមត្រូវជាង។ ទោះយ៉ាងណា វាត្រូវការទីគន្លង API ផែនទី Google ហើយត្រូវតែបញ្ចូលនៅក្នុងការកំណត់ក្រុមហ៊ុនដោយម្ចាស់ក្រុមហ៊ុន។

ពេលដែលអ្នកប្តូរទៅផែនទី Google វានឹងមានឥទ្ធិពលដល់ការមើលផែនទីទាំងអស់នៅក្នុង Blue រួមមានផែនទីក្រុមហ៊ុន, [ការមើលផែនទីនៅក្នុងគម្រោង](/docs/views/map), និង [វាលកំណត់ទីតាំង](/docs/custom-fields/location-map)។

ដើម្បីប្រើផែនទី Google នៅក្នុង Blue, APIs ខាងក្រោមត្រូវការត្រូវបើកនៅក្នុង Google Cloud Console:
- [Maps Tiles API](https://console.cloud.google.com/apis/library/tile.googleapis.com): នេះត្រូវការដើម្បីបង្ហាញផែនទី។
- [Places API](https://console.cloud.google.com/apis/library/places.googleapis.com): នេះត្រូវការដើម្បីស្វែងរកកន្លែង និងទទួលបានកូអូរឌីណេត។
- [Geocoding API](https://console.cloud.google.com/apis/library/geocoding-backend.googleapis.com): នេះត្រូវការដើម្បីទទួលបានកូអូរឌីណេតនៃកន្លែង។

អ្នកត្រូវការចុះឈ្មោះសម្រាប់ [គណនី Google Cloud](https://console.cloud.google.com/) និងបង្កើតគម្រោងដើម្បីទទួលបានកូនសោ API ដោយអនុវត្តតាមការណែនាំ [នៅទីនេះ](https://developers.google.com/maps/documentation/embed/get-api-key)។

## ការកំណត់ការប្រើប្រាស់កូនសោ API

សម្រាប់គោលបំណងសុវត្ថិភាព វាជារឿងសំខាន់ក្នុងការកំណត់កូនសោ API ផែនទី Google របស់អ្នកឱ្យធ្វើការជាមួយដូមែនដែលបានអនុញ្ញាតតែប៉ុណ្ណោះ។ នេះជួយការពារការប្រើប្រាស់កូនសោ API របស់អ្នកដោយមិនមានអាជ្ញាប័ណ្ណ ប្រសិនបើវាត្រូវបានបង្ហាញ។ អ្នកគួរតែបង្កើតកំណត់កូនសោ API ឱ្យធ្វើការជាមួយ៖

- app.blue.cc - កម្មវិធី Blue មុខងារ
- forms.blue.cc - សម្រាប់ទម្រង់សាធារណៈ
- ដូមែនផ្ទាល់ខ្លួនរបស់អ្នក ប្រសិនបើអ្នកកំពុងប្រើ [white label](/docs/start-guide/white-label)

ដើម្បីកំណត់កូនសោ API របស់អ្នក:

1. ចូលទៅកាន់ [ទំព័រ Google Cloud Console Credentials](https://console.cloud.google.com/apis/credentials)
2. ស្វែងរកកូនសោ API របស់អ្នក ហើយចុច "កែសម្រួល" (រូបតំណាងក្រឡាចត្រង់)
3. នៅក្រោម "ការកំណត់កំណត់កម្មវិធី" ជ្រើស "HTTP referrers (គេហទំព័រ)"
4. បន្ថែមអ្នកយោងដូចខាងក្រោម:
   - `*.app.blue.cc/*`
   - `*.forms.blue.cc/*`
   - `*.yourdomain.com/*` (ប្រសិនបើប្រើ white label)
5. ចុច "រក្សាទុក"
