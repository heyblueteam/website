Blue មាន API (Application Programming Interface) ដូច្នេះអ្នកអាចបង្កើតការបញ្ចូលរបស់អ្នកឯង ប្រសិនបើត្រូវការ។

**Blue មានការក覆 100% API**។ នេះមានន័យថា អ្វីដែលអ្នកអាចធ្វើបាននៅក្នុង [កម្មវិធីណាមួយរបស់យើង](/docs/start-guide/download-apps) អ្នកអាចធ្វើបានតាមកម្មវិធី។ យើងមាន [GraphQL](https://graphql.org/) API ដែលមានកន្លែងលេងពេញលេញដែលអ្នកអាចសាកល្បងសំណួររបស់អ្នក និងមានឯកសារពេញលេញ។

យើងមានឯកសារបញ្ជាក់ API ដែលមានការប្តេជ្ញាដែលអាចស្វែងរកបានពេញលេញ និងមានព័ត៌មានទាំងអស់ដែលអ្នកត្រូវការ។ អ្នកអាចស្វែងរកវានៅ [ទីនេះ](/api)។
