កំណត់ត្រានៅក្នុង Blue មានភាពបត់បែន និងអាចប្រើប្រាស់សម្រាប់គោលបំណងជាច្រើន ដូចជាការតាមដានទំនាក់ទំនងជាមួយអតិថិជន ការលក់ សញ្ញាសម្គាល់នៃគម្រោង ឬសារពើភ័ណ្ឌ។ ភាពអាចបត់បែននេះអនុញ្ញាតឱ្យអ្នកប្ដូរតាមបំណង Blue ឱ្យបំពេញតម្រូវការអាជីវកម្មរបស់អ្នក ធ្វើឱ្យវាក្លាយជាឧបករណ៍ដ៏មានតម្លៃសម្រាប់រៀបចំព័ត៌មានសំខាន់ៗ។

## ការប្ដូរឈ្មោះកំណត់ត្រា

អ្នកអាចប្ដូរឈ្មោះកំណត់ត្រានៅក្នុង Blue យ៉ាងងាយស្រួលដើម្បីឱ្យសមនឹងតម្រូវការរបស់អ្នក។ គ្រាន់តែចូលទៅកាន់ការកំណត់គម្រោង ហើយប្ដូរឈ្មោះទៅជាអ្វីដែលអ្នកចូលចិត្ត ដូចជាកិច្ចការ ឬការចាត់តាំង។ ការផ្លាស់ប្ដូរនេះនឹងអនុវត្តពេញមួយគម្រោង។

នេះជាឧទាហរណ៍មួយចំនួននៃឈ្មោះដែលអ្នកអាចប្រើសម្រាប់កំណត់ត្រា៖

- កិច្ចការ / កិច្ចការ
- ការចាត់តាំង / ការចាត់តាំង
- ជំហាន / ជំហាន
- បញ្ហា / បញ្ហា
- សំបុត្រ / សំបុត្រ
- អតិថិជនសក្តានុពល / អតិថិជនសក្តានុពល
- កិច្ចព្រមព្រៀង / កិច្ចព្រមព្រៀង
- ករណី / ករណី
- ឱកាស / ឱកាស
- អតិថិជន / អតិថិជន
- ផលិតផល / ផលិតផល
- ការបញ្ជាទិញ / ការបញ្ជាទិញ

សូមចងចាំថាត្រូវរួមបញ្ចូលទាំងទម្រង់ឯកវចនៈ និងពហុវចនៈនៃឈ្មោះដែលអ្នកបានជ្រើសរើស។ នេះជួយឱ្យប្រព័ន្ធបង្ហាញពាក្យត្រឹមត្រូវផ្អែកលើបរិបទ។

ការប្ដូរឈ្មោះកំណត់ត្រាតាមបំណងអនុញ្ញាតឱ្យអ្នកប្រើប្រាស់ពាក្យបច្ចេកទេសដែលសមស្របនឹងឧស្សាហកម្មរបស់អ្នក ឬចំណូលចិត្តផ្ទាល់ខ្លួន ធ្វើឱ្យវេទិកានេះកាន់តែងាយស្រួល និងងាយយល់សម្រាប់ក្រុមរបស់អ្នក។

![Records 1](/docs/Record_1.png "Records 1")

កំណត់ត្រាអាចតំណាងឱ្យធាតុជាច្រើនប្រភេទ ដូចជា៖

- ទំនាក់ទំនងជាមួយអតិថិជន
- កំណត់ត្រាការលក់
- សញ្ញាសម្គាល់នៃគម្រោង
- ធាតុនៃសារពើភ័ណ្ឌ

ភាពបត់បែននេះធានាថា Blue អាចត្រូវបានប្ដូរតាមបំណងដើម្បីសមនឹងដំណើរការអាជីវកម្មចម្រុះ និងតម្រូវការប្រតិបត្តិការ ធ្វើឱ្យពួកវាក្លាយជាឧបករណ៍ដ៏មានអានុភាពសម្រាប់រៀបចំ និងគ្រប់គ្រងទិន្នន័យសំខាន់ៗ។
