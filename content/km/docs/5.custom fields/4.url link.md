វាលផ្ទាល់ខ្លួន URL/តំណ អនុញ្ញាតឱ្យអ្នករក្សាទុកតំណ និងអាសយដ្ឋានគេហទំព័រនៅក្នុងកំណត់ត្រារបស់អ្នកនៅក្នុង Blue។ នៅពេលដែល URL ត្រូវបានបញ្ចូលទៅក្នុងវាលនេះ ប្រព័ន្ធនឹងត្រួតពិនិត្យថាវាជាតំណដែលមានរចនាសម្ព័ន្ធត្រឹមត្រូវ។

វាលផ្ទាល់ខ្លួន URL/តំណនៅក្នុង Blue

កំណត់ត្រានីមួយៗអាចមានតំណមួយ ដែលត្រូវបានរក្សាទុកនៅក្នុងវាលផ្ទាល់ខ្លួន URL/តំណតែមួយ។ តំណនេះអាចចុចបាន ដែលអនុញ្ញាតឱ្យអ្នករុករកទៅកាន់ទំព័រគេហទំព័រ ឬធនធានដោយផ្ទាល់តាមរយៈការចុចលើវា។ សម្រាប់ភាពងាយស្រួល វានឹងបើកដោយស្វ័យប្រវត្តិនៅក្នុងផ្ទាំងរុករកថ្មី។

អ្នកអាចបើក URL នៅក្នុងផ្ទាំងថ្មីតាមរយៈការចុចតែម្តង

វាលនេះមានប្រយោជន៍យ៉ាងខ្លាំងសម្រាប់ការភ្ជាប់ទៅកាន់ឯកសារដែលមានមូលដ្ឋានលើពពក ដូចជា Google Docs ឯកសារ Word ស្ព្រីតសៀត Excel ជាដើម។ ការរក្សាទុកតំណទាំងនេះអនុញ្ញាតឱ្យមានការចូលប្រើឯកសារពាក់ព័ន្ធដែលត្រូវបានភ្ជាប់ជាមួយនឹងកំណត់ត្រានីមួយៗតាមរយៈការចុចតែម្តង។

វាល URL/តំណគឺល្អសម្រាប់ករណីប្រើប្រាស់បន្ថែមដូចជា៖

- រក្សាទុកគេហទំព័រអតិថិជន ឬដៃគូ
- ចាប់យកតំណដែលត្រូវបានភ្ជាប់ជាមួយនឹងកំណត់ត្រា
- បង្កើតការយោងទៅកាន់ទំព័រគេហទំព័រ

ជាមួយនឹងចំណុចប្រទាក់ដែលបានសម្រួល និងសមត្ថភាពតំណដែលបានបង្កប់ វាលផ្ទាល់ខ្លួន URL/តំណអនុញ្ញាតឱ្យអ្នកបំបាត់កំណត់ត្រាជាមួយនឹងធនធានគេហទំព័រដែលមានបរិបទ និងចូលប្រើពួកវាភ្លាមៗតាមរយៈការចុច។
