Dashboards ក្នុង Blue មានអំណាចព្រោះវាអនុញ្ញាតឱ្យអ្នកបញ្ចូលទិន្នន័យពេលវេលាពិតពីគម្រោងជាច្រើននៅពេលតែមួយ។ វាអនុញ្ញាតឱ្យអ្នកបង្កើត dashboards ផ្ទាល់ដែលផ្តល់ព័ត៌មានមានអត្ថន័យសម្រាប់ការសម្រេចចិត្ត។

## ជម្រើសក្រាហ្វិក

Blue មានជម្រើសបីសម្រាប់ក្រាហ្វិក៖

1. [Stat Cards:](/docs/dashboards/stat-cards) គឺអត្ថប្រយោជន៍សម្រាប់ការមើលឃើញគម្រោងសំខាន់ៗ ដោយ stat cards បង្ហាញទិន្នន័យពេលវេលាពិតដែលមានចំនួនតែមួយដូចជា ចំណូលសរុបគម្រោង បញ្ហាបច្ចុប្បន្ន និងភាគរយនៃការបញ្ចប់គម្រោងសរុប។ វាផ្តល់ការយល់ដឹងភ្លាមៗអំពីទិន្នន័យសំខាន់ៗជាមួយនឹងអន្តរកម្មសាមញ្ញ និងច្បាស់លាស់។
2. [Pie Charts:](/docs/dashboards/pie-charts) គឺល្អសម្រាប់ការបង្ហាញការប្រមូលទិន្នន័យ ដែល pie charts ក្នុង Blue ជួយអ្នកយល់ពីភាគរយនៃធាតុផ្សេងៗនៅក្នុងសំណុំទិន្នន័យ។ ឧទាហរណ៍ អ្នកអាចប្រើ pie chart ដើម្បីបង្ហាញការចែកចាយនៃភារកិច្ចនៅក្នុងផ្នែកផ្សេងៗ ឬភាគរយនៃចំណូលពីអតិថិជនប្រភេទផ្សេងៗ។
3. [Bar Graphs:](/docs/dashboards/bar-graphs) មានប្រសិទ្ធភាពសម្រាប់ការប្រៀបធៀបបរិមាណ Bar charts អនុញ្ញាតឱ្យអ្នកប្រៀបធៀបក្រុមផ្សេងៗគ្នា។ ប្រើ bar charts ដើម្បីប្រៀបធៀបលេខសំគាល់លក់នៅក្នុងត្រីមាសផ្សេងៗ ចំនួនភារកិច្ចដែលសមាជិកក្រុមម្នាក់ម្នាក់បញ្ចប់ ឬកម្រិតនៃសំណួរអតិថិជនជាក់លាក់នៅលើពេលវេលា។

## ការចែករំលែក
Dashboards ក្នុង Blue មិនត្រឹមតែសម្រាប់ការមើលផ្ទាល់ខ្លួនទេ; វាអាចត្រូវបានចែករំលែកជាមួយសមាជិកក្រុមនៅក្នុងវេទិកា។ អ្នកអាចផ្ដល់សិទ្ធិកែសម្រួលឬមើល។ អ្នកណាមួយដែលមានការចូលដំណើរការទៅកាន់ dashboard អាចចែករំលែកវាជាមួយអ្នកប្រើផ្សេងទៀត។

សិទ្ធិចែករំលែកត្រូវបានចាត់តាំងតាមសិទ្ធិអ្នកប្រើ; ឧទាហរណ៍ អ្នកប្រើដែលមានសិទ្ធិមើលអាចអញ្ជើញអ្នកផ្សេងៗឱ្យមើល dashboard ដើម្បីធានាថាសិទ្ធិចូលដំណើរការត្រូវបានរក្សាទុក និងគោរព។
::

បច្ចុប្បន្ននេះ dashboards របស់ Blue ត្រូវបានរចនាសម្រាប់ការសហការផ្ទៃក្នុង ហើយមិនអាចចែករំលែកក្រៅជាមួយអ្នកប្រើដែលមិនមែនជាអ្នកប្រើ Blue តាមរយៈតំណភ្ជាប់សាធារណៈទេ។ ទោះយ៉ាងណា វាត្រូវបានគ្រោងសម្រាប់ការចេញផ្សាយនៅអនាគត។ ប្រសិនបើអ្នកមានគំនិតបន្ថែមសម្រាប់ dashboards អ្នកអាចចូលរួម [សហគមន៍របស់យើង](https://ask.blue.cc/) ឬទំនាក់ទំនងមកយើងដោយផ្ទាល់នៅ [support@blue.cc](support@blue.cc)

## ការការពារ

Dashboards របស់ Blue ផ្តល់នូវសមត្ថភាពកំណត់ត្រាដែលមានភាពស្មុគស្មាញ ដើម្បីអនុញ្ញាតឱ្យមានការបែងចែកទិន្នន័យយ៉ាងត្រឹមត្រូវតាមគម្រោង បញ្ជី តាក់ និងពេលវេលា។ បញ្ជី និងតាក់ដែលមានឈ្មោះដូចគ្នាត្រូវបានបញ្ចូលយ៉ាងឆ្លាតវៃនៅក្នុងគម្រោង ដើម្បីធ្វើឱ្យដំណើរការបែងចែកទិន្នន័យដែលពាក់ព័ន្ធនឹងការប្រកួតផ្សេងៗកាន់តែងាយស្រួល។ ការរចនាដែលឆ្លាតវៃនេះអនុញ្ញាតឱ្យមានការវិភាគពហុគម្រោងដោយគ្មានកំណត់ធ្វើឱ្យមានការយល់ដឹងទូលំទូលាយ និងពាក់ព័ន្ធ។

## អន្តរកម្ម និងអាចកំណត់ផ្ទាល់ខ្លួន

- **មុខងារ ធ្វើឱ្យចុះ និងទាញឡើង**: ការកំណត់រចនាប័ទ្ម dashboard របស់អ្នកគឺមានភាពងាយស្រួលជាមួយមុខងារ drag and drop របស់ Blue។ អ្នកអាចផ្លាស់ទីក្រាហ្វិកក្នុង dashboard យ៉ាងឆាប់រហ័សដើម្បីសម្រួលនឹងចំណង់ចំណូលចិត្តរបស់អ្នកឬដើម្បីបង្ហាញព័ត៌មានសំខាន់ៗបំផុត។
- **ភាពបត់បែនក្នុងការបង្ហាញ**: Blue យល់ថាការបង្ហាញទិន្នន័យមិនមែនជារូបរាងមួយទេ។ នេះហើយជាហេតុដែលវាបង្កើតជម្រើសបង្ហាញជាច្រើនសម្រាប់គ្រប់ក្រាហ្វិក និងការបង្ហាញទិន្នន័យនៅលើ dashboards។ មិនថាអ្នកត្រូវការក្រាហ្វិក pie ដើម្បីបំបែកការចែកចាយភារកិច្ច ឬក្រាហ្វិកបន្ទាត់ដើម្បីតាមដានការរីកចម្រើននៅលើពេលវេលា អ្នកមានភាពបត់បែនក្នុងការជ្រើសរើសទ្រង់ទ្រាយដែលល្អបំផុតសម្រាប់ការបង្ហាញទិន្នន័យរបស់អ្នក។

## ការលុប dashboards

ដើម្បីលុប dashboard មួយ អ្នកគ្រាន់តែចុចស្តាំលើ dashboard ក្នុងបញ្ជី ហើយចុចលុប។ សូមចំណាំថាអ្នកត្រូវតែជាម្ចាស់ dashboard - អ្នកដែលបានបង្កើត dashboard - ដើម្បីលុបវា។
