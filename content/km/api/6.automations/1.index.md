## បញ្ជីអូតូម៉ាទិនទាំងអស់

អូតូម៉ាទិនក្នុង Blue អនុញ្ញាតឱ្យអ្នកបង្កើតការងារដែលមានអំណាចដែលបញ្ចេញសកម្មភាពដោយផ្អែកលើព្រឹត្តិការណ៍ជាក់លាក់។ ចំណុចចូលនេះទាញយកអូតូម៉ាទិនទាំងអស់ដែលបានកំណត់សម្រាប់គម្រោងរបស់អ្នក រួមទាំងការបញ្ចេញសកម្មភាព សកម្មភាព និងស្ថានភាពបច្ចុប្បន្ន។

### ឧទាហរណ៍មូលដ្ឋាន

```graphql
query ListAutomations {
  automationList {
    items {
      id
      isActive
      trigger {
        type
        color
      }
      actions {
        type
        color
      }
      createdAt
    }
    pageInfo {
      totalItems
      hasNextPage
    }
  }
}
```

### ឧទាហរណ៍កម្រិតខ្ពស់

```graphql
query ListAutomationsAdvanced {
  automationList(
    filter: {
      customFieldIds: ["field_123", "field_456"]
    }
    skip: 0
    take: 50
  ) {
    items {
      id
      isActive
      createdAt
      updatedAt
      
      # Trigger details
      trigger {
        id
        type
        color
        metadata {
          ... on AutomationTriggerMetadataTodoOverdue {
            incompleteOnly
          }
        }
        customField {
          id
          name
          type
        }
        customFieldOptions {
          id
          title
          color
        }
        todoList {
          id
          title
        }
        tags {
          id
          title
          color
        }
        assignees {
          id
          name
          email
        }
      }
      
      # Action details
      actions {
        id
        type
        color
        duedIn
        assigneeTriggerer
        
        # Action-specific fields
        customField {
          id
          name
        }
        customFieldOptions {
          id
          title
        }
        todoList {
          id
          title
        }
        tags {
          id
          title
        }
        assignees {
          id
          name
        }
        
        # Email action metadata
        metadata {
          ... on AutomationActionMetadataSendEmail {
            email {
              subject
              to
              from
              content
            }
          }
          ... on AutomationActionMetadataCreateChecklist {
            checklists {
              title
              checklistItems {
                title
                duedIn
              }
            }
          }
        }
      }
      
      # Creator info
      createdBy {
        id
        name
        email
      }
    }
    
    pageInfo {
      totalItems
      totalPages
      hasNextPage
      hasPreviousPage
    }
    
    totalCount
  }
}
```

## ព័ត៌មានបញ្ចូល

### AutomationFilterInput

| ប៉ារ៉ាម៉ែត្រ | ប្រភេទ | ត្រូវការ | ការពិពណ៌នា |
|-----------|------|----------|-------------|
| `customFieldIds` | [String] | ទេ | ត្រួតពិនិត្យអូតូម៉ាទិនដែលពាក់ព័ន្ធនឹងវាលកំណត់ផ្ទាល់ខ្លួនជាក់លាក់ |  |

### ព័ត៌មានទំព័រ

| ប៉ារ៉ាម៉ែត្រ | ប្រភេទ | ត្រូវការ | ការពិពណ៌នា |
|-----------|------|----------|-------------|
| `skip` | Int | ទេ | ចំនួនធាតុដែលត្រូវរំលង (លំនាំដើម: 0) |  |
| `take` | Int | ទេ | ចំនួនធាតុដែលត្រូវត្រឡប់ (លំនាំដើម: 20) |  |

## វាលឆ្លើយតប

### AutomationPagination

| វាល | ប្រភេទ | ការពិពណ៌នា |
|-------|------|-------------|
| `items` | [Automation!]! | បញ្ជីវត្ថុអូតូម៉ាទិន |  |
| `pageInfo` | PageInfo! | ព័ត៌មានទំព័រ |  |
| `totalCount` | Int! | ចំនួនសរុបនៃអូតូម៉ាទិន |  |

### Automation

| វាល | ប្រភេទ | ការពិពណ៌នា |
|-------|------|-------------|
| `id` | ID! | អត្តសញ្ញាណឯកតា |  |
| `trigger` | AutomationTrigger! | ព្រឹត្តិការណ៍ដែលបញ្ចេញអូតូម៉ាទិននេះ |  |
| `actions` | [AutomationAction!]! | សកម្មភាពដែលអនុវត្តនៅពេលដែលបានបញ្ចេញ |  |
| `isActive` | Boolean! | ថាតើអូតូម៉ាទិននេះមានសកម្មភាពក្នុងពេលបច្ចុប្បន្ន |  |
| `createdBy` | User! | អ្នកប្រើដែលបានបង្កើតអូតូម៉ាទិន |  |
| `project` | Project! | គម្រោងដែលអូតូម៉ាទិននេះពាក់ព័ន្ធ |  |
| `createdAt` | DateTime! | ម៉ោងបង្កើត |  |
| `updatedAt` | DateTime! | ម៉ោងធ្វើបច្ចុប្បន្នបន្ទាប់ |  |

## អាជ្ញាប័ណ្ណដែលត្រូវការ

ការបញ្ជីអូតូម៉ាទិនត្រូវការការផ្ទៀងផ្ទាត់ និងការចូលដំណើរការគម្រោង៖

| តួនាទី | អាចបញ្ជីអូតូម៉ាទិន |
|------|---------------------|
| `OWNER` | ✅ បាទ |  |
| `ADMIN` | ✅ បាទ |  |
| `MEMBER` | ✅ បាទ |  |
| `CLIENT` | ✅ បាទ |  |

អ្នកប្រើដែលបានផ្ទៀងផ្ទាត់ទាំងអស់ដែលមានការចូលដំណើរការគម្រោងអាចមើលអូតូម៉ាទិន។

## កំណត់ចំណាំសំខាន់

- **ការតម្រៀប**: អូតូម៉ាទិនត្រូវបានត្រឡប់មកវិញជារឿយៗថ្មីជាង (ដោយកាលបរិច្ឆេទបង្កើត)
- **វិសាលភាពគម្រោង**: អូតូម៉ាទិនត្រូវបានកំណត់ទៅក្នុងបរិបទគម្រោងបច្ចុប្បន្ន
- **ស្ថានភាពសកម្ម**: ពិនិត្យ `isActive` ដើម្បីកំណត់ថាអូតូម៉ាទិនណាមួយកំពុងរត់
- **មេដាតា**: ការបញ្ចេញ និងសកម្មភាពខ្លះមានមេដាតាសម្រាប់ការកំណត់
- **សមត្ថភាព**: ប្រើការបង្ហាញទំព័សម្រាប់គម្រោងដែលមានអូតូម៉ាទិនច្រើន
- **ការត្រួតពិនិត្យ**: បច្ចុប្បន្នគាំទ្រតែការត្រួតពិនិត្យតាមលេខសម្គាល់វាលកំណត់ផ្ទាល់ខ្លួនเท่านั้น
