---

ដើម្បីបង្កើតក្រុមហ៊ុនថ្មីមួយ អ្នកអាចប្រើការប្រែប្រួលដូចខាងក្រោម៖

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

សូមចំណាំថាការបង្កើតក្រុមហ៊ុនថ្មីៗអាចមានឥទ្ធិពលលើការទូទាត់ប្រាក់នៃគណនី Blue របស់អ្នក ពីព្រោះក្រុមហ៊ុននីមួយៗត្រូវបានគិតថ្លៃបំបែក។

នេះគឺជាជម្រើសសម្រាប់ការប្រែប្រួល `createCompany` និងការពិពណ៌នារបស់ពួកវា៖

| ជម្រើស | ប្រភេទ | ការពិពណ៌នា |
|--------|------|-------------|
| name | String | ឈ្មោះនៃក្រុមហ៊ុនថ្មី។ នេះគឺជាដែនកំណត់មួយ។ |  |
| slug | String | កំណែដែលមានមិត្តភាពជាមួយ URL នៃឈ្មោះក្រុមហ៊ុន។ នេះត្រូវបានប្រើនៅក្នុង URL របស់ក្រុមហ៊ុន និងគួរតែមានអត្តសញ្ញាណឯកត្ត។ នេះគឺជាដែនកំណត់មួយ និងនឹងដំណើរការជាអត្តសញ្ញាណឯកត្តសម្រាប់ក្រុមហ៊ុន។ |
