## ទិដ្ឋភាពទូទៅ

ស្លាកគឺជាវិធីដ៏មានអំណាចមួយក្នុងការចាត់ចែង និងរៀបចំកំណត់ត្រានៅក្នុង Blue។ ប្រព័ន្ធស្លាកផ្តល់នូវមុខងារទូលំទូលាយសម្រាប់ការបង្កើត គ្រប់គ្រង និងស្វែងរកស្លាក ជាមួយនឹងការគាំទ្រការប្រតិបត្តិការប្រមូល និងការស្វែងរកកម្រិតខ្ពស់។

## បញ្ជីស្លាក

ទាញយកស្លាកពីគម្រោងជាមួយនឹងជម្រើសការស្វែងរក និងតម្រៀបដែលអាចបត់បែនបាន។

### ឧទាហរណ៍មូលដ្ឋាន

```graphql
query ListTags {
  tagList(
    filter: { 
      projectIds: ["project-123"] 
      excludeArchivedProjects: false 
    }
    first: 50
    orderBy: title_ASC
  ) {
    items {
      id
      uid
      title
      color
      project {
        id
        name
      }
      createdAt
      updatedAt
    }
    pageInfo {
      totalPages
      totalItems
      page
      perPage
      hasNextPage
      hasPreviousPage
    }
    totalCount
  }
}
```

### ឧទាហរណ៍ការស្វែងរកកម្រិតខ្ពស់

```graphql
query FilterTags {
  tagList(
    filter: {
      projectIds: ["project-123", "project-456"]
      search: "urgent"
      colors: ["#ff0000", "#ffaa00"]
      excludeArchivedProjects: true
    }
    orderBy: createdAt_DESC
    first: 20
  ) {
    items {
      id
      title
      color
      todos {
        id
        title
      }
    }
    totalCount
  }
}
```

## បង្កើតស្លាក

បង្កើតស្លាកថ្មីនៅក្នុងគម្រោងមួយ។

```graphql
mutation CreateTag {
  createTag(
    input: {
      title: "High Priority"
      color: "#ff0000"
    }
  ) {
    id
    uid
    title
    color
    project {
      id
      name
    }
    createdAt
  }
}
```

## ធ្វើបច្ចុប្បន្នភាពស្លាក

កែប្រែគុណសម្បត្តិស្លាកដែលមានស្រាប់។

```graphql
mutation UpdateTag {
  editTag(
    input: {
      id: "tag-123"
      title: "Critical Priority"
      color: "#cc0000"
    }
  ) {
    id
    title
    color
    updatedAt
  }
}
```

## លុបស្លាក

យកស្លាកចេញពីប្រព័ន្ធ។ នេះនឹងយកស្លាកចេញពីកំណត់ត្រាទាំងអស់ដែលទាក់ទង។

```graphql
mutation DeleteTag {
  deleteTag(id: "tag-123")
}
```

## ស្លាកកំណត់ត្រា

ភ្ជាប់ស្លាកជាមួយកំណត់ត្រា (todos) ដោយប្រើស្លាកដែលមានស្រាប់ ឬបង្កើតថ្មី។

### ប្រើស្លាកដែលមានស្រាប់

```graphql
mutation TagRecord {
  setTodoTags(
    input: {
      todoId: "todo-123"
      tagIds: ["tag-456", "tag-789"]
    }
  )
}
```

### បង្កើតស្លាកនៅពេលកំណត់ស្លាក

អ្នកអាចបង្កើតស្លាកថ្មីនៅពេលដែលអ្នកផ្តល់ឈ្មោះស្លាក៖

```graphql
mutation TagRecordWithNewTags {
  setTodoTags(
    input: {
      todoId: "todo-123"
      tagIds: ["existing-tag-id"]
      tagTitles: ["New Tag", "Another New Tag"]
    }
  )
}
```

## ការផ្តល់អនុសាសន៍ស្លាក AI

ទទួលបានការផ្តល់អនុសាសន៍ស្លាកដែលមានអំណាច AI សម្រាប់កំណត់ត្រា៖

```graphql
mutation GetTagSuggestions {
  aiTag(
    input: {
      todoId: "todo-123"
      suggestionsCount: 5
    }
  ) {
    suggestions {
      title
      confidence
      color
    }
  }
}
```

## ប៉ារ៉ាម៉ែត្រ​ចូល

### TagListFilter

| ប៉ារ៉ាម៉ែត្រ | ប្រភេទ | ត្រូវការ | ការពិពណ៌នា |
|-----------|------|----------|-------------|
| `projectIds` | [String!] | មិន | ស្វែងរកស្លាកតាម ID ឬ slug នៃគម្រោង |  |
| `excludeArchivedProjects` | Boolean | មិន | បដិសេធស្លាកពីគម្រោងដែលបានស្តារឡើងវិញ |  |
| `search` | String | មិន | ស្វែងរកឈ្មោះស្លាក (មិនគិតពីករណី) |  |
| `titles` | [String!] | មិន | ស្វែងរកតាមឈ្មោះស្លាកជាក់លាក់ |  |
| `colors` | [String!] | មិន | ស្វែងរកតាមពណ៌ជាក់លាក់ (ទ្រង់ទ្រាយ hex) |  |
| `tagIds` | [String!] | មិន | ស្វែងរកតាម ID ស្លាកជាក់លាក់ |  |

### CreateTagInput

| ប៉ារ៉ាម៉ែត្រ | ប្រភេទ | ត្រូវការ | ការពិពណ៌នា |
|-----------|------|----------|-------------|
| `title` | String | មិន | ឈ្មោះស្លាក (បង្កើតដោយស្វ័យប្រវត្តិប្រសិនបើមិនបានផ្តល់) |  |
| `color` | String! | ✅ មែន | ពណ៌ស្លាកក្នុងទ្រង់ទ្រាយ hex (ឧ. "#ff0000") |  |

### EditTagInput

| ប៉ារ៉ាម៉ែត្រ | ប្រភេទ | ត្រូវការ | ការពិពណ៌នា |
|-----------|------|----------|-------------|
| `id` | String! | ✅ មែន | ID នៃស្លាកដែលត្រូវធ្វើបច្ចុប្បន្នភាព |  |
| `title` | String | មិន | ឈ្មោះស្លាកថ្មី |  |
| `color` | String | មិន | ពណ៌ស្លាកថ្មីក្នុងទ្រង់ទ្រាយ hex |  |

### SetTodoTagsInput

| ប៉ារ៉ាម៉ែត្រ | ប្រភេទ | ត្រូវការ | ការពិពណ៌នា |
|-----------|------|----------|-------------|
| `todoId` | String! | ✅ មែន | ID នៃកំណត់ត្រាដែលត្រូវស្លាក |  |
| `tagIds` | [String!] | មិន | ID នៃស្លាកដែលមានស្រាប់ដែលត្រូវអនុវត្ត |  |
| `tagTitles` | [String!] | មិន | ឈ្មោះស្លាកថ្មីដែលត្រូវបង្កើត និងអនុវត្ត |  |

### TagOrderByInput Values

| តម្លៃ | ការពិពណ៌នា |
|-------|-------------|
| `id_ASC` / `id_DESC` | តម្រៀបតាម ID ស្លាក |  |
| `uid_ASC` / `uid_DESC` | តម្រៀបតាមឧបករណ៍ឯកត្ត |  |
| `title_ASC` / `title_DESC` | តម្រៀបតាមអក្សរបឋមតាមឈ្មោះ |  |
| `color_ASC` / `color_DESC` | តម្រៀបតាមតម្លៃពណ៌ |  |
| `createdAt_ASC` / `createdAt_DESC` | តម្រៀបតាមកាលបរិច្ឆេទបង្កើត |  |
| `updatedAt_ASC` / `updatedAt_DESC` | តម្រៀបតាមកាលបរិច្ឆេទកែប្រែចុងក្រោយ |  |

## វាលឆ្លើយតប

### ប្រភេទស្លាក

| វាល | ប្រភេទ | ការពិពណ៌នា |
|-------|------|-------------|
| `id` | ID! | អត្តសញ្ញាណឯកត្តសម្រាប់ស្លាក |  |
| `uid` | String! | អត្តសញ្ញាណឯកត្តដែលងាយស្រួលប្រើ |  |
| `title` | String! | ឈ្មោះ/ឈ្មោះស្លាក |  |
| `color` | String! | ពណ៌ស្លាកក្នុងទ្រង់ទ្រាយ hex |  |
| `project` | Project! | គម្រោងដែលស្លាកនេះជាប់ស្រប |  |
| `todos` | [Todo!]! | កំណត់ត្រាដែលទាក់ទងនឹងស្លាកនេះ |  |
| `createdAt` | DateTime! | ពេលវេលាដែលបានបង្កើតស្លាក |  |
| `updatedAt` | DateTime! | ពេលវេលាដែលបានធ្វើបច្ចុប្បន្នភាពស្លាកចុងក្រោយ |  |

| វាល | ប្រភេទ | ការពិពណ៌នា |

| `totalCount` | Int! | ចំនួនស្លាកសរុបដែលផ្គូផ្គងនឹងតម្រូវការស្វែងរក |  |

| វាល | ប្រភេទ | ការពិពណ៌នា |

| `page` | Int | លេខទំព័របច្ចុប្បន្ន |  |

| `hasNextPage` | Boolean! | តើមានទំព័របន្ទាប់ទេ |  |

## អាជ្ញាប័ណ្ណដែលត្រូវការ

### ការគ្រប់គ្រងស្លាក

| សកម្មភាព | តួនាទីដែលត្រូវការ |
|--------|--------------|
| Create tags | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |
| Edit tags | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |
| Delete tags | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |
| Apply tags to records | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |

### ការមើលឃើញស្លាក

ស្លាកអាចមើលឃើញដោយអ្នកប្រើទាំងអស់ដែលមានការចូលដំណើរការទៅកាន់គម្រោងដែលបានបង្កើតស្លាក។ ការតម្រៀបតាមតួនាទីអាចអនុវត្តសម្រាប់ប្រតិបត្តិការមួយចំនួន។

## ចម្លើយកំហុស

```json
{
  "errors": [{
    "message": "Tag not found",
    "extensions": {
      "code": "NOT_FOUND"
    }
  }]
}
```

```json
    "message": "Color must be in hex format (e.g., #ff0000)",
    "extensions": {
      "code": "BAD_USER_INPUT"
    }
  }]

```

```json
      "code": "FORBIDDEN"
    }
  }]
```

## អនុសាសន៍ល្អបំផុត

1. **ប្រើពណ៌ដែលមានភាពស្របគ្នា**: បង្កើតប្រព័ន្ធកូដពណ៌សម្រាប់ក្រុមរបស់អ្នក
2. **ឈ្មោះពណ៌នាដែលច្បាស់**: ប្រើឈ្មោះស្លាកដែលច្បាស់និងពណ៌នាដែលសមាជិកក្រុមអាចយល់បាន
3. **ប្រតិបត្តិការប្រមូល**: ប្រើ `setTodoTags` ដើម្បីអនុវត្តស្លាកច្រើនក្នុងពេលតែមួយ
4. **ស្វែងរក និងតម្រៀប**: ប្រើមុខងារស្វែងរកដើម្បីរកស្លាកដែលមានស្រាប់មុនពេលបង្កើតថ្មី
5. **ការគ្រប់គ្រងស្លាក**: ពិនិត្យ និងសម្អាតស្លាកដែលមិនបានប្រើជាប្រចាំ
6. **ច្បាប់ពណ៌**: ពិចារណាអំពីការប្រើពណ៌ស្តង់ដាសម្រាប់ប្រភេទស្លាកទូទៅ (ក្រហមសម្រាប់ចាំបាច់, បៃតងសម្រាប់បានបញ្ចប់, ជាដើម)
