យើងផ្តល់ជូននូវ [បណ្ណាល័យ Python](https://pypi.org/project/bluepm/) ដែលអាចប្រើប្រាស់ដើម្បីអន្តរកម្មជាមួយ Blue API។ វាមានការក覆ទាំងស្រុង 100% នៃ Blue API ជាមួយសុវត្ថិភាពប្រភេទ និងការបញ្ចូលកូដនៅក្នុង IDE សម័យទំនើបណាមួយ។

អ្នកអាចតំឡើងដោយការប្រតិបត្តិ៖

```bash
pip install bluepm
```

បន្ទាប់មកអ្នកអាចនាំចូល និងប្រើប្រាស់បណ្ណាល័យក្នុងកូដរបស់អ្នក៖

```python [python]
from bluepm import BlueAPIClient
```

អ្នកអាចកំណត់ព័ត៌មានសម្គាល់អត្តសញ្ញាណរបស់អ្នកជាផ្លាស់ប្តូរ​បរិយាកាស

ដើម្បីប្រើប្រាស់ផ្លាស់ប្តូរ​បរិយាកាស សូមកំណត់អ្វីខ្លះខាងក្រោមមុននឹងប្រតិបត្តិការ script របស់អ្នក៖

- BLUE_TOKEN_ID: អត្តសញ្ញាណកូដ API របស់ Blue
- BLUE_SECRET_ID: អត្តសញ្ញាណសម្ងាត់ API របស់ Blue
- BLUE_COMPANY_ID: អត្តសញ្ញាណក្រុមហ៊ុន Blue របស់អ្នក (ជាជម្រើស)
- BLUE_PROJECT_ID: អត្តសញ្ញាណគម្រោង Blue របស់អ្នក (ជាជម្រើស)

ឬ អ្នកអាចផ្តល់ពួកវាទៅកាន់កសាង BlueAPIClient ដោយផ្ទាល់៖

```python [python]
client = BlueAPIClient(
    token_id="your_token_id",
    secret_id="your_secret_id",
    company_id="your_company_id" #optional
    project_id = "your_project_id" #optional
)
```

មួយដងដែលអ្នកបានធ្វើរួចនេះ អ្នកអាចប្រើប្រាស់បណ្ណាល័យដើម្បីអន្តរកម្មជាមួយ Blue API។  យើងមានជំនួយខ្លះៗដើម្បីសម្រួលការងារទូទៅ ប៉ុន្តែ API ទាំងមូលត្រូវបានបង្ហាញដើម្បីអនុញ្ញាតឱ្យមានការអន្តរកម្មណាមួយដែលអ្នកត្រូវការ។

ឧទាហរណ៍ ដើម្បីទទួលបានបញ្ជីគម្រោងទាំងអស់ដែលអ្នកមានសិទ្ធិចូលប្រើ អ្នកអាចប្រើកូដខាងក្រោម៖

```python [python]
projects = client.projects.list(company_id)
```

ដើម្បីទទួលបានបញ្ជីបញ្ជីសម្រាប់គម្រោង អ្នកអាចប្រើកូដខាងក្រោម៖

```python [python]
todo_lists = client.get_todo_lists(project_id)
```
