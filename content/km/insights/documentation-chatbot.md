នៅ Blue, យើងតែងតែស្វែងរកវិធីដើម្បីធ្វើឱ្យជីវិតកាន់តែងាយស្រួលសម្រាប់អតិថិជនរបស់យើង។ យើងមាន [ឯកសារពិស្តារពីមុខងារទាំងអស់](https://documentation.blue.cc), [វីដេអូ YouTube](https://www.youtube.com/@workwithblue), [គន្លឹះ និងយុទ្ធសាស្ត្រ](/insights/tips-tricks), និង [ឆានែលគាំទ្រផ្សេងៗ](/support)។

យើងបានតាមដានការអភិវឌ្ឍន៍ AI (បញ្ញាសិប្បនិម្មិត)យ៉ាងខិតខំ ពីព្រោះយើងចូលចិត្ត [ការប្រព្រឹត្តកម្មវិធីគ្រប់គ្រងគម្រោង](/platform/features/automations)។ យើងក៏បានចេញផ្សាយមុខងារ ដូចជា [AI Auto Categorization](/insights/ai-auto-categorization) និង [AI Summaries](/insights/ai-content-summarization) ដើម្បីធ្វើឱ្យការងារកាន់តែងាយស្រួលសម្រាប់អតិថិជនរាប់ពាន់នាក់របស់យើង។

អ្វីមួយដែលច្បាស់ណាស់គឺ AI នេះគឺនៅទីនេះដើម្បីនៅ ហើយវានឹងមានឥទ្ធិពលអស្ចារ្យនៅក្នុងឧស្សាហកម្មភាគច្រើន ហើយការគ្រប់គ្រងគម្រោងគឺមិនមានការលំបាកទេ។ ដូច្នេះយើងបានសួរខ្លួនឯងថា តើយើងអាចប្រើប្រាស់ AI ដើម្បីជួយដល់ជីវិតពេញលេញរបស់អតិថិជនពីការរកឃើញ, ការលក់មុន, ការចូលរួម, និងក៏ជាមួយសំណួរដែលកំពុងបន្ត។

ចម្លើយគឺច្បាស់ណាស់៖ **យើងត្រូវការប្រព័ន្ធ Chatbot AI ដែលត្រូវបានបណ្តុះបណ្តាលលើឯកសាររបស់យើង។**

អនុវត្តន៍ទៅ៖ *អង្គភាព* ទាំងអស់គួរតែមាន Chatbot មួយ។ វាជាវិធីដ៏អស្ចារ្យសម្រាប់អតិថិជនក្នុងការទទួលបានចម្លើយភ្លាមៗចំពោះសំណួរទូទៅ ដោយមិនចាំបាច់ចូលទៅក្នុងទំព័រដែលមានឯកសារមានកម្រិតខ្ពស់ ឬគេហទំព័ររបស់អ្នក។ សារៈសំខាន់នៃ Chatbot នៅក្នុងគេហទំព័រលក់ទំនិញសម័យថ្មីមិនអាចនិយាយអស់បានទេ។

![](/insights/ai-chatbot-regular.png)

សម្រាប់ក្រុមហ៊ុនកម្មវិធីជាក់លាក់ មិនគួរតែចាត់ទុកគេហទំព័រលក់ទំនិញជាអ្វីដែលបំបែក — វាជាផ្នែកមួយនៃផលិតផលរបស់អ្នក។ នេះគឺដោយសារវាអាចសម្របសម្រួលទៅក្នុងជីវិតអតិថិជនទូទៅ៖

- **ការយល់ដឹង** (ការរកឃើញ): នេះគឺជាទីកន្លែងដែលអតិថិជនអាចស្គាល់ផលិតផលដ៏អស្ចារ្យរបស់អ្នក។ Chatbot របស់អ្នកអាចជាមគ្គុទេសក៍មិត្តភាពរបស់ពួកគេ បង្ហាញពួកគេពីមុខងារ និងអត្ថប្រយោជន៍សំខាន់ៗភ្លាមៗ។
- **ការពិចារណា** (ការអប់រំ): ឥឡូវនេះពួកគេមានចំណាប់អារម្មណ៍ ហើយចង់រៀនបន្ថែម។ Chatbot របស់អ្នកក្លាយជាគ្រូបង្រៀនផ្ទាល់ខ្លួនរបស់ពួកគេ បង្ហាញព័ត៌មានដែលសមស្របទៅនឹងតម្រូវការនិងសំណួរពិសេសរបស់ពួកគេ។
- **ការទិញ/ការបម្លែង**: នេះគឺជាពេលវេលានៃការពិត - នៅពេលដែលអ្នកចង់ទិញ និងក្លាយជាអតិថិជន។ Chatbot របស់អ្នកអាចធ្វើឱ្យមានការលំបាកចុងក្រោយកាត់បន្ថយ, ឆ្លើយសំណួរដែល "មុនពេលខ្ញុំទិញ" និងប្រហែលជាបញ្ចូលការផ្តល់ជូនដ៏ស្រស់ស្អាតដើម្បីបញ្ចប់ការទិញ។
- **ការចូលរួម**: ពួកគេបានទិញហើយឥឡូវនេះធ្វើដូចម្តេច? Chatbot របស់អ្នកក្លាយជាសមាជិកជួយ ដែលនាំអ្នកប្រើថ្មីតាមដំណើរការកំណត់, បង្ហាញពួកគេពីការប្រើប្រាស់, និងធានាថាពួកគេមិនមានអារម្មណ៍ខ្វះខាតនៅក្នុងពិភពរបស់ផលិតផលរបស់អ្នក។
- **ការរក្សាទុក**: ការរក្សាអតិថិជនឱ្យមានសុភមង្គលគឺជាឈ្មោះនៃហ្គេម។ Chatbot របស់អ្នកមានស្រាប់ 24/7, រៀបចំដើម្បីដោះសោបញ្ហា, ផ្តល់គន្លឹះ និងយុទ្ធសាស្ត្រ, និងធានាថាអតិថិជនរបស់អ្នកមានអារម្មណ៍ស្រឡាញ់។
- **ការពង្រីក**: ពេលវេលាសម្រាប់កម្រិតឡើង! Chatbot របស់អ្នកអាចផ្តល់អនុសាសន៍មុខងារថ្មីៗ, ការលក់បន្ថែម, ឬការលក់ឆ្លងកាត់ដែលសមស្របទៅនឹងរបៀបដែលអតិថិជនកំពុងប្រើផលិតផលរបស់អ្នក។ វាដូចជាការមានអ្នកលក់ដ៏ឆ្លាតវៃមួយដែលមិនមានការបង្ហាញត្រូវបានរៀបចំជាប់ជាមួយ។
- **ការគាំទ្រ**: អតិថិជនដែលមានសុភមង្គលក្លាយជាអ្នកគាំទ្រធំជាងគេរបស់អ្នក។ Chatbot របស់អ្នកអាចលើកទឹកចិត្តអ្នកប្រើដែលមានសុភមង្គលឱ្យផ្សព្វផ្សាយពាក្យ, ទុកមតិ, ឬចូលរួមក្នុងកម្មវិធីយោង។ វាដូចជាការមានម៉ាស៊ីនលើកទឹកចិត្តដែលបានបង្កើតចូលទៅក្នុងផលិតផលរបស់អ្នក!

## ការសម្រេចចិត្តបង្កើតប្រាក់

មុនពេលដែលយើងសម្រេចចិត្តអនុវត្ត Chatbot AI, សំណួរធំបន្ទាប់គឺ៖ បង្កើតឬទិញ? ជាក្រុមតូចដែលមានការផ្តោតលើផលិតផលសំខាន់របស់យើង យើងទូទៅចូលចិត្តដំណោះស្រាយ "ជាសេវាកម្ម" ឬវេទិកាដែលមានប្រជាប្រិយ។ យើងមិននៅក្នុងអាជីវកម្មនៃការបង្កើតកង់ឡើងវិញសម្រាប់គ្រប់ផ្នែកនៃប្រព័ន្ធបច្ចេកវិទ្យារបស់យើងទេ។

ដូច្នេះយើងបានរៀបចំដៃរបស់យើង ហើយចូលទៅក្នុងទីផ្សារ ស្វែងរកដំណោះស្រាយ Chatbot AI ដែលមានការទូទាត់ និងមានប្រភពបើក។

តម្រូវការរបស់យើងគឺត្រឹមត្រូវ ប៉ុន្តែគ្មានការចរចា៖

- **បទពិសោធន៍ដែលមិនមានម៉ាក**: Chatbot នេះមិនមែនគ្រាន់តែជាឧបករណ៍ដែលមានភាពស្រួលទេ; វាកំពុងទៅលើគេហទំព័រលក់ទំនិញរបស់យើង ហើយនៅទីបញ្ចប់ក្នុងផលិតផលរបស់យើង។ យើងមិនចង់ផ្សព្វផ្សាយម៉ាករបស់អ្នកដទៃនៅក្នុងអចលនទ្រព្យឌីជីថលរបស់យើងទេ។
- **UX ធ្វើឱ្យល្អ**: សម្រាប់អតិថិជនដែលមានសក្តានុពលជាច្រើន Chatbot នេះអាចជាចំណុចទំនាក់ទំនងដំបូងរបស់ពួកគេជាមួយ Blue។ វាកំណត់សំឡេងសម្រាប់ការយល់ឃើញរបស់ពួកគេអំពីក្រុមហ៊ុនរបស់យើង។ ចូរអនុវត្តន៍ទៅ៖ ប្រសិនបើយើងមិនអាចធ្វើឱ្យ Chatbot មានភាពត្រឹមត្រូវលើគេហទំព័ររបស់យើងទេ តើយើងអាចរំពឹងថាអតិថិជននឹងទុកចិត្តយើងជាមួយគម្រោង និងដំណើរការសំខាន់របស់ពួកគេបានយ៉ាងដូចម្តេច?
- **តម្លៃសមរម្យ**: ជាមួយមូលដ្ឋានអ្នកប្រើធំ និងផែនការដើម្បីបញ្ចូល Chatbot ទៅក្នុងផលិតផលសំខាន់របស់យើង យើងត្រូវការដំណោះស្រាយដែលមិនធ្វើឱ្យមានការចំណាយខ្ពស់នៅពេលដែលការប្រើប្រាស់កើនឡើង។ យើងចង់បានជម្រើស **BYOK (Bring Your Own Key)**។ នេះនឹងអនុញ្ញាតឱ្យយើងប្រើកូនសោ OpenAI ឬសេវាកម្ម AI ផ្សេងទៀតរបស់យើង ដោយបង់ប្រាក់សម្រាប់ការចំណាយអថេរ តែមិនមែនជាការកើនឡើងទៅអ្នកផ្គត់ផ្គង់ជាត្រីទីបីដែលមិនបានដំណើរការម៉ូដែលទេ។

- **[OpenAssistantGPT](https://www.openassistantgpt.io/)**: ការមើលទៅនេះម្តងទៀតពីមុខងារបើកប្រភពនេះត្រូវការឱ្យយើងបង្កើតឧបករណ៍មួយចំនួនដែលយើងមិនចង់ធ្វើ ដោយសារយើងចង់អនុវត្តន៍ឱ្យបានច្រើនបំផុតលើ OpenAI's Assistants API។

## ការបង្កើត ChatBot របស់យើង

ហើយដូច្នេះ ដោយមិនអាចស្វែងរកអ្វីដែលសមស្របជាមួយតម្រូវការទាំងអស់របស់យើង យើងបានសម្រេចចិត្តបង្កើត Chatbot AI របស់យើងដែលអាចភ្ជាប់ជាមួយ OpenAI Assistants API។ នេះ នៅចុងក្រោយ បានបង្ហាញថាជាការងារងាយស្រួល!

គេហទំព័ររបស់យើងប្រើ [Nuxt3](https://nuxt.com), [Vue3](https://vuejs.org/) (ដែលជាស៊ុមដូចគ្នាជាមួយវេទិកា Blue), និង [TailwindUI](https://tailwindui.com/)។

ជំហានដំបូងគឺការបង្កើត API (Application Programming Interface) នៅក្នុង Nuxt3 ដែលអាច "និយាយ" ជាមួយ OpenAI Assistants API។ នេះគឺចាំបាច់ ពីព្រោះយើងមិនចង់ធ្វើអ្វីៗទាំងអស់នៅផ្នែកមុខទេ ពីព្រោះវានឹងបង្ហាញកូនសោ OpenAI របស់យើងទៅកាន់ពិភពលោក ដែលមានសក្តានុពលសម្រាប់ការប្រើប្រាស់ខុស។

API ខាងក្រោយរបស់យើងធ្វើជាមធ្យមសុវត្ថិភាពរវាងកម្មវិធីរុករករបស់អ្នកនិង OpenAI។ នេះគឺអ្វីដែលវាធ្វើ៖

- **ការគ្រប់គ្រងការជជែក**: វាបង្កើត និងគ្រប់គ្រង "ខ្សែ" សម្រាប់ការជជែករៀងរាល់មួយ។ គិតថាខ្សែជាសមាជិកជជែកដ៏អន្ដរាគមន៍ដែលចាំអ្វីៗដែលអ្នកបាននិយាយ។
- **ការដោះសោសារ**: នៅពេលដែលអ្នកផ្ញើសារ API របស់យើងនឹងបន្ថែមវាទៅក្នុងខ្សែត្រឹមត្រូវ ហើយសួរឱ្យជំនួយ OpenAI បង្កើតចម្លើយមួយ។
- **ការរង់ចាំឆ្លាតវៃ**: ជំនួយ API របស់យើងមិនធ្វើឱ្យអ្នកមើលកញ្ចក់បង្ហាញនូវអ្វីដែលមិនមានទេ API របស់យើងត្រួតពិនិត្យជាមួយ OpenAI រៀងរាល់វិនាទីដើម្បីមើលថាចម្លើយរបស់អ្នកបានរៀបចំរួចហើយ។ វាដូចជាការមានអ្នករង់ចាំដែលរក្សាទំនាក់ទំនងលើការបញ្ជាទិញរបស់អ្នកដោយមិនបង្ហ扰ម្ចាស់ម្ហូបរៀងរាល់ពីរវិនាទី។
- **សុវត្ថិភាពជាមុន**: ដោយការគ្រប់គ្រងអ្វីៗទាំងនេះនៅលើម៉ាស៊ីនមេ យើងរក្សាទុកទិន្នន័យរបស់អ្នក និងកូនសោ API របស់យើងឱ្យមានសុវត្ថិភាព។

បន្ទាប់មកមានផ្នែកមុខ និងបទពិសោធន៍អ្នកប្រើ។ ដូចដែលបានពិភាក្សមុននេះ វាជាសារៈសំខាន់ *យ៉ាងខ្លាំង* ពីព្រោះយើងមិនទទួលបានឱកាសទីពីរនៅក្នុងការបង្កើតការបង្ហាញដំបូងទេ!

នៅក្នុងការរចនាអំពី Chatbot របស់យើង យើងបានផ្តល់អារម្មណ៍យ៉ាងម៉ត់ចត់ទៅលើបទពិសោធន៍អ្នកប្រើ ដើម្បីធានាថាការប្រតិបត្តិគ្រប់យ៉ាងមានភាពរលូន, ងាយស្រួល, និងបង្ហាញពីការប្តេជ្ញារបស់ Blue ទៅកាន់គុណភាព។ មុខមាត់ Chatbot ចាប់ផ្តើមដោយរង្វង់ខៀវស្រស់ស្អាត, ប្រើ [HeroIcons សម្រាប់រូបតំណាងរបស់យើង](https://heroicons.com/) (ដែលយើងប្រើនៅទូទាំងគេហទំព័រ Blue) ដើម្បីធ្វើជាឧបករណ៍បើក Chatbot របស់យើង។ ការជ្រើសរើសរចនានេះធានាថាមានភាពស្របគ្នា និងការទទួលស្គាល់ម៉ាកភ្លាមៗ។
