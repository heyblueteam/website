## 桌面应用

将 Blue 安装为桌面应用的优势包括：

- 更快的性能
- 没有浏览器界面包围 Blue 界面
- 可将 Blue 保存到 Mac Dock 或 Windows 桌面快捷方式
- 真正的全屏模式
- 原生桌面通知
- 自动更新

![Download Apps1](/docs/download-apps1.webp "Download Apps1")

要将 Blue 安装为桌面应用，您需要使用基于 Chromium 的浏览器，例如：

- [Google Chrome](https://www.google.com/chrome/)
- [Brave](https://brave.com/)
- [Microsoft Edge](https://www.microsoft.com/edge)
- [Opera](https://www.opera.com/)
- [Vivaldi](https://vivaldi.com/)
- [Chromium](https://www.chromium.org/)

安装上述任一浏览器后，只需按常规方式[登录 Blue](https://app.blue.cc/)。然后您会注意到在 URL 地址栏的右侧有一个安装图标：

![Download App2](/docs/download-apps2.png "Download App2")


点击浏览器地址栏中的安装图标来安装 Blue 桌面应用。该图标通常显示为小电脑或加号符号。

点击后，系统会引导您完成简单的安装过程。请仔细按照提示操作，确保 Blue 应用正确安装到您的计算机上。安装完成后，您就可以直接从计算机访问 Blue，无需打开网页浏览器。

您也可以观看此操作视频获取说明：

<youtube url="https://www.youtube.com/watch?v=TZye8hnk4ac" />


对于 Mac 用户，Blue 应用安装完成后，您可以将其添加到 Dock。这样您就可以直接从 Dock 快速轻松地访问 Blue。操作方法是：打开 Blue 应用，右键点击（或按住 Control 键点击）Dock 中的图标。从出现的菜单中选择"选项"，然后选择"在 Dock 中保留"。这将确保即使您关闭应用后，Blue 图标仍保留在 Dock 中，方便将来更快更便捷地访问。

![Download App3](/docs/download-apps3.png "Download App3")

Windows 用户也可以通过在桌面创建 Blue 应用快捷方式来获得快速访问的便利。安装 Blue 后，导航到系统中应用的安装位置。右键点击 Blue 应用图标，选择"创建快捷方式"。然后选择将此快捷方式放置在桌面上。这将在桌面创建一个易于访问的图标，您可以双击启动 Blue，无需浏览系统菜单或文件资源管理器。

## PWA

Blue 网页应用是一个 PWA（渐进式网页应用），这意味着它可以直接在 iOS 和 Android 设备上安装为应用，适用于无法访问 App Store 或 Google Play 商店，或希望在移动设备上使用完整 Blue 功能的情况。

### iOS 上的 PWA

1. **打开 Chrome**：在您的 iOS 设备上启动 Chrome 应用。
2. **导航到 Blue**：在 Chrome 地址栏中输入 Blue 网页应用的 URL (https://app.blue.cc)。
3. **访问分享菜单**：Blue 网页应用加载完成后，点击屏幕底部的分享图标（带向上箭头的矩形）。
4. **添加到主屏幕**：在分享菜单中，向下滚动并选择"添加到主屏幕"。这将允许您安装 Blue PWA。
5. **命名您的 PWA**：在添加到主屏幕之前，系统会提示您为快捷方式命名。您可能希望将其命名为"Blue"以便识别。
6. **完成安装**：点击屏幕右上角的"添加"。Blue PWA 图标现在将出现在您的主屏幕上，就像任何其他应用图标一样。
7. **启动 Blue PWA**：从主屏幕点击 Blue 图标打开 PWA，像使用原生应用一样使用 Blue。

### Android 上的 PWA

1. **打开 Chrome**：在您的 Android 设备上启动 Chrome 应用。
2. **前往 Blue**：在 Chrome 地址栏中输入 Blue 网页应用的 URL (https://app.blue.cc)。
3. **访问菜单**：点击浏览器右上角的三个点打开菜单。
4. **安装应用**：在菜单中查找"安装"选项。这也可能显示为"添加到主屏幕"或表示方框内向下箭头的图标。
5. **确认安装**：会出现一个提示询问您是否要安装应用。点击"安装"或"添加"确认。
6. **打开 Blue PWA**：安装完成后，Blue 图标将出现在您的主屏幕或应用抽屉中，就像原生应用一样。
7. **使用 Blue**：只需点击 Blue 图标即可使用该应用，就像在桌面上一样，但具有移动体验的额外优势。
