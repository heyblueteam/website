Blue 有一个 API（应用程序编程接口），因此如果需要，您可以构建自己的集成。

**Blue 提供 100% 的 API 覆盖**。这意味着您可以在 [我们的任何应用程序](/docs/start-guide/download-apps) 中所做的任何操作，都可以通过编程方式完成。我们有一个强大的 [GraphQL](https://graphql.org/) API，提供完整的游乐场，您可以在其中测试查询，并且有完整的文档。

::callout
