[Make.com](https://www.make.com) 是一个集成平台，帮助连接不同的网络应用程序并自动化工作流。Blue **官方集成** 了 Make.com，使得在 2000 多个应用程序之间实现无缝自动化——无需编码！

![Blue Make.com 模块概述](/docs/blue_make_modules_overview.png "Overview of available Blue modules in Make.com")

## 用例

以下是一些通过 Make.com 集成扩展 Blue 功能的强大方式：
Blue 的 Make.com 集成使得在各种业务功能之间实现强大的自动化。对于客户体验，您可以将 Blue 与 Intercom、HubSpot 和 Zendesk 等工具连接，以创建全面的客户记录、同步互动并通过 Slack 和 Microsoft Teams 维护实时沟通。在项目管理中，Blue 与 Jira、GitHub、Trello 和 Monday.com 集成，以同步任务、跟踪进度并自动化通知。

对于营销和销售，Blue 与 Mailchimp、Stripe 和 Calendly 连接，以跟踪活动、管理支付并自动化会议工作流。通过与 Google Drive、Dropbox、OneDrive 和 SharePoint 的集成，文档管理得以简化，而像 Slack、Gmail 和 WhatsApp 这样的沟通工具则通过 Blue 的中央任务管理系统统一。通过与 Google Sheets、Power BI、Airtable 和 Tableau 的连接，数据分析能力得以增强，针对金融服务、电子商务和远程团队管理的行业特定解决方案也可用。

## 连接并验证您的 Blue 账户

要安全地将 Blue 与 Make.com 连接：

1. 登录到您的 Blue 账户并打开您的个人资料 → 转到 **API** 标签
2. 点击 **生成令牌**，提供一个名称（和可选的过期时间）
3. 复制以下两项：
   - **令牌 ID** (`x‑bloo‑token‑id`)
   - **密钥** (`x‑bloo‑token‑secret`)

::callout
 ---
icon: lightbulb
---
**重要说明** <br/>
密钥仅显示一次——请安全存储！
::

5. 在 Make.com 中创建新连接时，输入：
   - **令牌 ID**
   - **密钥**

## 模块参数参考

### 监视事件
- 事件（可选）：多选事件类型
- 公司 ID（可选）：多选公司 ID 或别名
- 项目 ID（可选）：多选项目 ID 或别名

### 创建记录
- 公司 ID：公司 ID 或别名
- 项目 ID：项目 ID 或别名
- 列表：列表 ID
- 记录名称：记录的名称
- 默认字段：默认值
- 单一日期：截止日期
- 日期范围：开始和截止日期
- 被分配人：多选用户
- 标签：多选标签
- 描述：文本字段，支持 HTML
- 检查清单：检查清单标题的数组

**支持的自定义字段：**
| 字段类型 | 描述 |
|------------|-------------|
| 单行文本 | 单行文本字段，带可选的验证和字符限制 |
| 多行文本 | 多行文本字段，支持富文本和格式选项 |
| 单选 | 下拉菜单，单选选项和自定义选项 |
| 多选 | 下拉菜单，多选选项和自定义选项 |
| 位置 | 纬度和经度，带地图视图和地址查找 |
| 复选框 | 真/假切换，带可选的默认值和自定义标签 |
| 国家 | 两个字母的 ISO 国家代码选择器，带国家名称 |
| 货币 | 带货币符号和汇率支持的数字字段（例如，USD200，200USD，KHR4000） |
| 日期 | 日期选择器，带可选的时间组件和时区支持 |
| 电话 | 电话号码字段，支持 E.164、国际格式或国家代码 + 号码，带验证 |
| 电子邮件 | 电子邮件字段，带格式验证和域名验证 |
| 星级评分 | 1-5 的数字评分，带可视星星和自定义标签 |
| 百分比 | 带百分号（0-100）和小数精度的数字字段 |
| URL | 网络地址字段，带格式验证和预览 |
| 数字 | 带可选小数位和单位显示的数字字段 |
| 公式 | 基于其他字段的计算字段，带数学运算 |
| 唯一 ID | 带自定义前缀和格式的自动生成标识符 |
| 参考 | 链接到另一个记录，带搜索和过滤选项 |
| 查找 | 动态字段，从引用记录中提取数据，带映射选项 |
| 时间持续 | 以小时和分钟表示的时间跨度，带自定义显示格式 |
