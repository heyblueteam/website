Blue 中的记录非常灵活，可用于多种用途，例如跟踪客户互动、销售、项目里程碑或库存。这种适应性允许您自定义 Blue 以满足您的业务需求，使其成为组织重要信息的宝贵工具。

## 重命名记录

您可以轻松地在 Blue 中重命名记录以满足您的需求。只需前往项目设置，将名称更改为您想要的任何名称，例如任务或分配。此更改将应用于整个项目。

以下是您可以用于记录的一些名称示例：

- 任务 / 任务
- 分配 / 分配
- 步骤 / 步骤
- 问题 / 问题
- 工单 / 工单
- 线索 / 线索
- 交易 / 交易
- 案例 / 案例
- 机会 / 机会
- 客户 / 客户
- 产品 / 产品
- 订单 / 订单

请记住包含您所选名称的单数和复数形式。这有助于系统根据上下文显示正确的术语。

自定义记录名称允许您使用适合您的行业或个人偏好的术语，使平台对您的团队来说更加简单直观。

![Records 1](/docs/Record_1.png "Records 1")

记录可以代表各种实体，例如：

- 客户互动
- 销售记录
- 项目里程碑
- 库存项目

这种灵活性确保 Blue 可以定制以适应不同的业务流程和运营需求，使其成为组织和管理关键数据的强大工具。
