## 项目如何工作

Blue 中的每个项目都是完全独立的。只有被明确邀请加入项目的用户才能查看数据并进行协作。您可以在一个 Blue 账户中创建无限数量的项目。

在 Blue 中，可以从两个角度来理解项目：

1. **项目**：项目通常有明确的开始和结束时间，专注于特定的目标或事件。例如：举办活动、品牌重塑或设计网站。这些项目以目标为导向，有既定的目标和时间表。一旦完成，它们通常会被[归档](/docs/projects/archiving-projects)。
2. **流程**：Blue 也支持没有固定结束日期的持续流程。这些可以包括客户服务、帮助台运营、销售跟踪和招聘等任务。这些项目需要定期管理和组织，但本质上是连续的。

## 项目设置

[项目管理员](/docs/user-management/roles/project-administrator)会在顶部栏的项目名称旁边看到一个设置图标。

</video>

有多个可用的设置选项：

- **编辑项目设置**：您可以更改项目名称、编写项目描述、添加项目照片或图标，还可以将[记录](/docs/records/introduction)的名称更改为其他名称（例如：任务、销售）
- **设置图标和颜色**：您可以为项目选择图标和颜色；这将在侧边栏中显示给项目中的所有用户。
- **管理**：这会打开一个子菜单，您可以在其中编辑项目中的可用功能和[自定义字段](/docs/custom-fields/introduction)。
- [转换为模板](/docs/projects/templates)：这允许将项目转换为可重复使用的模板。
- [复制](/docs/projects/copying-projects)：允许您复制项目
- [归档](/docs/projects/archiving-projects)：归档项目以供将来参考，将其转换为只读项目。
- [导入](/docs/integrations/csv-import) **和** [导出](/docs/integrations/csv-export)：允许从项目中导入和导出记录
- **OAuth 连接**：允许您将项目连接到外部服务，如 Github 或 Quickbooks。
- **删除**：允许永久删除项目。
