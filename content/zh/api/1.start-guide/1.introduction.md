Blue API 通过现代的 [GraphQL](https://graphql.org/) 接口提供对 Blue 的流程管理平台的完全访问。我们的 API 受到 **17,000+ 客户** 的信任，每年处理 **数十亿个 API 请求**，为关键任务工作流提供支持，具有 **[99.99% 的正常运行时间](/platform/status)**。

## 为什么选择 Blue API？

- **100% 功能覆盖** - 您在 Blue 的用户界面中可以做的所有事情，您都可以通过 API 实现
- **企业级准备** - 在大规模环境中经过实战检验，具有全面的速率限制和安全性
- **开发者友好** - GraphQL 提供您所需的确切数据，不多也不少
- **实时更新** - WebSocket 订阅实现即时数据同步

## 快速开始

准备好进行您的第一次 API 调用了吗？这就是它的简单之处：

```graphql
query MyProjects {
  projectList(filter: { companyIds: ["your-company-id"] }) {
    items {
      id
      name
      updatedAt
    }
  }
}
```

**[在我们的 GraphQL Playground 中尝试 →](https://api.blue.cc/graphql)**

- 将 Blue 与外部 CRM、ERP 和专业工具连接
