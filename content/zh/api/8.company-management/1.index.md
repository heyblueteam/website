---
title: 创建公司
description: 如何使用 Blue API 创建新公司。
---

要创建新公司，您可以使用以下变更：

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

请注意，创建新公司可能会影响您 Blue 账户的计费，因为每个公司都是单独计费的。

以下是 `createCompany` 变更的选项及其描述：

| 选项 | 类型 | 描述 |
|------|------|------|
| name | String | 新公司的名称。此字段为必填项。 |  |
| slug | 字符串 | 公司名称的 URL 友好版本。此名称用于公司的 URL，并且应该是唯一的。此字段为必填项，并将作为公司的唯一标识符。 |
