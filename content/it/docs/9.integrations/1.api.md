Blue ha un'API (Interfaccia di Programmazione delle Applicazioni), quindi puoi costruire le tue integrazioni se necessario.

**Blue ha una copertura API del 100%**. Questo significa che qualsiasi cosa tu possa fare in [una delle nostre app](/docs/start-guide/download-apps), puoi farla programmaticamente. Abbiamo una potente API [GraphQL](https://graphql.org/) che ha un playground completo dove puoi testare le tue query ed è completamente documentata.
