Zapier è una piattaforma di integrazione che aiuta a connettere diverse app web e automatizzare i flussi di lavoro.

**Casi d'uso**

Ecco alcuni dei modi più popolari in cui Blue è integrato con altre app tramite Zapier:

**_Sincronizza i record di Blue con Google Sheets_**

Man mano che i record di Blue vengono creati o aggiornati, aggiungili automaticamente come nuove righe in un Google Sheet per reportistica centralizzata.

**_Invia conferma d'ordine dai record di Blue_**

Quando selezioni un determinato campo personalizzato su Blue, invia automaticamente un'email per confermare l'ordine con il tuo cliente.

**_Aggiungi commenti di Blue a Slack_**

Ricevi notifiche in tempo reale di Blue pubblicate nei giusti canali Slack in modo che il tuo team sia a conoscenza degli aggiornamenti importanti dei record.

## Vantaggi chiave

**Automazione tra strumenti**: Gli Zap creati in Zapier possono inviare e ricevere automaticamente dati tra Blue e migliaia di altre app. Questo elimina il lavoro manuale di spostamento dei dati tra i sistemi.

**Integrazioni in tempo reale**: Utilizzando i trigger webhook di Blue, gli Zap reagiscono istantaneamente a eventi in Blue come un nuovo record, un commento aggiunto, un cambiamento di stato, ecc. Questo consente la sincronizzazione in tempo reale e l'automazione dei processi.

**Facile da costruire e mantenere**: Gli Zap forniscono un modo intuitivo e senza codice per integrare gli strumenti. Non è necessario alcun lavoro di sviluppo. E continuano a funzionare senza manutenzione continua.

1. Accedi al tuo account Blue nelle impostazioni del profilo
2. Vai alle impostazioni del profilo e clicca su "Webhook"
3. Clicca su "Aggiungi Webhook"
4. Dai un nome al webhook e inserisci l'URL del webhook di Zapier
5. Scegli quali eventi devono attivare il webhook
6. Clicca su "Crea Webhook"

## Da Blue a Zapier

1. Accedi al tuo account Zapier
2. Clicca su Crea uno Zap  
3. Per il primo passaggio, cerca e seleziona l'app di trigger ["Webhooks by Zapier"](https://zapier.com/apps/webhook)
4. Configuralo per accettare payload dal tuo URL del webhook di Blue
5. Per il secondo passaggio, cerca e seleziona l'app di destinazione a cui inviare i dati
6. Mappa i campi del payload JSON ai campi di input dell'app di destinazione
7. Attiva lo Zap

Ora, quando si verificano gli eventi specificati in Blue, il payload del webhook verrà inviato a Zapier che passerà automaticamente i dati alla tua app integrata secondo le impostazioni dello Zap.

## Da Zapier a Blue

Per inviare dati da altre app a Blue, segui questi passaggi:

**1. Ottieni le credenziali API di Blue**
- Vai su Impostazioni di Blue > Token API
- Crea un nuovo token o utilizza le credenziali esistenti:
  - ID Token
  - Segreto Token
  - ID Azienda
  - ID Progetto

**2. Crea uno Zap in Zapier**
1. Configura la tua app di trigger (ad es., Google Sheets, Slack, ecc.)
2. Aggiungi "Webhooks by Zapier" come app di azione
3. Scegli il tipo di evento "Richiesta personalizzata"

**3. Configura il Webhook**
- Metodo: POST
- URL: `https://api.blue.cc/graphql` (o endpoint beta se necessario)
- Passaggio dati: FALSO
- Non appiattire: NO

**4. Imposta le intestazioni:**
```json
{
  "Content-Type": "application/json",
  "X-Bloo-Token-ID": "Your_Token_ID",
  "X-Bloo-Token-Secret": "Your_Token_Secret", 
  "X-Bloo-Company-ID": "Your_Company_ID",
  "X-Bloo-Project-ID": "Your_Project_ID"
}
```

**5. Configura la Query (Esempio di Mutazione):**
```json
{
  "query": "mutation CreateTodo($input: CreateTodoInput!) {\n  createTodo(input: $input) {\n    id\n    title\n    customFields {\n      value\n    }\n  }\n}",
  "variables": {
    "input": {
      "todoListId": "your-todo-list-id",
      "title": "New task from Zapier",
      "customFields": [
        {
          "customFieldId": "priority-field-id",
          "value": "High"
        },
        {
          "customFieldId": "due-date-field-id",
          "value": "2024-03-01"
        }
      ]
    }
  }
}
```

**6. Testa e Attiva**
- Usa il Playground API di Blue per testare prima le mutazioni
- Mappa i valori dinamici dalle app di trigger utilizzando i campi dati di Zapier
- Abilita lo Zap dopo il test riuscito
