Blue ti consente di utilizzare Google Maps per la visualizzazione della mappa invece della mappa predefinita di OpenStreetMap. Google Maps spesso ha mappe più dettagliate di alcune aree e dati di posizione più accurati. Tuttavia, questo richiede una chiave API di Google Maps che deve essere inserita nelle impostazioni aziendali dal proprietario dell'azienda.

Quando passi a Google Maps, questo influisce su tutte le visualizzazioni della mappa in Blue, inclusa la mappa aziendale, la [visualizzazione della mappa nei progetti](/docs/views/map) e il [campo personalizzato della posizione](/docs/custom-fields/location-map).

Per utilizzare Google Maps in Blue, è necessario attivare le seguenti API nel Google Cloud Console:
- [Maps Tiles API](https://console.cloud.google.com/apis/library/tile.googleapis.com): Questa è necessaria per rendere le tessere della mappa.
- [Places API](https://console.cloud.google.com/apis/library/places.googleapis.com): Questa è necessaria per cercare luoghi e ottenere le loro coordinate.
- [Geocoding API](https://console.cloud.google.com/apis/library/geocoding-backend.googleapis.com): Questa è necessaria per ottenere le coordinate di un luogo.

Dovrai registrarti per un [account Google Cloud](https://console.cloud.google.com/) e creare un progetto per ottenere le chiavi API seguendo le istruzioni [qui](https://developers.google.com/maps/documentation/embed/get-api-key).

## Limitare l'uso della chiave API

Per motivi di sicurezza, è fondamentale limitare la tua chiave API di Google Maps affinché funzioni solo con domini autorizzati. Questo previene l'uso non autorizzato della tua chiave API se viene esposta. Dovresti limitare la chiave API affinché funzioni solo con:

- app.blue.cc - L'applicazione principale di Blue
- forms.blue.cc - Per moduli pubblici
- Il tuo dominio personalizzato se stai utilizzando [white label](/docs/start-guide/white-label)

Per limitare la tua chiave API:

1. Vai alla [pagina delle credenziali di Google Cloud Console](https://console.cloud.google.com/apis/credentials)
2. Trova la tua chiave API e fai clic su "Modifica" (icona della matita)
3. Sotto "Restrizioni dell'applicazione", seleziona "Riferimenti HTTP (siti web)"
4. Aggiungi i seguenti riferimenti:
   - `*.app.blue.cc/*`
   - `*.forms.blue.cc/*`
   - `*.yourdomain.com/*` (se utilizzi white label)
5. Fai clic su "Salva"
