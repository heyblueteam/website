## Panoramica

Le immagini di copertina ti permettono di aggiungere contesto visivo ai tuoi record mostrando anteprime di immagini direttamente sulle schede dei record. Questa funzionalità estrae automaticamente le immagini da varie fonti all'interno dei tuoi record e le visualizza come miniature, rendendo più semplice identificare e organizzare il contenuto visivo a colpo d'occhio.

Che tu stia gestendo progetti di design, calendari editoriali, cataloghi prodotti o qualsiasi workflow che coinvolga risorse visive, le immagini di copertina ti aiutano a scorrere rapidamente i record e trovare quello che stai cercando. La funzionalità include il monitoraggio dell'avanzamento in tempo reale e opzioni di visualizzazione flessibili per adattarsi alle tue specifiche esigenze di workflow.

## Caratteristiche Principali

- **Estrazione Automatica delle Immagini**: Blue trova e visualizza automaticamente le immagini dalla fonte che hai scelto
- **Fonti di Immagini Multiple**: Estrae immagini dalle descrizioni dei record, dai commenti o da campi file personalizzati
- **Opzioni di Adattamento Visualizzazione**: Scegli come appaiono le immagini con Cover o Contain
- **Selezione Intelligente**: Configura se utilizzare la prima o l'ultima immagine dalla tua fonte
- **Aggiornamenti in Tempo Reale**: Le immagini di copertina si aggiornano automaticamente quando cambia il contenuto del record
- **Monitoraggio Avanzamento in Tempo Reale**: Monitora l'elaborazione in blocco con sottoscrizioni di avanzamento in tempo reale

## Come Funziona

### Abilitazione delle Immagini di Copertina

Le immagini di copertina sono configurate a livello di progetto dagli Amministratori di Progetto:

1. Apri il tuo progetto e naviga in **Impostazioni Progetto**
2. Seleziona **Impostazioni Record**
3. Trova la sezione **Opzioni Copertina Immagine**
4. Attiva **Abilita Immagini di Copertina** per attivare la funzionalità

### Opzioni di Configurazione

#### Fonte Immagine
Scegli dove Blue dovrebbe cercare le immagini:
- **Descrizione**: Estrae immagini incorporate nelle descrizioni dei record (predefinito)
- **Commenti**: Prende la prima/ultima immagine dai commenti del record
- **Campo Personalizzato**: Utilizza file da un campo personalizzato specifico di tipo file (supporta qualsiasi tipo di file)

#### Selezione Immagine
Quando sono disponibili più immagini, scegli quale visualizzare:
- **Prima Immagine**: Utilizza la prima immagine trovata (predefinito)
- **Ultima Immagine**: Utilizza l'immagine più recente

#### Adattamento Visualizzazione
Controlla come appaiono le immagini nell'area di anteprima:
- **Cover**: L'immagine riempie l'intera area di anteprima, può ritagliare i bordi per mantenere le proporzioni (predefinito)
- **Contain**: Mostra l'immagine completa all'interno dell'area di anteprima, può avere spazi vuoti

## Casi d'Uso

**Progetti di Design** - Visualizza mockup e iterazioni di design direttamente sulle schede delle attività, rendendo facile vedere lo stato attuale di ogni attività di design senza aprire singoli record.

**Pianificazione Contenuti** - Per calendari editoriali e workflow editoriali, mostra immagini in evidenza o miniature per visualizzare la tua pipeline di contenuti a colpo d'occhio.

**Gestione Prodotti** - Visualizza foto prodotti sui record di inventario o richieste di funzionalità, aiutando i team a identificare rapidamente gli elementi durante le discussioni.

**Immobiliare** - Mostra foto delle proprietà sui record di annunci, permettendo agli agenti di navigare rapidamente attraverso le proprietà visivamente.

**Pianificazione Eventi** - Visualizza foto di location, concetti di design o layout di eventi sulle attività di pianificazione per una migliore organizzazione visiva.

## Configurazione delle Immagini di Copertina

### Passo 1: Scegli la Tua Fonte di Immagini

Considera dove esistono naturalmente le immagini nel tuo workflow:
- Se le immagini fanno parte delle descrizioni delle attività o dei brief, usa **Descrizione**
- Se le immagini vengono aggiunte attraverso la collaborazione del team, usa **Commenti**
- Se hai un campo dedicato per le immagini, usa **Campo Personalizzato**

### Passo 2: Configura le Impostazioni di Visualizzazione

1. Seleziona il tuo **Adattamento Immagine** preferito basato sui tuoi tipi di immagine:
   - Usa **Cover** per foto e design dove il ritaglio è accettabile
   - Usa **Contain** per diagrammi o screenshot dove devi vedere tutto

2. Scegli **Selezione Immagine** basata sul tuo workflow:
   - Usa **Prima** per workflow dove l'immagine primaria viene aggiunta inizialmente
   - Usa **Ultima** per workflow dove l'immagine più recente è più rilevante

### Passo 3: Elabora i Record Esistenti

Quando abiliti per la prima volta le immagini di copertina o cambi la configurazione, Blue elabora automaticamente i tuoi record:
1. Gli aggiornamenti di avanzamento in tempo reale mostrano lo stato di elaborazione
2. I record vengono elaborati in lotti (50 record per lotto) per prestazioni ottimali
3. Puoi continuare a lavorare mentre l'elaborazione avviene in background
4. Le liste si aggiorneranno automaticamente quando l'elaborazione è completata
5. L'elaborazione si attiva automaticamente quando le modifiche alla configurazione vengono salvate

## Dettagli Tecnici

### Formati Supportati
- Quando si utilizzano fonti Descrizione o Commenti:
  - Immagini JPEG/JPG
  - Immagini PNG (inclusa trasparenza)
  - Immagini GIF (visualizzazione statica)
- Quando si utilizza la fonte Campo Personalizzato:
  - Qualsiasi tipo di file può essere visualizzato (immagini, PDF, documenti)
  - I file non-immagine mostrano icone appropriate del tipo di file
- Tutti i file devono essere caricati su Blue (URL esterni non supportati)

### Limitazioni
- Massimo 6.000 record per progetto per la generazione in blocco delle copertine
- Solo i campi personalizzati di tipo file possono essere utilizzati come fonti
- Le immagini di copertina sono di sola visualizzazione e non possono essere modificate direttamente

### Aggiornamenti Automatici
Le copertine si rigenerano automaticamente quando:
- La descrizione del record viene modificata (se si utilizza la fonte Descrizione)
- Vengono aggiunti nuovi commenti con immagini (se si utilizza la fonte Commenti)
- I file nei campi personalizzati vengono aggiornati (se si utilizza la fonte Campo Personalizzato)
- Le impostazioni di configurazione della copertina vengono modificate
- L'opzione di adattamento visualizzazione viene modificata

### Accesso API
Le immagini di copertina sono completamente integrate con l'API GraphQL:
- Configura tramite la mutazione `setTodoCoverConfig`
- Accedi ai dati della copertina attraverso il campo `cover` sul tipo Todo
- Monitora l'avanzamento con la sottoscrizione `subscribeToCoverGenerationProgress`
- URL delle miniature generate automaticamente a `/uploads/{uid}/thumbnail/cover`
