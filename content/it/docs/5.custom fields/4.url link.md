Il campo personalizzato URL/Link vi consente di memorizzare collegamenti e indirizzi web all'interno dei vostri record in Blue. Quando viene inserito un URL in questo campo, il sistema verificherà che sia un collegamento strutturato correttamente.

![URL-Link 1](/docs/URLlink_1.png "URL-Link 1")

Il campo personalizzato URL/Link in Blue

Ogni record può contenere un collegamento, memorizzato in un singolo campo personalizzato URL/Link. Il collegamento è cliccabile, consentendovi di navigare direttamente alla pagina web o risorsa facendo clic su di esso. Per comodità, si aprirà automaticamente in una nuova scheda del browser.

![URL-Link 2](/docs/URLlink_2.png "URL-Link 2")

Potete aprire gli URL in una nuova scheda con un clic

Questo campo è estremamente utile per collegarsi a documenti basati su cloud come Google Docs, file Word, fogli di calcolo Excel, ecc. Memorizzare questi collegamenti consente l'accesso con un clic ai file rilevanti associati a ogni record.

Il campo URL/Link è ottimo per casi d'uso aggiuntivi come:

- Memorizzare siti web di clienti o partner
- Catturare collegamenti associati ai record
- Creare riferimenti a pagine web

Con la sua interfaccia semplificata e le capacità di collegamento integrate, il campo personalizzato URL/Link vi consente di arricchire i record con risorse web contestuali e accedervi istantaneamente con un clic.
