L'API di Blue fornisce accesso completo alla piattaforma di gestione dei processi di Blue attraverso un'interfaccia moderna [GraphQL](https://graphql.org/). Fidato da **oltre 17.000 clienti** che elaborano **milioni di richieste API annuali**, la nostra API alimenta flussi di lavoro mission-critical con **[99,99% di uptime](/platform/status)**.

## Perché l'API di Blue?

- **Copertura Completa delle Funzionalità** - Tutto ciò che puoi fare nell'interfaccia di Blue, puoi farlo tramite API
- **Pronto per le Aziende** - Testato in battaglia su larga scala con limitazione della velocità e sicurezza complete
- **Friendly per gli Sviluppatori** - GraphQL fornisce esattamente i dati di cui hai bisogno, nient'altro
- **Aggiornamenti in Tempo Reale** - Sottoscrizioni WebSocket per la sincronizzazione istantanea dei dati

## Inizio Veloce

Pronto a fare la tua prima chiamata API? Ecco quanto è semplice:

```graphql
query MyProjects {
  projectList(filter: { companyIds: ["your-company-id"] }) {
    items {
      id
      name
      updatedAt
    }
  }
}
```

**[Provalo nel nostro GraphQL Playground →](https://api.blue.cc/graphql)**

## Casi d'Uso Popolari

### Gestione dei Dati
- Usa Blue come un database flessibile per applicazioni personalizzate
- Sincronizza i dati tra Blue e i tuoi sistemi esistenti
- Crea dashboard personalizzate e strumenti di reporting

### Automazione dei Flussi di Lavoro
- Crea record da trigger esterni (moduli, webhook, email)
- Aggiorna più progetti in base alla logica aziendale
- Automatizza l'assegnazione di compiti e notifiche

### Sviluppo di Integrazioni
- Collega Blue con CRM, ERP e strumenti specializzati esterni
- Crea integrazioni personalizzate per il tuo settore
- Estendi le capacità di Blue con le tue funzionalità
