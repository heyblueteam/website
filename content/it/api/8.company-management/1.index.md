---
title: Crea un'Azienda
description: Come creare una nuova azienda utilizzando l'API di Blue.
---

Per creare una nuova azienda, puoi utilizzare la seguente mutazione:

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

Nota che la creazione di nuove aziende può influenzare la fatturazione del tuo account Blue, poiché ogni azienda viene fatturata separatamente.

Ecco le opzioni per la mutazione `createCompany` e le loro descrizioni:

| Opzione | Tipo | Descrizione |
|--------|------|-------------|
| name | String | Il nome della nuova azienda. Questo è un campo obbligatorio. |  |
| slug | Stringa | Una versione del nome dell'azienda adatta agli URL. Questo viene utilizzato nell'URL dell'azienda e dovrebbe essere unico. Questo è un campo obbligatorio e fungerà anche da identificatore unico per l'azienda. |
