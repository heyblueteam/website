---
title: 会社を作成する
description: Blue APIを使用して新しい会社を作成する方法。
---

新しい会社を作成するには、次のミューテーションを使用できます：

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

新しい会社を作成すると、各会社が別々に請求されるため、Blueアカウントの請求に影響を与える可能性があることに注意してください。

以下は、`createCompany`ミューテーションのオプションとその説明です：

| オプション | タイプ | 説明 |
|--------|------|-------------|
| name | String | 新しい会社の名前。これは必須フィールドです。 |  |
| スラッグ | 文字列 | 会社名のURLフレンドリーなバージョン。これは会社のURLに使用され、ユニークである必要があります。これは必須フィールドであり、会社のユニーク識別子としても機能します。 |
