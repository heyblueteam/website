La API de Blue proporciona acceso completo a la plataforma de gestión de procesos de Blue a través de una moderna interfaz [GraphQL](https://graphql.org/). Confiada por **más de 17,000 clientes** que procesan **miles de millones de solicitudes API anualmente**, nuestra API impulsa flujos de trabajo críticos con **[99.99% de tiempo de actividad](/platform/status)**.

## ¿Por qué la API de Blue?

- **Cobertura de Funciones al 100%** - Todo lo que puedes hacer en la interfaz de usuario de Blue, lo puedes hacer a través de la API
- **Listo para Empresas** - Probado a gran escala con limitación de tasa y seguridad completas
- **Amigable para Desarrolladores** - GraphQL proporciona exactamente los datos que necesitas, nada más
- **Actualizaciones en Tiempo Real** - Suscripciones WebSocket para sincronización instantánea de datos

## Inicio Rápido

¿Listo para hacer tu primera llamada a la API? Así de simple es:

```graphql
query MyProjects {
  projectList(filter: { companyIds: ["your-company-id"] }) {
    items {
      id
      name
      updatedAt
    }
  }
}
```

**[Pruébalo en nuestro Playground de GraphQL →](https://api.blue.cc/graphql)**

## Casos de Uso Populares

### Gestión de Datos
- Usa Blue como una base de datos flexible para aplicaciones personalizadas
- Sincroniza datos entre Blue y tus sistemas existentes
- Crea paneles de control y herramientas de informes personalizadas

### Automatización de Flujos de Trabajo
- Crea registros a partir de desencadenantes externos (formularios, webhooks, correos electrónicos)
- Actualiza múltiples proyectos basados en la lógica empresarial
- Automatiza la asignación de tareas y notificaciones

### Desarrollo de Integraciones
- Conecta Blue con CRMs, ERPs y herramientas especializadas externas
- Crea integraciones personalizadas para tu industria
- Amplía las capacidades de Blue con tus propias funciones
