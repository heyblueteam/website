---
title: Crear una Empresa
description: Cómo crear una nueva empresa utilizando la API de Blue.
---

Para crear una nueva empresa, puedes usar la siguiente mutación:

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

Ten en cuenta que crear nuevas empresas puede afectar la facturación de tu cuenta de Blue, ya que cada empresa se factura por separado.

Aquí están las opciones para la mutación `createCompany` y sus descripciones:

| Opción | Tipo | Descripción |
|--------|------|-------------|
| name | String | El nombre de la nueva empresa. Este es un campo obligatorio. |  |
| slug | Cadena | Una versión amigable para URL del nombre de la empresa. Esto se utiliza en la URL de la empresa y debe ser único. Este es un campo obligatorio y también actuará como el identificador único para la empresa. |
