Blue tiene una API (Interfaz de Programación de Aplicaciones), para que puedas construir tus propias integraciones si es necesario.

**Blue tiene una cobertura del 100% de API**. Esto significa que cualquier cosa que puedas hacer en [cualquiera de nuestras aplicaciones](/docs/start-guide/download-apps), puedes hacerlo programáticamente. Tenemos una poderosa API [GraphQL](https://graphql.org/) que tiene un playground completo donde puedes probar tus consultas y está completamente documentada.
