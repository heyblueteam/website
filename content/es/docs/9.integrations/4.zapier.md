Zapier es una plataforma de integración que ayuda a conectar diferentes aplicaciones web y automatizar flujos de trabajo.

**Casos de Uso**

Aquí hay algunas de las formas más populares en que Blue está integrado con otras aplicaciones a través de Zapier:

**_Sincronizar Registros de Blue a Google Sheets_**

A medida que se crean o actualizan registros de Blue, añádelos automáticamente como nuevas filas en una hoja de Google para informes centralizados.

**_Enviar Confirmación de Pedido desde registros de Blue_**

Cuando seleccionas ciertos campos personalizados en Blue, envía automáticamente un correo electrónico para confirmar el pedido con tu cliente.

**_Agregar Comentarios de Blue a Slack_**

Recibe notificaciones en tiempo real de Blue publicadas en los canales de Slack adecuados para que tu equipo esté al tanto de actualizaciones importantes de registros.

## Beneficios Clave

**Automatización entre Herramientas**: Los Zaps creados en Zapier pueden enviar y recibir datos automáticamente entre Blue y miles de otras aplicaciones. Esto elimina el trabajo manual de mover datos entre sistemas.

**Integraciones en Tiempo Real**: Usando los disparadores de webhook de Blue, los Zaps reaccionan instantáneamente a eventos en Blue como un nuevo registro, un comentario añadido, cambio de estado, etc. Esto permite la sincronización en tiempo real y la automatización de procesos.

**Fácil de Construir y Mantener**: Los Zaps proporcionan una forma fácil y sin código para integrar herramientas. No se necesita trabajo de desarrollo. Y siguen funcionando sin mantenimiento continuo.

1. Inicia sesión en tu cuenta de Blue en la configuración del perfil
2. Ve a la configuración del perfil y haz clic en "Webhooks"
3. Haz clic en "Agregar Webhook"
4. Dale un nombre al webhook e ingresa la URL del webhook de Zapier
5. Elige qué eventos deberían activar el webhook
6. Haz clic en "Crear Webhook"

## Blue a Zapier

1. Inicia sesión en tu cuenta de Zapier
2. Haz clic en Hacer un Zap  
3. Para el primer paso, busca y selecciona la [aplicación desencadenadora "Webhooks by Zapier"](https://zapier.com/apps/webhook)
4. Configúralo para aceptar cargas útiles de tu URL de webhook de Blue
5. Para el segundo paso, busca y selecciona la aplicación de destino para enviar datos
6. Mapea los campos de carga útil JSON a los campos de entrada de la aplicación de destino
7. Activa el Zap

Ahora, cuando ocurran los eventos especificados en Blue, la carga útil del webhook se enviará a Zapier, que automáticamente pasará los datos a tu aplicación integrada según la configuración del Zap.

## Zapier a Blue

Para enviar datos de otras aplicaciones a Blue, sigue estos pasos:

**1. Obtener Credenciales de API de Blue**
- Ve a Configuración de Blue > Tokens de API
- Crea un nuevo token o usa credenciales existentes:
  - ID del Token
  - Secreto del Token
  - ID de la Empresa
  - ID del Proyecto

**2. Crear Zap en Zapier**
1. Configura tu aplicación desencadenadora (por ejemplo, Google Sheets, Slack, etc.)
2. Agrega "Webhooks by Zapier" como la aplicación de Acción
3. Elige el tipo de evento "Solicitud Personalizada"

**3. Configurar Webhook**
- Método: POST
- URL: `https://api.blue.cc/graphql` (o endpoint beta si es necesario)
- Pasar Datos: FALSO
- Desenrollar: NO

**4. Configurar Encabezados:**
```json
{
  "Content-Type": "application/json",
  "X-Bloo-Token-ID": "Your_Token_ID",
  "X-Bloo-Token-Secret": "Your_Token_Secret", 
  "X-Bloo-Company-ID": "Your_Company_ID",
  "X-Bloo-Project-ID": "Your_Project_ID"
}
```

**5. Configurar Consulta (Ejemplo de Mutación):**
```json
{
  "query": "mutation CreateTodo($input: CreateTodoInput!) {\n  createTodo(input: $input) {\n    id\n    title\n    customFields {\n      value\n    }\n  }\n}",
  "variables": {
    "input": {
      "todoListId": "your-todo-list-id",
      "title": "New task from Zapier",
      "customFields": [
        {
          "customFieldId": "priority-field-id",
          "value": "High"
        },
        {
          "customFieldId": "due-date-field-id",
          "value": "2024-03-01"
        }
      ]
    }
  }
}
```

**6. Probar y Activar**
- Usa el API Playground de Blue para probar las mutaciones primero
- Mapea valores dinámicos de las aplicaciones desencadenadoras usando los campos de datos de Zapier
- Activa el Zap después de una prueba exitosa
