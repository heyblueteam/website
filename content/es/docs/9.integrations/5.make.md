[Make.com](https://www.make.com) es una plataforma de integración que ayuda a conectar diferentes aplicaciones web y automatizar flujos de trabajo. Blue está **oficialmente integrado** con Make.com, lo que permite una automatización fluida a través de más de 2,000 aplicaciones, ¡sin necesidad de codificación!

![Resumen de Módulos de Blue Make.com](/docs/blue_make_modules_overview.png "Overview of available Blue modules in Make.com")

## Casos de Uso

Aquí hay algunas formas poderosas de extender las capacidades de Blue con integraciones de Make.com:
La integración de Blue con Make.com permite una automatización poderosa en varias funciones comerciales. Para la experiencia del cliente, puedes conectar Blue con herramientas como Intercom, HubSpot y Zendesk para crear registros completos de clientes, sincronizar interacciones y mantener comunicación en tiempo real a través de Slack y Microsoft Teams. En la gestión de proyectos, Blue se integra con Jira, GitHub, Trello y Monday.com para sincronizar tareas, rastrear el progreso y automatizar notificaciones.

Para marketing y ventas, Blue se conecta con Mailchimp, Stripe y Calendly para rastrear campañas, gestionar pagos y automatizar flujos de trabajo de reuniones. La gestión de documentos se simplifica a través de integraciones con Google Drive, Dropbox, OneDrive y SharePoint, mientras que herramientas de comunicación como Slack, Gmail y WhatsApp se unifican a través del sistema central de gestión de tareas de Blue. Las capacidades de análisis de datos se mejoran a través de conexiones con Google Sheets, Power BI, Airtable y Tableau, y hay soluciones específicas de la industria disponibles para servicios financieros, comercio electrónico y gestión de equipos remotos.

## Conectar y Autenticar tu Cuenta de Blue

Para conectar de manera segura Blue con Make.com:

1. Inicia sesión en tu cuenta de Blue y abre tu perfil → ve a la pestaña **API**
2. Haz clic en **Generar un Token**, proporciona un nombre (y una fecha de caducidad opcional)
3. Copia ambos:
   - **ID de Token** (`x‑bloo‑token‑id`)
   - **Clave Secreta** (`x‑bloo‑token‑secret`)

::callout
 ---
icon: lightbulb
---
**Nota Importante** <br/>
La clave secreta se muestra solo una vez—¡almacénala de forma segura!
::

5. En Make.com, al crear una nueva conexión, ingresa:
   - **ID de Token**
   - **Clave Secreta**

## Referencia de Parámetros del Módulo

### Observar Eventos
- Eventos (opcional): Tipos de eventos de selección múltiple
- ID de Empresa (opcional): IDs o slugs de empresas de selección múltiple
- ID de Proyecto (opcional): IDs o slugs de proyectos de selección múltiple

### Crear un Registro
- ID de Empresa: ID o slug de la empresa
- ID de Proyecto: ID o slug del proyecto
- Lista: ID de la lista
- Nombre del Registro: Nombre del registro
- Campo Predeterminado: Valor predeterminado
- Fecha Única: Fecha de vencimiento
- Rango de Fechas: Fechas de inicio y vencimiento
- Asignados: Usuarios de selección múltiple
- Etiquetas: Etiquetas de selección múltiple
- Descripción: Campo de texto, soporta HTML
- Listas de Verificación: Array de títulos de listas de verificación

**Campos Personalizados Soportados:**
| Tipo de Campo | Descripción |
|---------------|-------------|
| Texto de Línea Única | Campo de texto de línea única con validación opcional y límite de caracteres |
| Texto de Múltiples Líneas | Campo de texto de múltiples líneas con soporte para texto enriquecido y opciones de formato |
| Selección Única | Desplegable con selección de una sola opción y opciones personalizadas |
| Selección Múltiple | Desplegable con selección de múltiples opciones y opciones personalizadas |
| Ubicación | Latitud y longitud con vista de mapa y búsqueda de direcciones |
| Casilla de Verificación | Alternar verdadero/falso con valor predeterminado opcional y etiquetas personalizadas |
| País | Selector de código de país ISO de dos letras con nombres de países |
| Moneda | Campo numérico con símbolo de moneda y soporte para tasas de cambio (por ejemplo, USD200, 200USD, KHR4000) |
| Fecha | Selector de fecha con componente de tiempo opcional y soporte para zona horaria |
| Teléfono | Campo de número de teléfono que soporta E.164, formato internacional o código de país + número con validación |
| Correo Electrónico | Campo de correo electrónico con validación de formato y verificación de dominio |
| Calificación por Estrellas | Calificación numérica de 1-5 con estrellas visuales y etiquetas personalizadas |
| Porcentaje | Campo numérico con símbolo de porcentaje (0-100) y precisión decimal |
| URL | Campo de dirección web con validación de formato y vista previa |
| Número | Campo numérico con lugares decimales opcionales y visualización de unidades |
| Fórmula | Campo calculado basado en otros campos con operaciones matemáticas |
| ID Único | Identificador autogenerado con prefijo y formato personalizados |
| Referencia | Enlace a otro registro con opciones de búsqueda y filtrado |
| Búsqueda | Campo dinámico que extrae datos de registros referenciados con opciones de mapeo |
| Duración de Tiempo | Duración en horas y minutos con formato de visualización personalizado |

### Ejecutar GraphQL
- Encabezados: Array de pares clave-valor para autenticación y encabezados personalizados
- Método: GET o POST con opciones de solicitud
- Consulta: Consulta o mutación GraphQL con soporte para variables
- Nombre de la Operación: Nombre de la operación para seguimiento
- Fuente de Datos de Variables: Raw, Form o Collection con verificación de tipo
- Variables: Contenido de variables GraphQL con validación
