Blue te permite usar Google Maps para la vista de mapa en lugar del OpenStreetMap predeterminado. Google Maps a menudo tiene mapas más detallados de ciertas áreas y datos de ubicación más precisos. Sin embargo, esto requiere una clave de API de Google Maps que debe ser ingresada en la configuración de la empresa por el propietario de la empresa.

Cuando cambias a Google Maps, esto afecta todas las vistas de mapa en Blue, incluyendo el mapa de la empresa, la [vista de mapa en proyectos](/docs/views/map) y el [campo personalizado de ubicación](/docs/custom-fields/location-map).

Para usar Google Maps en Blue, se requieren las siguientes APIs activadas en Google Cloud Console:
- [Maps Tiles API](https://console.cloud.google.com/apis/library/tile.googleapis.com): Esto es necesario para renderizar los mosaicos del mapa.
- [Places API](https://console.cloud.google.com/apis/library/places.googleapis.com): Esto es necesario para buscar lugares y obtener sus coordenadas.
- [Geocoding API](https://console.cloud.google.com/apis/library/geocoding-backend.googleapis.com): Esto es necesario para obtener las coordenadas de un lugar.

Necesitarás registrarte para obtener una [cuenta de Google Cloud](https://console.cloud.google.com/) y crear un proyecto para obtener las claves de API siguiendo las instrucciones [aquí](https://developers.google.com/maps/documentation/embed/get-api-key).

## Restringiendo el Uso de la Clave de API

Por razones de seguridad, es crucial restringir tu clave de API de Google Maps para que solo funcione con dominios autorizados. Esto previene el uso no autorizado de tu clave de API si se expone. Debes restringir la clave de API para que solo funcione con:

- app.blue.cc - La aplicación principal de Blue
- forms.blue.cc - Para formularios públicos
- Tu dominio personalizado si estás usando [marca blanca](/docs/start-guide/white-label)

Para restringir tu clave de API:

1. Ve a la [página de Credenciales de Google Cloud Console](https://console.cloud.google.com/apis/credentials)
2. Encuentra tu clave de API y haz clic en "Editar" (ícono de lápiz)
3. En "Restricciones de aplicación", selecciona "Referentes HTTP (sitios web)"
4. Agrega los siguientes referentes:
   - `*.app.blue.cc/*`
   - `*.forms.blue.cc/*`
   - `*.yourdomain.com/*` (si usas marca blanca)
5. Haz clic en "Guardar"
