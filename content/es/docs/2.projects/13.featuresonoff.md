---
title: Activar/Desactivar Funciones
description: Las funciones en Blue se pueden activar o desactivar para un proyecto.
---

Los administradores del proyecto pueden activar o desactivar funciones para un proyecto para todos los usuarios. Esto es útil si deseas limitar las funciones disponibles para un proyecto. Por ejemplo, no todos los proyectos pueden necesitar tener habilitada la Wiki. Los administradores también pueden reorganizar el orden de las pestañas en el proyecto.
