El campo de Texto de Una Línea le permite agregar cadenas de texto sin formato a sus registros en Blue. No existe límite de caracteres, por lo que puede ingresar texto de cualquier longitud en este campo. Sin embargo, el texto está restringido a una sola línea y no admite opciones de formato como negrita, cursiva o enlaces.

![Single Line Text](/docs/SingleLineText.png "Single Line Text")

Una forma flexible de capturar datos de texto estructurados

El campo de texto es muy flexible, permitiendo el ingreso tanto de letras como de números, incluyendo caracteres especiales. No hay restricciones sobre qué caracteres se pueden ingresar. El sistema no aplica ninguna validación o enmascaramiento al texto ingresado por el usuario.

El campo de Texto de Una Línea es excelente para capturar todo tipo de información, incluyendo nombres, descripciones, números de modelo, SKUs, direcciones, notas, IDs, y más. Puede usar este campo flexible para almacenar nombres de clientes, proveedores o contactos, indicar códigos de inventario e identificadores, hacer notas generales cortas sobre registros, o almacenar IDs únicos para relaciones y búsquedas de bases de datos. El campo de texto de una línea le permite ingresar eficientemente cualquier información importante basada en texto relacionada con sus registros en Blue.

Puede aprovechar las [capacidades avanzadas de filtrado de registros](/es/docs/views/filters) de Blue para encontrar y filtrar registros que contengan texto específico dentro del campo de Texto de Una Línea para localizar registros coincidentes.

Con su diseño simple de una sola línea, capacidad de texto sin restricciones, e integración con filtrado, el campo de Texto de Una Línea ofrece una experiencia optimizada para capturar datos no estructurados. Proporciona la flexibilidad para almacenar cualquier información según sea necesario por registro.
