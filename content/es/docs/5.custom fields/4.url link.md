El campo personalizado URL/Enlace le permite almacenar enlaces y direcciones web dentro de sus registros en Blue. Cuando se ingresa una URL en este campo, el sistema validará que sea un enlace con estructura adecuada.

![URL-Link 1](/docs/URLlink_1.png "URL-Link 1")

El campo personalizado URL/Enlace en Blue

Cada registro puede contener un enlace, almacenado en un solo campo personalizado URL/Enlace. El enlace es clickeable, permitiéndole navegar directamente a la página web o recurso haciendo clic en él. Por conveniencia, se abrirá automáticamente en una nueva pestaña del navegador.

![URL-Link 2](/docs/URLlink_2.png "URL-Link 2")

Puede abrir URLs en una nueva pestaña con un clic

Este campo es extremadamente útil para enlazar a documentos basados en la nube como Google Docs, archivos de Word, hojas de cálculo de Excel, etc. Almacenar estos enlaces permite el acceso con un clic a archivos relevantes asociados con cada registro.

El campo URL/Enlace es excelente para casos de uso adicionales como:

- Almacenar sitios web de clientes o socios
- Capturar enlaces asociados con registros
- Crear referencias a páginas web

Con su interfaz optimizada y capacidades de enlace incorporadas, el campo personalizado URL/Enlace le permite enriquecer registros con recursos web contextuales y acceder a ellos instantáneamente con un clic.
