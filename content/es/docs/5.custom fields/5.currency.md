Almacene información financiera en el campo de moneda

Al agregar valores de moneda, el campo formateará el número según las convenciones locales y mostrará el símbolo de moneda relevante. Por ejemplo, $1,000.50 para dólares estadounidenses o €1.500,25 para euros.

El campo de moneda valida las entradas para asegurar que solo se capturen cantidades de moneda válidas. Una vez agregado a un proyecto, el campo almacenará las cantidades en el tipo de moneda designado singular. No proporciona conversión automática de moneda entre múltiples denominaciones.

Este campo ofrece una experiencia optimizada para rastrear datos monetarios asociados con registros. El formateo automatizado elimina la complejidad de ingresar manualmente valores de moneda en formatos locales, mientras que la validación protege la integridad de los datos.

Los usos potenciales para el campo de moneda incluyen:

- Capturar montos de transacciones
- Almacenar presupuestos de proyectos
- Rastrear totales de facturas
- Registrar montos de pagos
- Calcular ingresos

Con un sólido soporte para formatos de moneda globales y entrada de datos simplificada, el campo personalizado de Moneda le permite centralizar información financiera clave dentro de sus registros en Blue.
