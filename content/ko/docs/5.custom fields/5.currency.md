통화 필드에 금융 정보를 저장하세요

통화 값을 추가할 때, 필드는 현지 관례에 따라 숫자를 형식화하고 관련 통화 기호를 표시합니다. 예를 들어, 미국 달러의 경우 $1,000.50, 유로의 경우 €1.500,25와 같이 표시됩니다.

통화 필드는 유효한 통화 금액만 입력되도록 항목을 검증합니다. 프로젝트에 추가되면, 필드는 지정된 단일 통화 유형으로 금액을 저장합니다. 여러 통화 간의 자동 환율 변환 기능은 제공하지 않습니다.

이 필드는 레코드와 관련된 금전적 데이터를 추적하는 데 최적화된 경험을 제공합니다. 자동 형식화 기능은 현지 형식으로 통화 값을 수동 입력하는 복잡성을 제거하며, 검증 기능은 데이터 무결성을 보호합니다.

통화 필드의 잠재적 용도는 다음과 같습니다:

- 거래 금액 기록
- 프로젝트 예산 저장
- 송장 총액 추적
- 결제 금액 기록
- 수익 계산

강력한 글로벌 통화 형식 지원과 간소화된 데이터 입력 기능을 통해, 통화 사용자 정의 필드는 Blue의 레코드 내에서 핵심 금융 정보를 중앙화할 수 있게 해줍니다.

::callout
