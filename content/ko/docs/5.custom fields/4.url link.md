URL/링크 사용자 정의 필드를 사용하면 Blue의 레코드 내에서 링크와 웹 주소를 저장할 수 있습니다. 이 필드에 URL을 입력하면 시스템에서 올바르게 구조화된 링크인지 검증합니다.

![URL-Link 1](/docs/URLlink_1.png "URL-Link 1")

Blue의 URL/링크 사용자 정의 필드

각 레코드는 하나의 URL/링크 사용자 정의 필드에 저장된 하나의 링크를 포함할 수 있습니다. 링크는 클릭 가능하여 클릭하면 웹 페이지나 리소스로 직접 이동할 수 있습니다. 편의를 위해 자동으로 새 브라우저 탭에서 열집니다.

![URL-Link 2](/docs/URLlink_2.png "URL-Link 2")

한 번의 클릭으로 새 탭에서 URL을 열 수 있습니다

이 필드는 Google 문서, Word 파일, Excel 스프레드시트 등의 클라우드 기반 문서에 연결하는 데 매우 유용합니다. 이러한 링크를 저장하면 각 레코드와 관련된 관련 파일에 한 번의 클릭으로 액세스할 수 있습니다.

URL/링크 필드는 다음과 같은 추가 사용 사례에 유용합니다:

- 고객 또는 파트너 웹사이트 저장
- 레코드와 관련된 링크 캡처
- 웹페이지에 대한 참조 생성

간소화된 인터페이스와 내장된 링크 기능을 통해 URL/링크 사용자 정의 필드를 사용하면 상황에 맞는 웹 리소스로 레코드를 풍부하게 만들고 클릭 한 번으로 즉시 액세스할 수 있습니다.
