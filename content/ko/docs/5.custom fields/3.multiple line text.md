여러 줄 텍스트 필드는 사용자가 Blue의 레코드에 더 길고 여러 단락으로 구성된 텍스트 콘텐츠를 입력할 수 있도록 합니다. 다음과 같은 기능을 제공합니다:

- 문자 제한 없음, 무제한 텍스트 길이
- 굵게, 기울임꼴, 하이퍼링크와 같은 서식 지원
- 줄 바꿈, 간격, 텍스트 정렬

![여러 줄 텍스트](/docs/MultipleLineText.png "여러 줄 텍스트")

**사용 사례**

이 필드는 다음과 같은 상황에 이상적입니다:

- 설명 - 제품 사양, 작업 세부사항, 프로젝트 범위
- 메모 및 지침
- 메시지 - 업데이트, 설명
- 콘텐츠 - 기사, 블로그, 가이드
- 요약 및 보고서
- 이메일 콘텐츠
- 피드백 및 리뷰

기본적으로 모든 유형의 긴 텍스트 콘텐츠를 입력할 수 있습니다.
