Blue의 파이 차트는 분류된 기록에 걸쳐 비율 데이터를 시각화하기 위한 직관적인 원형 그래프를 제공합니다. 이러한 그래프는 색상으로 구분된 조각을 사용하여 데이터 세트 내의 상대적 크기와 관계를 설명합니다.

## 파이 차트 만들기

### 제목 설정

파이 차트를 만들 때 가장 먼저 해야 할 일은 차트의 제목을 설정한 다음 조각 값에 대한 데이터를 가져오는 것입니다. 개별 조각은 여러 개의 하위 그룹을 포함할 수 있어 하나의 차트에서 다층 분석을 가능하게 합니다. 예를 들어, 각 제품 조각 내에서 지역 및 채널 구분을 쌓을 수 있습니다.

![파이 차트 1](/docs/dashboard/pieChart/PieChart_1.png "Pie Chart 1")

### 값 또는 여러 값 만들기

차트당 여러 개의 조각을 생성하여 원을 구성할 수 있습니다. 각 조각은 의미가 있도록 다른 프로젝트의 사용자 정의 필드에서 데이터를 가져와야 합니다. [사용자 정의 필드 "통화"](/docs/custom-fields/currency)에서 데이터를 선택하는 경우, 나머지 조각은 동일한 데이터에서 가져와야 합니다. 이를 통해 하나의 파이 차트에서 프로젝트 간 동일한 데이터를 비교할 수 있습니다.

파이 차트의 값은 특정 프로젝트에서 파생될 수 있으며, 이를 통해 다음과 같은 작업을 수행할 수 있습니다:

- 지역, 채널, 시장 등에서의 판매 구성 분석
- 카테고리, 우선순위, 담당자 등에 따른 작업 또는 문제 분포 분석
- 워크플로우 단계 진행 및 병목 현상 이해
- 제품, 캠페인, 부문 등에서의 수익 또는 이익 비율 벤치마킹
- 팀 및 수준 간의 다양성 및 포용성 지표 추적

![파이 차트 2](/docs/dashboard/pieChart/PieChart_2.png "Pie Chart 2")

파이 차트의 값이 가능한 한 관련성 있고 목표 지향적이도록 보장하기 위해, Blue는 강력한 필터링 옵션을 제공합니다 — 모든 프로젝트 보기에서 [모든 필터](/docs/views/filters)를 사용할 수 있습니다!

### 논리 추가

Blue에서 통계 카드 내의 논리 기능은 데이터 기반 통찰력을 계산하고 제시하는 엔진입니다. 논리는 사용자 정의 수식과 작업을 통해 데이터의 힘을 활용할 수 있게 하여 원시 프로젝트 데이터를 실행 가능한 정보로 변환합니다.

'{ '를 입력하여 프로젝트 내의 특정 값을 참조하는 것으로 시작합니다. 이 작업은 논리 수식에 포함할 수 있는 사용 가능한 값 목록을 표시합니다.

![파이 차트 3](/docs/dashboard/pieChart/PieChart_3.png "Pie Chart 3")

'='를 입력하여 다양한 기능으로 논리를 향상시킬 수 있습니다. Blue는 다음과 같은 여러 기능을 제공합니다:

- **AVERAGE**: 숫자 집합의 평균을 계산합니다.
- **AVERAGEA**: 텍스트를 0으로 간주하여 평균을 결정합니다.
- **COUNT**: 숫자 필드의 항목 수를 계산합니다.
- **COUNTA**: 비어 있지 않은 값의 수를 계산합니다.
- **MAX**: 집합에서 최대 값을 찾습니다.
- **MIN**: 집합에서 최소 값을 식별합니다.
- **MINUS**: 한 값을 다른 값에서 뺍니다.
- **MULTIPLY**: 두 값을 곱합니다.
- **SUM**: 일련의 숫자를 더합니다.

![파이 차트 4](/docs/dashboard/pieChart/PieChart_4.png "Pie Chart 4")

이러한 기능은 데이터 분석의 필수 도구로, 프로젝트 메트릭을 효과적으로 요약하고 분석할 수 있게 해줍니다. 최대 판매를 파악하거나 평균 작업 기간을 계산하거나 예산 할당을 합산하려는 경우, Blue의 논리는 이를 간단하게 만들어 줍니다.

![파이 차트 5](/docs/dashboard/pieChart/PieChart_5.png "Pie Chart 5")

파이 차트는 프로젝트 데이터를 비율 기여, 진행 상황 및 Blue의 분류된 기록 간의 차이에 대한 실행 가능하고 시각적으로 직관적인 통찰력으로 변환합니다.
