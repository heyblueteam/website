### 제목 설정

통계 카드를 만들 때 가장 먼저 해야 할 일은 제목을 설정하는 것입니다. 통계 카드에 제목을 붙이는 것은 명확성과 사용성을 위한 필수 단계입니다. 제목은 통계 카드가 나타내는 중요한 메트릭이나 통찰력을 요약해야 하며, 예를 들어 "월별 판매", "열린 이슈" 또는 "주간 활성 사용자"와 같은 형식이어야 합니다.

## 값 생성

통계 카드당 여러 값을 생성할 수 있으며, 각 값은 프로젝트에 중요한 차원이나 메트릭을 나타냅니다. 이 다중 값 기능은 프로젝트 성과를 실시간으로 반영하는 포괄적인 대시보드를 가능하게 합니다.

통계 카드의 값은 특정 프로젝트에서 파생될 수 있으며, 이를 통해 다음과 같은 작업을 수행할 수 있습니다:
- **레코드 수**: 프로젝트 내의 총 레코드 수를 계산하여 작업이나 항목의 양을 빠르게 파악할 수 있습니다.
- **사용자 정의 필드 계산**: 사용자 정의 필드에 포함된 데이터를 기반으로 계산을 수행합니다. 예를 들어, "프로젝트 가격" 필드의 값을 합산하여 전체 예산 수치를 평가할 수 있습니다.

통계 카드의 값이 최대한 관련성 있고 목표 지향적이도록 하기 위해, Blue는 강력한 필터링 옵션을 제공합니다 — 모든 프로젝트 보기에서 [모든 필터](/docs/views/filters)를 사용할 수 있습니다!

## 논리 추가
Blue에서 통계 카드 내의 논리 기능은 데이터 기반 통찰력을 계산하고 제시하는 엔진입니다. 논리는 맞춤형 수식과 연산을 통해 데이터의 힘을 활용하여 원시 프로젝트 데이터를 실행 가능한 정보로 변환할 수 있게 합니다.

'{ '를 입력하여 프로젝트 내의 특정 값을 참조하는 것으로 시작합니다. 이 작업은 논리 수식에 포함할 수 있는 사용 가능한 값 목록을 표시합니다.

'='를 입력하여 다양한 함수로 논리를 향상시킬 수 있습니다. Blue는 다음과 같은 함수 모음을 제공합니다:

- **AVERAGE**: 숫자 집합의 평균을 계산합니다.
- **AVERAGEA**: 텍스트를 0으로 간주하여 평균을 결정합니다.
- **COUNT**: 숫자 필드의 항목 수를 계산합니다.
- **COUNTA**: 비어 있지 않은 값의 수를 계산합니다.
- **MAX**: 집합에서 최대 값을 찾습니다.
- **MIN**: 집합에서 최소 값을 식별합니다.
- **MINUS**: 한 값을 다른 값에서 뺍니다.
- **MULTIPLY**: 두 값을 곱합니다.
- **SUM**: 일련의 숫자를 합산합니다.

이 함수들은 데이터 분석의 필수 도구로, 프로젝트 메트릭을 효과적으로 요약하고 분석할 수 있게 합니다. 최대 판매를 파악하거나 평균 작업 기간을 계산하거나 예산 할당을 합산하려는 경우, Blue의 논리는 간단하게 만들어 줍니다.

복잡한 계산을 단순화하고 명확하게 제시함으로써, Blue는 프로젝트 데이터를 실행 가능한 통찰력으로 변환하여 전략적 의사 결정을 지원합니다.

## 표시 구성

통계 카드를 만드는 마지막 단계는 데이터를 표시하는 방법을 결정하는 것입니다.

통계 카드는 세 가지 표시 옵션을 제공합니다:

1. **숫자**: 간단한 숫자 표시를 선택하며, 정밀도 수준을 설정할 수 있는 추가 기능이 있습니다. 이는 소수점 자릿수를 지정할 수 있어 데이터 분석에 필요한 세부 수준에 맞는 정밀도를 제공합니다.
2. **통화**: 이 옵션을 선택하여 금전적 용어로 값을 나타냅니다. 예산 추적, 수익, 비용 또는 기타 재무 통계와 같은 재무 메트릭에 이상적입니다. 통화 형식은 적절한 지역 형식 및 통화 기호를 적용합니다.
3. **백분율**: 값을 백분율로 표현하는 데 이 형식을 사용합니다. 완료율, 성장 메트릭 또는 기타 비율 데이터를 표시하는 데 적합합니다. 이 옵션은 계산된 값을 자동으로 백분율 형식으로 변환하여 비율과 점유율을 즉시 이해할 수 있게 합니다.
