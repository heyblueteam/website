[Pabbly Connect](https://payments.pabbly.com/api/affurl/RVYZ07kQyUZ0Z1HUKZ1m/zp2GoChM1JcZ0dQBo?target=nMfdbf0I90K3UdJn)를 사용하면 Blue를 Google Calendar, Stripe, MailChimp 등과 같은 많은 인기 앱과 연결할 수 있습니다. 이를 통해 도구 간에 데이터를 전송하여 워크플로우를 자동화할 수 있습니다.

<youtube url="https://www.youtube.com/watch?v=IRxYAe5snzY" />

Pabbly Connect를 사용하여 Blue를 통합할 수 있는 무한한 가능성이 있습니다. 다음은 주요 단계입니다:

 ## Pabbly 대시보드에 액세스하기

1. [Pabbly 대시보드](https://connect.pabbly.com/workflows)에 로그인합니다.
2. "+ 워크플로우 생성" 버튼을 클릭합니다.
3. 워크플로우 이름을 지정합니다.
4. Blue를 대상 앱으로 선택합니다.

## Blue 계정에 연결하기

1. Blue 프로필에서 [API 토큰](/docs/integrations/api)을 생성합니다.
2. Pabbly에서 다음 정보를 입력합니다:
- 토큰 ID
- 비밀 키
- 회사 URL (예: `https://app.blue.cc/company/mycompany`)
- 프로젝트 ID (예: `myproject`)

## 테스트 통합 생성하기
1. 트리거 앱과 이벤트를 선택합니다 (예: 새로운 Google Calendar 이벤트).
2. 테스트 요청을 보냅니다.
3. Blue에서 지정한 프로젝트에 테스트 데이터가 수신되었는지 확인합니다.
