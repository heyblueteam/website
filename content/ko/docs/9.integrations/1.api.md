Blue는 API(응용 프로그램 프로그래밍 인터페이스)를 제공하므로 필요에 따라 자체 통합을 구축할 수 있습니다.

**Blue는 100% API 지원을 제공합니다**. 이는 [우리의 모든 앱](/docs/start-guide/download-apps)에서 할 수 있는 모든 작업을 프로그래밍 방식으로 수행할 수 있음을 의미합니다. 우리는 쿼리를 테스트할 수 있는 전체 플레이그라운드가 있는 강력한 [GraphQL](https://graphql.org/) API를 제공하며, 완전히 문서화되어 있습니다.

::callout
