## 데이터 내보내기

1. 내보내고자 하는 프로젝트에서 설정 아이콘(톱니바퀴)을 클릭합니다.
2. "내보내기"를 선택합니다.
3. "내보내기"를 클릭하여 Blue 서버에서 내보내기 프로세스를 시작합니다.
4. 완료되면 내보낸 CSV 파일을 다운로드할 수 있는 링크가 포함된 이메일을 받게 됩니다.

::callout
 ---

icon: lightbulb
---

**데이터가 포함된 이메일을 받는 데 얼마나 걸리나요?** <br/> <br/>
일반적으로 몇 초 이내에 도착하지만, 다른 고객의 내보내기 프로세스 수와 데이터 크기에 따라 달라질 수 있습니다. 100,000개 이상의 항목이 포함된 데이터 세트의 경우 몇 분이 걸릴 수 있습니다.
::

::

## 활성 필터로 내보내기

프로젝트에 이미 필터를 적용한 경우 "활성 필터로 내보내기" 옵션을 사용하여 필터링된 기록만 쉽게 내보낼 수 있습니다. 이렇게 하면 내보낸 데이터를 수동으로 필터링할 필요가 없으며, 현재 필터 설정과 일치하는 기록만 자동으로 포함됩니다.

활성 필터로 내보내는 방법은 다음과 같습니다:

1. 담당자, 태그 또는 고급 필터와 같은 사용 가능한 필터 옵션을 사용하여 프로젝트에 필터를 적용합니다.
2. 내보내고자 하는 필터링된 뷰를 얻은 후, 프로젝트 설정으로 이동하여 가져오기 및 내보내기를 클릭합니다.
3. 내보내기 옵션 메뉴에서 "활성 필터로 내보내기"를 선택합니다.
4. 내보내기가 준비되면 이메일로 데이터를 받게 됩니다.

## 주요 이점

- **데이터 이식성**: 필요할 때 Blue에서 프로젝트를 다운로드하여 백업 또는 외부 분석을 수행할 수 있습니다.
- [API를 통한 트리거](/docs/integrations/api): 타사 시스템을 사용하여 정기적으로 Blue 내보내기를 자동으로 트리거하여 데이터 백업 요구 사항을 충족할 수 있습니다.
- **이해관계자 접근성**: 내보낸 CSV를 클라이언트, 관리자 및 Blue 외부의 다른 사람들과 쉽게 공유할 수 있습니다.

내보내기는 다양한 잠재적 통합 및 보고 요구 사항에 맞는 데이터 접근성과 유연성을 제공합니다. [가져오기](/docs/integrations/csv-import)와 결합하여 Blue 데이터에 대한 완전한 제어를 제공합니다.
