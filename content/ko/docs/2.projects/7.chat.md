채팅을 통해 구성원들은 플랫폼 내에서 서로 소통할 수 있습니다. 주제를 논의하고, 업데이트를 공유하며, 질문을 하고, 협업하는 방법입니다.

::callout
---

icon: lightbulb
---

Blue는 프로젝트 내 주제 기반 채팅을 지원하지만, 개인에게 직접 메시지를 보내는 기능은 지원하지 않습니다.
::

## 채팅 생성

새 채팅을 시작하려면:
1. 왼쪽 사이드바의 채팅 아이콘을 클릭합니다.
2. 상단의 "새 채팅" 버튼을 클릭합니다.
3. 준비가 되면 "채팅 생성"을 클릭합니다.

다양한 주제를 논의하기 위해 여러 채팅을 생성할 수 있습니다.

![Chat 1](/docs/Chat_1.png "Chat 1")

"+" 버튼을 사용하여 새 채팅을 생성하고 주제 이름을 설정하세요

## 채팅에서 소통하기

채팅에 참여한 구성원들은 다음과 같은 작업을 할 수 있습니다:
- 텍스트 메시지 전송
- 파일을 드래그 앤 드롭하여 이미지 업로드
- 다른 구성원을 @멘션하여 알림 보내기
- YouTube 비디오 URL을 붙여넣어 비디오 임베드

![Chat 2](/docs/Chat_2.png "Chat 2")

이미지를 업로드하고 토론을 맥락에 맞게 유지하세요

채팅을 통해 긴 이메일 스레드 없이도 유동적이고 지속적인 토론이 가능합니다.

## 채팅 기능

- 채팅 메시지 검색
- 이전 메시지 편집
- 메시지 삭제

Blue의 채팅을 사용하면 주요 대화와 공유 파일을 한 곳에서 체계적으로 관리할 수 있습니다. 통합된 알림과 멘션 기능을 통해 원활한 팀 커뮤니케이션이 가능합니다.
