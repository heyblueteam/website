Blue API는 현대적인 [GraphQL](https://graphql.org/) 인터페이스를 통해 Blue의 프로세스 관리 플랫폼에 완전한 접근을 제공합니다. **17,000명 이상의 고객**이 **연간 수십억 건의 API 요청**을 처리하는 신뢰받는 API로, **[99.99% 가동 시간](/platform/status)**으로 미션 크리티컬 워크플로우를 지원합니다.

## 왜 Blue API인가?

- **100% 기능 커버리지** - Blue의 UI에서 할 수 있는 모든 것을 API를 통해 할 수 있습니다.
- **기업 준비 완료** - 포괄적인 속도 제한 및 보안으로 대규모에서 검증됨
- **개발자 친화적** - GraphQL은 필요한 데이터만 정확하게 제공합니다.
- **실시간 업데이트** - 즉각적인 데이터 동기화를 위한 WebSocket 구독

## 빠른 시작

첫 번째 API 호출을 할 준비가 되셨나요? 이렇게 간단합니다:

```graphql
query MyProjects {
  projectList(filter: { companyIds: ["your-company-id"] }) {
    items {
      id
      name
      updatedAt
    }
  }
}
```

**[우리의 GraphQL Playground에서 시도해 보세요 →](https://api.blue.cc/graphql)**

## 인기 사용 사례

### 데이터 관리
- 맞춤형 애플리케이션을 위한 유연한 데이터베이스로 Blue 사용
- Blue와 기존 시스템 간 데이터 동기화
- 맞춤형 대시보드 및 보고 도구 구축

### 워크플로우 자동화
- 외부 트리거(양식, 웹훅, 이메일)로부터 레코드 생성
- 비즈니스 로직에 따라 여러 프로젝트 업데이트
- 작업 할당 및 알림 자동화

### 통합 개발
- Blue를 외부 CRM, ERP 및 전문 도구와 연결
- 귀하의 산업에 맞는 맞춤형 통합 구축
- 귀하의 기능으로 Blue의 기능 확장
