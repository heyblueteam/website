---
title: 회사를 생성하기
description: Blue API를 사용하여 새로운 회사를 생성하는 방법.
---

새로운 회사를 생성하려면 다음 변형을 사용할 수 있습니다:

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

새로운 회사를 생성하면 각 회사가 별도로 청구되므로 Blue 계정의 청구에 영향을 미칠 수 있습니다.

다음은 `createCompany` 변형의 옵션과 그 설명입니다:

| 옵션 | 유형 | 설명 |
|--------|------|-------------|
| name | String | 새로운 회사의 이름입니다. 필수 입력란입니다. |  |
| slug | 문자열 | 회사 이름의 URL 친화적인 버전입니다. 회사의 URL에 사용되며 고유해야 합니다. 필수 입력란이며 회사의 고유 식별자로도 작용합니다. |
