우리는 Blue API와 상호작용하는 데 사용할 수 있는 [파이썬 라이브러리](https://pypi.org/project/bluepm/)를 제공합니다. 이 라이브러리는 타입 안전성과 현대 IDE에서의 코드 완성을 갖춘 100% Blue API 커버리지를 제공합니다.

다음 명령어를 실행하여 설치할 수 있습니다:

```bash
pip install bluepm
```

그런 다음 코드에서 라이브러리를 가져와 사용할 수 있습니다:

```python [python]
from bluepm import BlueAPIClient
```

인증 세부정보를 환경 변수로 설정할 수 있습니다.

환경 변수를 사용하려면 스크립트를 실행하기 전에 다음을 설정하십시오:

- BLUE_TOKEN_ID: 귀하의 Blue API 토큰 ID
- BLUE_SECRET_ID: 귀하의 Blue API 비밀 ID
- BLUE_COMPANY_ID: 귀하의 Blue 회사 ID (선택 사항)
- BLUE_PROJECT_ID: 귀하의 Blue 프로젝트 ID (선택 사항)

또는, 이를 BlueAPIClient 생성자에 직접 전달할 수 있습니다:

```python [python]
client = BlueAPIClient(
    token_id="your_token_id",
    secret_id="your_secret_id",
    company_id="your_company_id" #optional
    project_id = "your_project_id" #optional
)
```

이 작업을 완료하면 라이브러리를 사용하여 Blue API와 상호작용할 수 있습니다. 우리는 일반적인 작업을 단순화하기 위한 몇 가지 도우미를 제공하지만, 필요한 모든 상호작용을 허용하기 위해 전체 API가 노출되어 있습니다.

예를 들어, 접근할 수 있는 모든 프로젝트의 목록을 가져오려면 다음 코드를 사용할 수 있습니다:

```python [python]
projects = client.projects.list(company_id)
```

프로젝트에 대한 목록의 목록을 가져오려면 다음 코드를 사용할 수 있습니다:

```python [python]
todo_lists = client.get_todo_lists(project_id)
```
