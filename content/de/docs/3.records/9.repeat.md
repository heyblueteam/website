title: Wiederholen

Sie können wiederkehrende Datensätze in Blue verwenden, um sich wiederholende Aufgaben zu automatisieren, sodass Sie nicht jedes Mal denselben Datensatz manuell neu erstellen müssen. Dies ist nützlich für geplante Arbeiten wie Wartungsüberprüfungen oder wiederkehrende Tätigkeiten. Wiederkehrende Datensätze folgen derselben Struktur, indem sie automatisch Checklisten und benutzerdefinierte Felddaten übernehmen.


## Erstellen von wiederkehrenden Datensätzen

Um einen wiederkehrenden Datensatz einzurichten, öffnen Sie den Datensatz, den Sie wiederholen möchten, und klicken Sie auf das ••• Menü und dann auf Wiederholen:

![Repeat 1](/docs/Repeat_1.png "Repeat 1")


Ihnen werden dann die folgenden Optionen angezeigt:

![Repeat 2](/docs/Repeat_2.png "Repeat 2")


## Wiederholungsintervalle

Sie können wählen zwischen:
- Täglich
- Wöchentlich
- Monatlich
- Jährlich

Oder erstellen Sie einen benutzerdefinierten Wiederholungszeitplan, der Ihnen deutlich mehr Flexibilität und Anpassungsoptionen bietet:

![Repeat 3](/docs/Repeat_3.png "Repeat 3")

Hier können Sie wählen:

- **Beginnend ab:** Wählen Sie das genaue Datum, ab dem der Datensatz wiederholt werden soll.
- **Wiederholen alle:** Wählen Sie, ob der Datensatz täglich, wöchentlich, monatlich oder jährlich wiederholt werden soll.
- **Endbedingung:** Wählen Sie, ob der Datensatz unbegrenzt wiederholt werden soll oder nach einem bestimmten Datum oder einer bestimmten Anzahl von Wiederholungen enden soll.

## Kopieroptionen

Wenn eine wiederkehrende Aufgabe neu erstellt wird, können Sie wählen, welche Daten von der ursprünglichen Aufgabe übernommen werden:

- Zugewiesene Personen
- Tags
- Benutzerdefinierte Felder
- Checklisten
- Kommentare
- Beschreibung

![Repeat 4](/docs/Repeat_4.png "Repeat 4")


Sie können auch wählen, in welcher Liste der Datensatz wiederholt werden soll. Derzeit unterstützen wir nur die Wiederholung von Datensätzen innerhalb desselben Projekts, nicht projektübergreifend.

::callout
---

icon: lightbulb
---

Beachten Sie, dass das Löschen des ursprünglichen Datensatzes dazu führt, dass keine neuen Datensätze mehr mit der Wiederholungsoption erstellt werden.
::

## Aktive wiederkehrende Datensätze

Für alle Datensätze, die sich wiederholen, wird das Wiederholungssymbol auf der Datensatzkarte in der [Board-Ansicht](/de/docs/views/board) angezeigt:

![Repeat 5](/docs/Repeat_5.png "Repeat 5")

Und auch innerhalb des einzelnen Datensatzes:

![Repeat 6](/docs/Repeat_6.png "Repeat 6")


## Beenden der Wiederholung

Um die Wiederholung eines Datensatzes zu beenden:
1. Öffnen Sie den ursprünglichen wiederkehrenden Datensatz
2. Klicken Sie auf das Wiederholungssymbol
3. Wählen Sie die Option "Wiederholung beenden"

![Repeat 7](/docs/Repeat_7.png "Repeat 7")


Dies verhindert, dass weitere wiederkehrende Aufgaben erstellt werden.
