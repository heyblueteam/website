## Daten exportieren

1. Klicken Sie im Projekt, das Sie exportieren möchten, auf das Zahnradsymbol (Einstellungen).
2. Wählen Sie "Exportieren".
3. Klicken Sie auf "Exportieren", um den Exportprozess auf den Blue-Servern auszulösen.
4. Nach Abschluss erhalten Sie eine E-Mail mit einem Link zum Herunterladen der exportierten CSV-Datei.

::callout
 ---

icon: lightbulb
---

**Wie lange dauert es, bis ich die E-Mail mit den Daten erhalte?** <br/> <br/>
Typischerweise kommt diese innerhalb weniger Sekunden an, kann jedoch davon abhängen, wie viele andere Exportprozesse von anderen Kunden laufen und wie groß Ihre Daten sind. Bei Datensätzen mit über 100.000 Elementen kann dies einige Minuten dauern.
::

Bitte beachten Sie, dass die Exportfunktion nur Ihre Datensätze exportiert. Dateien im Tab "Dateien" müssen manuell in großen Mengen heruntergeladen werden.
::

## Export mit aktiven Filtern

Wenn Sie bereits Filter auf Ihr Projekt angewendet haben, können Sie ganz einfach nur die gefilterten Datensätze exportieren, indem Sie die Option "Export mit aktiven Filtern" verwenden. Auf diese Weise müssen Sie sich keine Gedanken über das manuelle Filtern der exportierten Daten machen, da nur die Datensätze, die Ihren aktuellen Filtereinstellungen entsprechen, automatisch einbezogen werden.

So können Sie mit aktiven Filtern exportieren:

1. Wenden Sie die Filter auf Ihr Projekt an, indem Sie die verfügbaren Filteroptionen wie Zuweisungen, Tags oder erweiterte Filter verwenden.
2. Sobald Sie die gefilterte Ansicht haben, die Sie exportieren möchten, gehen Sie zu den Projekteinstellungen und klicken Sie auf Import & Export.
3. Wählen Sie im Menü der Exportoptionen "Export mit aktiven Filtern".
4. Sobald Ihr Export bereit ist, erhalten Sie die Daten in Ihrer E-Mail.

## Hauptvorteile

- **Datenportabilität**: Projekte bei Bedarf aus Blue herunterladen, um sie für Backups oder externe Analysen zu verwenden.
- [Auslösen über API](/docs/integrations/api): Sie können Drittsysteme verwenden, um Blue-Exporte regelmäßig automatisch auszulösen, um Ihren Anforderungen an die Datensicherung gerecht zu werden.
- **Zugänglichkeit für Stakeholder**: Exportierte CSVs einfach mit Kunden, Managern und anderen außerhalb von Blue teilen.

Der Export bietet Datenzugänglichkeit und Flexibilität, um verschiedenen potenziellen Integrations- und Berichtsanforderungen gerecht zu werden. In Kombination mit [Import](/docs/integrations/csv-import) bietet es vollständige Kontrolle über Ihre Blue-Daten.
