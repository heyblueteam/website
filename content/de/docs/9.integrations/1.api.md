Blue hat eine API (Application Programming Interface), sodass Sie bei Bedarf Ihre eigenen Integrationen erstellen können.

**Blue hat eine 100%ige API-Abdeckung**. Das bedeutet, dass alles, was Sie in [einer unserer Apps](/docs/start-guide/download-apps) tun können, auch programmgesteuert möglich ist. Wir haben eine leistungsstarke [GraphQL](https://graphql.org/) API, die einen vollständigen Playground bietet, in dem Sie Ihre Abfragen testen können und die vollständig dokumentiert ist.
