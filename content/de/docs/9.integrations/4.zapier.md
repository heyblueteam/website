Zapier ist eine Integrationsplattform, die hilft, verschiedene Webanwendungen zu verbinden und Arbeitsabläufe zu automatisieren.

**Anwendungsfälle**

Hier sind einige der beliebtesten Möglichkeiten, wie Blue über Zapier mit anderen Apps integriert ist:

**_Synchronisieren Sie Blue-Datensätze mit Google Sheets_**

Wenn Blue-Datensätze erstellt oder aktualisiert werden, fügen Sie sie automatisch als neue Zeilen in ein Google Sheet für zentrale Berichterstattung hinzu.

**_Bestellbestätigung aus Blue-Datensätzen senden_**

Wenn Sie ein bestimmtes benutzerdefiniertes Feld in Blue auswählen, senden Sie automatisch eine E-Mail, um die Bestellung bei Ihrem Kunden zu bestätigen.

**_Blue-Kommentare zu Slack hinzufügen_**

Erhalten Sie Echtzeit-Benachrichtigungen von Blue, die in die richtigen Slack-Kanäle gepostet werden, damit Ihr Team über wichtige Aktualisierungen der Datensätze informiert ist.

## Hauptvorteile

**Automatisierung über Tools hinweg**: Zaps, die in Zapier erstellt werden, können automatisch Daten zwischen Blue und Tausenden von anderen Apps senden und empfangen. Dies eliminiert manuelle Arbeiten beim Verschieben von Daten zwischen Systemen.

**Echtzeit-Integrationen**: Mit den Webhook-Triggern von Blue reagieren Zaps sofort auf Ereignisse in Blue, wie z. B. einen neuen Datensatz, hinzugefügten Kommentar, Statusänderung usw. Dies ermöglicht eine Echtzeitsynchronisation und Prozessautomatisierung.

**Einfach zu erstellen und zu warten**: Zaps bieten eine benutzerfreundliche, codefreie Möglichkeit zur Integration von Tools. Es sind keine Entwicklungsarbeiten erforderlich. Und sie funktionieren weiterhin ohne laufende Wartung.

1. Melden Sie sich in Ihrem Blue-Konto unter Profileinstellungen an
2. Gehen Sie zu den Profileinstellungen und klicken Sie auf "Webhooks"
3. Klicken Sie auf "Webhook hinzufügen"
4. Geben Sie dem Webhook einen Namen und geben Sie die Zapier-Webhook-URL ein
5. Wählen Sie aus, welche Ereignisse den Webhook auslösen sollen
6. Klicken Sie auf "Webhook erstellen"

## Blue zu Zapier

1. Melden Sie sich in Ihrem Zapier-Konto an
2. Klicken Sie auf "Zap erstellen"
3. Suchen Sie im ersten Schritt nach der App ["Webhooks by Zapier" Trigger-App](https://zapier.com/apps/webhook) und wählen Sie sie aus
4. Konfigurieren Sie es so, dass es Payloads von Ihrer Blue-Webhook-URL akzeptiert
5. Suchen und wählen Sie im zweiten Schritt die Ziel-App aus, an die Daten gesendet werden sollen
6. Ordnen Sie die JSON-Payload-Felder den Eingabefeldern der Ziel-App zu
7. Aktivieren Sie den Zap

Jetzt, wenn die angegebenen Ereignisse in Blue auftreten, wird die Webhook-Payload an Zapier gesendet, das die Daten automatisch gemäß den Zap-Einstellungen an Ihre integrierte App weitergibt.

## Zapier zu Blue

Um Daten von anderen Apps in Blue zu übertragen, befolgen Sie diese Schritte:

**1. Blue API-Anmeldeinformationen erhalten**
- Gehen Sie zu Blue Einstellungen > API-Token
- Erstellen Sie ein neues Token oder verwenden Sie vorhandene Anmeldeinformationen:
  - Token-ID
  - Token-Geheimnis
  - Unternehmens-ID
  - Projekt-ID

**2. Zap in Zapier erstellen**
1. Richten Sie Ihre Trigger-App ein (z. B. Google Sheets, Slack usw.)
2. Fügen Sie "Webhooks by Zapier" als Aktions-App hinzu
3. Wählen Sie den Ereignistyp "Benutzerdefinierte Anfrage"

**3. Webhook konfigurieren**
- Methode: POST
- URL: `https://api.blue.cc/graphql` (oder Beta-Endpunkt, falls erforderlich)
- Daten-Pass-Through: FALSCH
- Unflatten: NEIN

**4. Header festlegen:**
```json
{
  "Content-Type": "application/json",
  "X-Bloo-Token-ID": "Your_Token_ID",
  "X-Bloo-Token-Secret": "Your_Token_Secret", 
  "X-Bloo-Company-ID": "Your_Company_ID",
  "X-Bloo-Project-ID": "Your_Project_ID"
}
```

**5. Abfrage konfigurieren (Beispielmutation):**
```json
{
  "query": "mutation CreateTodo($input: CreateTodoInput!) {\n  createTodo(input: $input) {\n    id\n    title\n    customFields {\n      value\n    }\n  }\n}",
  "variables": {
    "input": {
      "todoListId": "your-todo-list-id",
      "title": "New task from Zapier",
      "customFields": [
        {
          "customFieldId": "priority-field-id",
          "value": "High"
        },
        {
          "customFieldId": "due-date-field-id",
          "value": "2024-03-01"
        }
      ]
    }
  }
}
```

**6. Testen & Aktivieren**
- Verwenden Sie den API Playground von Blue, um Mutationen zuerst zu testen
- Ordnen Sie dynamische Werte von Trigger-Apps mithilfe der Datenfelder von Zapier zu
- Aktivieren Sie den Zap nach erfolgreichem Testen
