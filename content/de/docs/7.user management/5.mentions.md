Durch die Verwendung des @-Symbols gefolgt von dem Namen einer Person können Sie sie in einem Kommentar oder einer Beschreibung markieren.

![Erwähnungen 1](/docs/mention.gif "Mentions 1")

Einfach @Erwähnungen von Teammitgliedern in jedem Textbereich

Dies löst eine Benachrichtigung für diesen Benutzer aus, die ihn auf die Erwähnung aufmerksam macht. Es lenkt ihre Aufmerksamkeit genau dorthin, wo sie benötigt wird.

@Erwähnungen können verwendet werden, um:

- **Aufgaben zu delegieren**: Arbeiten oder Verantwortlichkeiten zuweisen, indem Sie den Eigentümer @erwähnen.
- **Fragen zu stellen**: Antworten oder Input von Experten suchen, indem Sie sie in Ihrer Anfrage @erwähnen.
- **Anerkennung zu geben**: Wertschätzung zeigen, indem Sie Teamkollegen @erwähnen, die großartige Arbeit geleistet haben.
- **Eigentum zu klären**: Dokumentieren, wer verantwortlich ist, indem Sie den Eigentümer in Notizen @erwähnen.
- **Stakeholder zu benachrichtigen**: Kunden oder Manager auf dem Laufenden halten, indem Sie sie in Updates @erwähnen.

::callout
 ---

icon: lightbulb
---

Sie können nur Personen @erwähnen, die in diesem bestimmten Projekt sind. Wenn eine Person nicht im Projekt ist, wird sie nicht in der Dropdown-Vorschlagsliste der Personen angezeigt, die Sie @erwähnen können.
::

## Gruppen @Erwähnungen

Sie können Gruppen von Benutzern basierend auf ihren zugewiesenen Rollen einfach @erwähnen oder alle Beteiligten in einem Projekt @erwähnen. Diese Funktion stellt sicher, dass relevante Mitglieder rechtzeitig Benachrichtigungen erhalten und über wichtige Updates oder Diskussionen informiert bleiben.

Um Benutzer basierend auf ihren Rollen @zu erwähnen, können Sie die folgenden Optionen verwenden:

**Vordefinierte Rollen:**

- @Admins: Alle Benutzer mit administrativen Rechten @erwähnen.
- @Teammitglieder: Alle Benutzer, die Mitglieder des Projekts sind, @erwähnen.
- @Kunden: Alle Kunden, die mit dem Projekt verbunden sind, @erwähnen.
- @Nur Ansicht: Benutzer @erwähnen, die nur Lesezugriff auf das Projekt haben.
- @Nur Kommentar: Benutzer @erwähnen, die nur Kommentare im Projekt hinterlassen können.
- @jeder: Alle @erwähnen, die bezüglich ihrer Rollen im Projekt relevant sind.

**Benutzerdefinierte Rollen:**

- @Benutzerdefinierte Benutzerrollen: Benutzer @erwähnen, die spezifischen benutzerdefinierten Rollen zugewiesen sind, die Sie im Projekt definiert haben.

Wenn Sie eine Gruppe nach ihrer Rolle @erwähnen, erhalten alle Benutzer innerhalb dieser ausgewählten Rolle Benachrichtigungen über ihre konfigurierten Kanäle (z. B. E-Mail oder Push-Benachrichtigungen). Darüber hinaus wird die @Erwähnung in ihrem speziellen "@Erwähnung"-Bereich innerhalb von Blue angezeigt, um sicherzustellen, dass sie keine wichtigen Updates oder Gespräche verpassen.
