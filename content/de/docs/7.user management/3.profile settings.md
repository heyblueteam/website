## Übersicht

Sie können auf Ihr Profil zugreifen, indem Sie auf Ihr Profilbild/Ihre Initialen in der oberen rechten Ecke von Blue klicken.

Die wichtigsten Einstellungen, die Sie anpassen können, sind:

- Name und E-Mail
- Profilbild
- Farbthema
- Datumsformat
- Geburtstag

![Profileinstellung 1](/docs/ProfileSetting_1.png "Profile Setting 1")

Profileinstellung

Lassen Sie uns jede dieser Einstellungen erkunden:

## Name und E-Mail

Sie können Ihren Namen und Ihre E-Mail-Adresse in Ihrem Profil aktualisieren. Ihr Name wird in Blue angezeigt und Ihre E-Mail wird für Benachrichtigungen verwendet.

## Profilbild

Das Hinzufügen eines Profilbildes erleichtert es Ihrem Team, Sie zu erkennen. Sie können jedes beliebige Bild hochladen, es zuschneiden und als Ihr Profilbild festlegen.

## Farbthema

Passen Sie das Farbthema an Ihr Branding an, indem Sie aus vordefinierten Optionen wie Blau, Pink, Grün, Lila, Orange, Rot und Schwarz wählen. Diese Farbe wird in Hervorhebungen und Akzenten auf der Plattform widergespiegelt.

## Datumsformat

Stellen Sie Ihr bevorzugtes Datumsformat ein - Sie können zwischen dem US-Datumsstil MM/DD/YYYY oder dem weltweiten Stil DD/MM/YYYY wählen.

![Profileinstellung 2](/docs/ProfileSetting_2.png "Profile Setting 2")

## Geburtstag

Fügen Sie Ihren Geburtstag hinzu, damit Ihr Blue-Team Ihren besonderen Tag feiern kann! Ihr Geburtstag wird auf Ihrer Profilseite angezeigt.

## Benutzerdefinierte Zeitzone

Stellen Sie eine benutzerdefinierte Zeitzone basierend auf Ihrem Standort im Hauptquartier ein, um sicherzustellen, dass alle Fälligkeiten und Zeitpläne nahtlos mit Ihren Kollegen übereinstimmen. Diese Einstellung ist besonders wichtig für Teams auf der ganzen Welt oder für ein Remote-Büro.
