---
title: Ein Unternehmen erstellen
description: So erstellen Sie ein neues Unternehmen mit der Blue API.
---

Um ein neues Unternehmen zu erstellen, können Sie die folgende Mutation verwenden:

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

Bitte beachten Sie, dass die Erstellung neuer Unternehmen die Abrechnung Ihres Blue-Kontos beeinflussen kann, da jedes Unternehmen separat abgerechnet wird.

Hier sind die Optionen für die `createCompany` Mutation und deren Beschreibungen:

| Option | Typ | Beschreibung |
|--------|------|-------------|
| name | String | Der Name des neuen Unternehmens. Dies ist ein Pflichtfeld. |  |
| slug | String | Eine URL-freundliche Version des Unternehmensnamens. Dies wird in der URL des Unternehmens verwendet und sollte einzigartig sein. Dies ist ein Pflichtfeld und fungiert auch als eindeutiger Identifikator für das Unternehmen. |
