[Pabbly Connect](https://payments.pabbly.com/api/affurl/RVYZ07kQyUZ0Z1HUKZ1m/zp2GoChM1JcZ0dQBo?target=nMfdbf0I90K3UdJn) stelt je in staat om Blue te verbinden met veel populaire apps zoals Google Calendar, Stripe, MailChimp, en meer. Dit maakt de automatisering van workflows mogelijk door gegevens tussen tools te verzenden.

<youtube url="https://www.youtube.com/watch?v=IRxYAe5snzY" />

Er zijn eindeloze mogelijkheden voor het integreren van Blue met Pabbly Connect. Hier zijn de belangrijkste stappen:

 ## Toegang tot je Pabbly Dashboard

1. Log in op je [Pabbly Dashboard](https://connect.pabbly.com/workflows)
2. Klik op de knop "+ Workflow maken".
3. Geef je workflow een naam.
4. Selecteer Blue als de bestemmingsapp.

## Verbinden met je Blue-account

1. Genereer een [API-token](/docs/integrations/api) in je Blue-profiel.
2. Voer in Pabbly je in:
- Token ID
- Secret Key
- Bedrijfs-URL (bijv. `https://app.blue.cc/company/mycompany`)
- Project ID (bijv. `myproject`)

::callout
 ---

icon: lightbulb
---

*Kopieer je Secret Key onmiddellijk, aangezien deze niet opnieuw zal worden weergegeven.* 
::

## Maak een testintegratie
1. Selecteer een trigger-app en gebeurtenis (bijv. nieuwe Google Calendar-gebeurtenis).
2. Stuur een testverzoek.
3. Controleer Blue om te bevestigen dat de testgegevens zijn ontvangen in je opgegeven project.
