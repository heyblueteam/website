Blue heeft een API (Application Programming Interface), zodat je je eigen integraties kunt bouwen indien nodig.

**Blue heeft 100% API-dekking**. Dit betekent dat alles wat je kunt doen in [een van onze apps](/docs/start-guide/download-apps), je ook programmatisch kunt doen. We hebben een krachtige [GraphQL](https://graphql.org/) API die een volledige playground heeft waar je je queries kunt testen en die volledig gedocumenteerd is.
