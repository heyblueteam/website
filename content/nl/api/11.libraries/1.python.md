We bieden een [Python-bibliotheek](https://pypi.org/project/bluepm/) die kan worden gebruikt om te communiceren met de Blue API. Dit heeft 100% dekking van de Blue API met typeveiligheid en code-aanvulling in elke moderne IDE.

Je kunt installeren door het volgende uit te voeren:

```bash
pip install bluepm
```

Je kunt de bibliotheek vervolgens importeren en gebruiken in je code:

```python [python]
from bluepm import BlueAPIClient
```

Je kunt je authenticatiegegevens instellen als omgevingsvariabelen.

Om omgevingsvariabelen te gebruiken, stel je het volgende in voordat je je script uitvoert:

- BLUE_TOKEN_ID: Je Blue API-token-ID
- BLUE_SECRET_ID: Je Blue API-geheim-ID
- BLUE_COMPANY_ID: Je Blue bedrijfs-ID (optioneel)
- BLUE_PROJECT_ID: Je Blue project-ID (optioneel)

Of, geef ze direct door aan de constructor van BlueAPIClient:

```python [python]
client = BlueAPIClient(
    token_id="your_token_id",
    secret_id="your_secret_id",
    company_id="your_company_id" #optional
    project_id = "your_project_id" #optional
)
```

Zodra je dit hebt gedaan, kun je de bibliotheek gebruiken om te communiceren met de Blue API. We hebben enkele hulpprogramma's om veelvoorkomende taken te vereenvoudigen, maar de gehele API is beschikbaar om elke interactie die je nodig hebt mogelijk te maken.

Bijvoorbeeld, om een lijst van alle projecten waar je toegang toe hebt te krijgen, kun je de volgende code gebruiken:

```python [python]
projects = client.projects.list(company_id)
```

Om een lijst van lijsten voor een project te krijgen, kun je de volgende code gebruiken:

```python [python]
todo_lists = client.get_todo_lists(project_id)
```
