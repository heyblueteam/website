## Overzicht

Tags zijn een krachtige manier om records in Blue te categoriseren en te organiseren. Het tagsysteem biedt uitgebreide functionaliteit voor het maken, beheren en filteren van tags, met ondersteuning voor bulkbewerkingen en geavanceerd filteren.

## Lijst Tags

Haal tags op uit projecten met flexibele filter- en sorteermogelijkheden.

### Basisvoorbeeld

```graphql
query ListTags {
  tagList(
    filter: { 
      projectIds: ["project-123"] 
      excludeArchivedProjects: false 
    }
    first: 50
    orderBy: title_ASC
  ) {
    items {
      id
      uid
      title
      color
      project {
        id
        name
      }
      createdAt
      updatedAt
    }
    pageInfo {
      totalPages
      totalItems
      page
      perPage
      hasNextPage
      hasPreviousPage
    }
    totalCount
  }
}
```

### Geavanceerd Filtervoorbeeld

```graphql
query FilterTags {
  tagList(
    filter: {
      projectIds: ["project-123", "project-456"]
      search: "urgent"
      colors: ["#ff0000", "#ffaa00"]
      excludeArchivedProjects: true
    }
    orderBy: createdAt_DESC
    first: 20
  ) {
    items {
      id
      title
      color
      todos {
        id
        title
      }
    }
    totalCount
  }
}
```

## Maak Tags

Maak nieuwe tags binnen een project.

```graphql
mutation CreateTag {
  createTag(
    input: {
      title: "High Priority"
      color: "#ff0000"
    }
  ) {
    id
    uid
    title
    color
    project {
      id
      name
    }
    createdAt
  }
}
```

## Update Tags

Wijzig bestaande tag-eigenschappen.

```graphql
mutation UpdateTag {
  editTag(
    input: {
      id: "tag-123"
      title: "Critical Priority"
      color: "#cc0000"
    }
  ) {
    id
    title
    color
    updatedAt
  }
}
```

## Verwijder Tags

Verwijder tags uit het systeem. Dit verwijdert ook de tag uit alle bijbehorende records.

```graphql
mutation DeleteTag {
  deleteTag(id: "tag-123")
}
```

## Tag Records

Koppel tags aan records (to-do's) met behulp van bestaande tags of maak nieuwe aan.

### Gebruik Bestaande Tags

```graphql
mutation TagRecord {
  setTodoTags(
    input: {
      todoId: "todo-123"
      tagIds: ["tag-456", "tag-789"]
    }
  )
}
```

### Maak Tags Tijdens Taggen

Je kunt nieuwe tags on-the-fly maken door tagtitels op te geven:

```graphql
mutation TagRecordWithNewTags {
  setTodoTags(
    input: {
      todoId: "todo-123"
      tagIds: ["existing-tag-id"]
      tagTitles: ["New Tag", "Another New Tag"]
    }
  )
}
```

## Invoervariabelen

### TagListFilter

| Parameter | Type | Vereist | Beschrijving |
|-----------|------|---------|--------------|
| `projectIds` | [String!] | Nee | Filter tags op project-ID's of slugs |  |
| `excludeArchivedProjects` | Boolean | Nee | Sluit tags uit gearchiveerde projecten uit |  |
| `search` | String | Nee | Zoek tagtitels (hoofdletterongevoelig) |  |
| `titles` | [String!] | Nee | Filter op specifieke tagtitels |  |
| `colors` | [String!] | Nee | Filter op specifieke kleuren (hex-formaat) |  |
| `tagIds` | [String!] | Nee | Filter op specifieke tag-ID's |  |

### CreateTagInput

| Parameter | Type | Vereist | Beschrijving |
|-----------|------|---------|--------------|
| `title` | String | Nee | Tagtitel (automatisch gegenereerd als niet opgegeven) |  |
| `color` | String! | ✅ Ja | Tagkleur in hex-formaat (bijv. "#ff0000") |  |

### EditTagInput

| Parameter | Type | Vereist | Beschrijving |
|-----------|------|---------|--------------|
| `id` | String! | ✅ Ja | ID van de tag om te updaten |  |
| `title` | String | Nee | Nieuwe tagtitel |  |
| `color` | String | Nee | Nieuwe tagkleur in hex-formaat |  |

### SetTodoTagsInput

| Parameter | Type | Vereist | Beschrijving |
|-----------|------|---------|--------------|
| `todoId` | String! | ✅ Ja | ID van het record om te taggen |  |
| `tagIds` | [String!] | Nee | ID's van bestaande tags om toe te passen |  |
| `tagTitles` | [String!] | Nee | Titels van nieuwe tags om te maken en toe te passen |  |

### TagOrderByInput Waarden

| Waarde | Beschrijving |
|--------|--------------|
| `id_ASC` / `id_DESC` | Sorteer op tag-ID |  |
| `uid_ASC` / `uid_DESC` | Sorteer op unieke identificatie |  |
| `title_ASC` / `title_DESC` | Sorteer alfabetisch op titel |  |
| `color_ASC` / `color_DESC` | Sorteer op kleurwaarde |  |
| `createdAt_ASC` / `createdAt_DESC` | Sorteer op aanmaakdatum |  |
| `updatedAt_ASC` / `updatedAt_DESC` | Sorteer op laatste update datum |  |

## Antwoordvelden

### Tag Type

| Veld | Type | Beschrijving |
|------|------|--------------|
| `id` | ID! | Unieke identificatie voor de tag |  |
| `uid` | String! | Gebruiksvriendelijke unieke identificatie |  |
| `title` | String! | Tagtitel/naam |  |
| `color` | String! | Tagkleur in hex-formaat |  |
| `project` | Project! | Project waartoe deze tag behoort |  |
| `todos` | [Todo!]! | Records die aan deze tag zijn gekoppeld |  |
| `createdAt` | DateTime! | Wanneer de tag is aangemaakt |  |
| `updatedAt` | DateTime! | Wanneer de tag voor het laatst is bijgewerkt |  |

### TagPagination Antwoord

| Veld | Type | Beschrijving |
|------|------|--------------|
| `items` | [Tag!]! | Array van tagrecords |  |
| `pageInfo` | PageInfo! | Pagineringsinformatie |  |
| `totalCount` | Int! | Totaal aantal tags dat overeenkomt met filters |  |

### PageInfo Velden

| Veld | Type | Beschrijving |
|------|------|--------------|
| `totalPages` | Int | Totaal aantal pagina's |  |
| `totalItems` | Int | Totaal aantal items over alle pagina's |  |
| `page` | Int | Huidig paginanummer |  |
| `perPage` | Int | Aantal items per pagina |  |
| `hasNextPage` | Boolean! | Of er een volgende pagina is |  |
| `hasPreviousPage` | Boolean! | Of er een vorige pagina is |  |

## Vereiste Machtigingen

### Tagbeheer

| Actie | Vereiste Rol |
|-------|--------------|
| Create tags | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |
| Edit tags | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |
| Delete tags | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |
| Apply tags to records | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |

### Tagzichtbaarheid

Tags zijn zichtbaar voor alle gebruikers die toegang hebben tot het project waar de tag is aangemaakt. Rolgebaseerde filtering kan van toepassing zijn voor bepaalde bewerkingen.

## Beste Praktijken

1. **Gebruik Consistente Kleuren**: Stel een kleurcoderingssysteem voor je team in
2. **Beschrijvende Titels**: Gebruik duidelijke, beschrijvende tagnamen die teamleden zullen begrijpen
3. **Batchbewerkingen**: Gebruik `setTodoTags` om meerdere tags tegelijk toe te passen
4. **Zoeken en Filteren**: Gebruik de zoekfunctionaliteit om bestaande tags te vinden voordat je nieuwe maakt
5. **Tagbeheer**: Beoordeel en ruim regelmatig ongebruikte tags op
6. **Kleurconventies**: Overweeg om standaardkleuren te gebruiken voor veelvoorkomende tagtypes (rood voor urgent, groen voor voltooid, enz.)

## Geavanceerde Functies

### Standaarden voor Tagcreatie

- Nieuwe tags die worden aangemaakt via `setTodoTags` met `tagTitles` standaard op kleur `#4a9fff`
- Tags zonder gespecificeerde kleuren vallen terug op `#00a0d2`
- Tagtitels worden automatisch ontdaan van spaties

### Automatiseringsintegratie

Tags kunnen zijn:
- Automatisch toegepast door automatiseringsregels
- Gebruikt als triggers voor automatiseringsworkflows
- Gewijzigd via automatiseringsacties

### Real-time Updates

Tagwijzigingen worden uitgezonden via GraphQL-abonnementen, waardoor real-time updates mogelijk zijn in verbonden clients.

## Belangrijke Notities

- Het verwijderen van een tag verwijdert deze uit alle bijbehorende records
- Tags zijn projectgebonden en kunnen niet tussen projecten worden gedeeld
- Kleurwaarden moeten geldige hex-codes zijn (bijv. "#ff0000", "#00aa00")
- Tagtitels zijn hoofdlettergevoelig
- Maximaal 100 tags kunnen op een enkel record worden toegepast
- Tags ondersteunen full-text zoekopdrachten over titels
