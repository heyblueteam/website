---
title: Een Bedrijf Aanmaken
description: Hoe een nieuw bedrijf aan te maken met de Blue API.
---

Om een nieuw bedrijf aan te maken, kunt u de volgende mutatie gebruiken:

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

Houd er rekening mee dat het aanmaken van nieuwe bedrijven invloed kan hebben op de facturering van uw Blue-account, aangezien elk bedrijf afzonderlijk wordt gefactureerd.

Hier zijn de opties voor de `createCompany` mutatie en hun beschrijvingen:

| Optie | Type | Beschrijving |
|-------|------|--------------|
| name | String | De naam van het nieuwe bedrijf. Dit is een verplicht veld. |  |
| slug | String | Een URL-vriendelijke versie van de bedrijfsnaam. Dit wordt gebruikt in de URL van het bedrijf en moet uniek zijn. Dit is een verplicht veld en zal ook fungeren als de unieke identificator voor het bedrijf. |
