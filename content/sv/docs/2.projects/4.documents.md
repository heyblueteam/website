Dokument fungerar som samarbetsdokument med rik text där team kan skissera idéer, skapa specifikationer, hantera mötesanteckningar, bygga interna wikis och mer.

Med realtidsredigeringsmöjligheter, enkla formateringsverktyg och tät integration med andra Blue-funktioner möjliggör dokument intuitiv innehållsskapande och oavbrutet teamarbete.

## Skapa dokument

För att skapa ett nytt dokument:

1. Klicka på fliken "Dokument"
2. Tryck på knappen "+ Skapa dokument"
3. Ange en titel för ditt dokument
4. Tryck enter för att generera ett tomt dokument

![Document 2](/docs/Document_2.png "Document 2")

Skapa ett nytt dokument i Blue

## Redigera dokument

Textredigeraren med rik text möjliggör enkel formatering och innehållsskapande.

Du kan tillämpa formatering som fet och kursiv stil genom tangentbordsgenvägar med hjälp av /snedstreck-kommandot - `/bold`, `/italic`

![Document 3](/docs/Document_3.png "Document 3")

Skriv "/" för att öppna kommandon i textredigeraren

Som överallt i Blue där du kan lämna kommentarer kan du @omnämna teammedlemmar för att meddela dem eller hoppa till deras markörposition.

![Document 4](/docs/Document_4.png "Document 4")

Omnämn enkelt andra användare i dokument och se vem som för närvarande tittar på dokumentet.
Du kan använda "@@" för att omnämna en specifik post, och sedan kan användare klicka på den länken för att öppna posten inom dokumentvyn. Detta är extremt användbart eftersom det undviker kontextväxling för alla som läser ditt dokument.

![Document 5](/docs/Document_5.png "Document 5")

Klicka på vilken omnämnd post som helst för att öppna den direkt

- Bädda in bilder via dra-och-släpp eller kommandot `/attachment`
- Rubriker, checklistor, markeringar och mer

## Samarbetsredigering

Dokument underlättar sömlös samredigering och teamarbete:
- Se uppdateringar från andra redigerare i realtid
- Klicka på användarprofilikoner i högra sidofältet för att direkt hoppa till deras position i dokumentet
- Förhandsgranska redigeringar live medan de skrivs
- Ändringar sparas automatiskt - ingen manuell sparning krävs

## Användningsområden

### Planeringsdokument

Dokumentredigeraren är idealisk för att skapa strategiska planer som är anpassade till projektmål och milstolpar. Dokument kan beskriva omfattning, krav, resurser, budgetar, tidslinjer och mer. Samarbete i realtid gör det möjligt för planerare att samförfatta samtidigt som versionshistorik bevaras.

### Designspecifikationer

För kreativa projekt gör dokument det möjligt för team att enas om detaljerade specifikationer, tillgångskrav, innehållsplaner, stilguider och andra designelement. Dokument centraliserar denna information för enkel referens, medan ändringsspårning fångar iterativ utveckling.

### Kunduppdrag

Kundbehov och förväntningar kan konsolideras i delbara dokument. Uppdragsbeskrivningar klargör projektparametrar, framgångsmått, målgrupper, budskap och erforderliga leveranser. Dokument håller intressenter samordnade samtidigt som de behåller ett tillgängligt revisionsspår av överenskomna mål.

### Interna wikisidor

Wikis och intern dokumentation underlättar tillgång till institutionell kunskap. Dokument minskar dubbelarbete genom att centralisera processdokumentation, utbildningsmaterial, policyer, instruktioner och referensinformation.

### Mötesagendor och anteckningar

Förberedelsematerial och protokoll från diskussioner hjälper till att hålla möten strukturerade och resultat genomförbara. Agendor beskriver diskussionspunkter, medan anteckningar fångar närvaro, beslut och nästa steg i ett lättillgängligt system för dokumentation.
