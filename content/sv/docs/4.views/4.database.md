## Interaktiva Kolumner

Varje kolumn representerar ett anpassat fält, med möjligheten att omedelbart redigera data inom celler. Omorganisera kolumnplacering via dra och släpp.

Notera att för att ändra kolumnnamnen måste du ändra namnen på de relaterade anpassade fälten. Standardfält kan inte döpas om.

## Oändlig Rullning

- Databastabellen möjliggör oändlig vertikal rullning för att hantera enorma datamängder som sträcker sig över tiotusentals eller hundratusentals poster.

## Enkelt Åtkomst till Postdetaljer

- Högerklicka på vilken post som helst för att öppna en detaljpanel som visar fullständig postinformation. Detta möjliggör nedgrävning utan att behöva öppna hela posten.

## Massåtgärder

Om det är ett nummer/formelfält kan du omedelbart se den totala summan.

## Kolumnhantering

Du kan klicka på en kolumnrubrik för att sortera alla dina poster efter den kolumnen:

![Database 5](/docs/Database_5.gif "Database 5")

Sortera efter vilken kolumn som helst, och till och med flera kolumner!

Du kan även klicka på inställningsikonen på varje kolumnrubrik för att få olika alternativ presenterade för dig:

![Database 6](/docs/Database_6.png "Database 6")

- **Fäst Kolumn**: Du kan välja att fästa kolumnen antingen till höger eller vänster. Detta håller kolumnen alltid synlig när du rullar horisontellt.
- **Autostorlek för Denna Kolumn**: Säkerställer att kolumnbredden är tillräcklig för att visa all data i cellerna som finns i den kolumnen
- **Autostorlek för Alla Kolumner**: Samma som "*Autostorlek för Denna Kolumn*" men gäller för alla kolumner
- **Gruppera efter Tilldelade**: Gruppera alla poster efter denna kolumn, [som diskuterades tidigare i denna guide.](/docs/views/database#grouping)
