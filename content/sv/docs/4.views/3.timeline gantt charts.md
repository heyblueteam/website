Denna Gantt-diagramliknande vy gör det enkelt att spåra scheman, deadlines och tidsmässiga relationer mellan dina poster.

![Timeline 1](/docs/Timeline_1.png "Timeline 1")

Tidslinjevyn används vanligtvis för:

- Visualisera scheman, milstolpar och deadlines för projekt
- Spåra framsteg för uppgifter och leveranser
- Identifiera beroenden och schemaläggningskonflikter mellan poster
- Planera resursallokering baserat på uppgiftstidsramar
- Analysera flöde och varaktighet för viktiga arbetsflöden och processer


## Dra och släpp-schemaläggning
Schemalägg enkelt om poster genom att dra dem till olika datum/tidsluckor på tidslinjen. Förläng eller förkorta varaktigheten genom att dra i kanterna.

![Timeline 2](/docs/Timeline_2.gif "Timeline 2")

## Oändlig rullning

Tidslinjen sträcker sig oändligt, vilket gör att du kan schemalägga uppgifter och deadlines så långt in i framtiden som behövs. Använd zoomkontrollerna för att växla mellan högre nivå och mer detaljerade vyer.

## Filtrering

Alla Blues sofistikerade filtreringsmöjligheter är tillgängliga, vilket gör att du kan fokusera på särskilda listor, ansvariga, taggar, tidsintervall, etc. Filterinställningar sparas när du byter vyer.

## Idag-knapp

Hoppa omedelbart till dagens datum på tidslinjen genom att klicka på Idag-knappen. Användbart för att orientera dig och fokusera på förestående deadlines.

## Beroenden

Tidslinjevyn visar visuellt [Beroenden](/docs/records/dependencies) som är inställda mellan poster (som visas i skärmdumpen ovan). Detta gör att du kan se hur uppgifter relaterar och kan påverka andra objekt.

För att ställa in ett beroende:

1. Gå till posten som ska slutföras först.
2. Klicka på Beroenden under Taggar-sektionen.
3. Välj "Blockerande" för att få denna post att blockera en annan.
4. Välj posten du vill att denna ska blockera från rullgardinsmenyn.
5. Klicka på Lägg till.
