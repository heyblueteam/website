Enkelradstextfältet gör det möjligt för dig att lägga till oformaterade textsträngar till dina poster i Blue. Det finns ingen teckengräns, så du kan skriva text av vilken längd som helst i detta fält. Texten är dock begränsad till en enda rad och stöder inte formateringsalternativ som fetstil, kursivstil eller länkar.

![Single Line Text](/docs/SingleLineText.png "Single Line Text")

Ett flexibelt sätt att fånga strukturerad textdata

Textfältet är mycket flexibelt och tillåter inmatning av både bokstäver och siffror, inklusive specialtecken. Det finns inga begränsningar för vilka tecken som kan matas in. Systemet tillämpar ingen validering eller maskering på den text som användaren matar in.

Enkelradstextfältet är utmärkt för att fånga alla typer av information, inklusive namn, beskrivningar, modellnummer, SKU:er, adresser, anteckningar, ID:n och mer. Du kan använda detta flexibla fält för att lagra namn på kunder, leverantörer eller kontakter, ange lagerkoder och identifierare, göra allmänna korta anteckningar om poster, eller lagra unika ID:n för databasrelationer och uppslagningar. Enkelradstextfältet gör det möjligt för dig att effektivt mata in viktig textbaserad information relaterad till dina poster i Blue.

Du kan dra nytta av Blues [avancerade postfiltreringsmöjligheter](/sv/docs/views/filters) för att hitta och filtrera poster som innehåller specifik text inom Enkelradstextfältet för att lokalisera matchande poster.

Med sin enkla enkelradsdesign, obegränsade textkapacitet och integration med filtrering levererar Enkelradstextfältet en optimerad upplevelse för att fånga ostrukturerad data. Det ger flexibiliteten att lagra all information som behövs på en per-post-basis.
