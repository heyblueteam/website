Dashboards i Blue är kraftfulla eftersom de låter dig kombinera realtidsdata från flera projekt samtidigt. Detta gör att du kan bygga live dashboards som ger dig meningsfull information för beslutsfattande.

## Diagramalternativ

Blue kommer med tre alternativ för diagram:

1. [Statistikkort:](/docs/dashboards/stat-cards) Idealiskt för att snabbt se nyckeltal, statistikkort visar realtidsdata som total projektintäkt, aktuella aktiva problem eller den övergripande projektfullföljande procenten. De ger omedelbar insikt i kritiska data med ett enkelt, tydligt gränssnitt.
2. [Cirkeldiagram:](/docs/dashboards/pie-charts) Bäst för att visualisera datakomposition, cirkeldiagram i Blue hjälper dig att förstå proportionen av olika element inom en datamängd. Till exempel kan du använda ett cirkeldiagram för att visa fördelningen av uppgifter över olika avdelningar eller procentfördelningen av intäkter från olika typer av kunder.
3. [Stapeldiagram:](/docs/dashboards/bar-graphs) Effektiva för att jämföra kvantiteter, stapeldiagram låter dig kontrastera olika grupper sida vid sida. Använd stapeldiagram för att jämföra försäljningssiffror över olika kvartal, antalet uppgifter som varje teammedlem slutför, eller frekvensen av vissa kundförfrågningar över tid.

## Dela
Dashboards i Blue är inte bara för personlig visning; de kan delas med teammedlemmar inom plattformen. Du kan ge redigerings- eller visningsbehörigheter. Alla som har tillgång till en dashboard kan dela den med andra användare.

::callout
 ---

icon: lightbulb
---

Delningsprivilegier är indelade enligt användarbehörigheter; till exempel kan användare med visningsbehörigheter bjuda in andra att se dashboarden, vilket säkerställer att åtkomsträttigheter upprätthålls och respekteras.
::

För närvarande är Blues dashboards utformade för intern samarbete och kan inte delas externt med icke-Blue-användare via offentliga länkar. Detta är dock planerat för en framtida release. Om du har fler idéer för dashboards kan du alltid [gå med i vårt communityforum](https://ask.blue.cc/) eller kontakta oss direkt på [support@blue.cc](support@blue.cc)

## Filter

Blue dashboards erbjuder sofistikerade filtreringsmöjligheter, vilket möjliggör exakt datasegmentering efter projekt, lista, tagg och tidsram. Listor och taggar som delar samma namn konsolideras smart över projekt, vilket förenklar processen att filtrera data som sträcker sig över flera initiativ. Denna intelligenta design möjliggör sömlös analys över projekt, vilket säkerställer omfattande och relevanta insikter.

## Interaktiv och Anpassningsbar Gränssnitt

- **Dra och Släpp Funktionalitet**: Att anpassa din dashboard-layout är intuitivt med Blues dra och släpp-funktion. Du kan snabbt flytta diagram inom en dashboard för att passa dina visningspreferenser eller för att framhäva den mest kritiska informationen.
- **Visningsflexibilitet**: Blue förstår att datavisualisering inte är en storlek som passar alla. Därför erbjuder det olika visningsalternativ för alla diagram och datadarställningar på dashboards. Oavsett om du behöver ett cirkeldiagram för att bryta ner uppgiftsfördelningen eller ett linjediagram för att spåra framsteg över tid, har du flexibiliteten att välja det format som bäst förmedlar dina data.

## Ta bort Dashboards

För att ta bort en dashboard, högerklicka helt enkelt på dashboarden i listan och tryck på ta bort. Observera att du måste vara dashboardens ägare — den person som skapade dashboarden — för att kunna ta bort den.
