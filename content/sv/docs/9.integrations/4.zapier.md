Zapier är en integrationsplattform som hjälper till att koppla samman olika webbappar och automatisera arbetsflöden.

**Användningsfall**

Här är några av de mest populära sätten som Blue integreras med andra appar via Zapier:

**_Synkronisera Blue-poster till Google Sheets_**

När Blue-poster skapas eller uppdateras, läggs de automatiskt till som nya rader i ett Google Sheet för centraliserad rapportering.

**_Skicka orderbekräftelse från Blue-poster_**

När du väljer ett visst anpassat fält på Blue, skicka automatiskt ett e-postmeddelande för att bekräfta beställningen med din kund.

**_Lägg till Blue-kommentarer i Slack_**

Få realtidsnotifikationer från Blue publicerade i rätt Slack-kanaler så att ditt team är medvetet om viktiga uppdateringar av poster.

## Nyckelfördelar

**Automatisering mellan verktyg**: Zaps som skapats i Zapier kan automatiskt skicka och ta emot data mellan Blue och tusentals andra appar. Detta eliminerar manuellt arbete med att flytta data mellan system.

**Realtidsintegrationer**: Genom att använda Blues webhook-utlösare reagerar Zaps omedelbart på händelser i Blue som en ny post, tillagd kommentar, statusändring etc. Detta möjliggör realtids-synkronisering och processautomatisering.

**Enkelt att bygga och underhålla**: Zaps erbjuder ett användarvänligt, kodfritt sätt att integrera verktyg. Ingen utvecklingsarbete behövs. Och de fortsätter att fungera utan pågående underhåll.

1. Logga in på ditt Blue-konto under profilinställningar
2. Gå till profilinställningarna och klicka på "Webhooks"
3. Klicka på "Lägg till Webhook"
4. Ge webhooken ett namn och ange Zapier webhook-URL
5. Välj vilka händelser som ska utlösa webhooken
6. Klicka på "Skapa Webhook"

## Blue till Zapier

1. Logga in på ditt Zapier-konto
2. Klicka på Skapa en Zap  
3. För det första steget, sök efter och välj ["Webhooks by Zapier" trigger-app](https://zapier.com/apps/webhook)
4. Konfigurera den för att ta emot payloads från din Blue webhook-URL
5. För det andra steget, sök och välj destinationsappen för att skicka data till
6. Mappa JSON-payload-fälten till inmatningsfälten i destinationsappen
7. Slå på Zap

Nu när de angivna händelserna inträffar i Blue, kommer webhook-payloaden att skickas till Zapier som automatiskt vidarebefordrar datan till din integrerade app enligt Zap-inställningarna.

## Zapier till Blue

För att skicka data från andra appar till Blue, följ dessa steg:

**1. Skaffa Blue API-uppgifter**
- Gå till Blue-inställningar > API-token
- Skapa en ny token eller använd befintliga uppgifter:
  - Token-ID
  - Token-hemlighet
  - Företags-ID
  - Projekt-ID

**2. Skapa Zap i Zapier**
1. Ställ in din trigger-app (t.ex. Google Sheets, Slack, etc.)
2. Lägg till "Webhooks by Zapier" som Action-app
3. Välj "Anpassad begäran" händelsetyp

**3. Konfigurera Webhook**
- Metod: POST
- URL: `https://api.blue.cc/graphql` (eller beta-slutpunkt om det behövs)
- Data Pass-Through: FALSE
- Unflatten: NO

**4. Ställ in Headers:**
```json
{
  "Content-Type": "application/json",
  "X-Bloo-Token-ID": "Your_Token_ID",
  "X-Bloo-Token-Secret": "Your_Token_Secret", 
  "X-Bloo-Company-ID": "Your_Company_ID",
  "X-Bloo-Project-ID": "Your_Project_ID"
}
```

**5. Konfigurera Fråga (Exempel på Mutation):**
```json
{
  "query": "mutation CreateTodo($input: CreateTodoInput!) {\n  createTodo(input: $input) {\n    id\n    title\n    customFields {\n      value\n    }\n  }\n}",
  "variables": {
    "input": {
      "todoListId": "your-todo-list-id",
      "title": "New task from Zapier",
      "customFields": [
        {
          "customFieldId": "priority-field-id",
          "value": "High"
        },
        {
          "customFieldId": "due-date-field-id",
          "value": "2024-03-01"
        }
      ]
    }
  }
}
```

**6. Testa & Aktivera**
- Använd Blues API Playground för att testa mutationer först
- Mappa dynamiska värden från trigger-appar med hjälp av Zapier's datafält
- Aktivera Zap efter framgångsrik testning
