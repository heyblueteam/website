## Exportera data

1. I projektet du vill exportera, klicka på inställningsikonen (växel)
2. Välj "Exportera"
3. Klicka på "Exportera" för att starta exportprocessen på Blue-servrarna.
4. När det är klart, kommer du att få ett e-postmeddelande med en länk för att ladda ner den exporterade CSV-filen

::callout
 ---

icon: lightbulb
---

**Hur lång tid tar det att få e-postmeddelandet med datan?** <br/> <br/>
Vanligtvis kommer detta att anlända inom några sekunder, men det kan bero på hur många andra exportprocesser som körs från andra kunder och storleken på dina data. För dataset som har över 100 000 objekt kan detta ta några minuter.
::

::

## Exportera med aktiva filter

Om du redan har tillämpat filter på ditt projekt kan du enkelt exportera endast de filtrerade posterna genom att använda alternativet "Exportera med aktiva filter". På så sätt behöver du inte oroa dig för att manuellt filtrera den exporterade datan, eftersom den automatiskt kommer att inkludera endast de poster som matchar dina aktuella filterinställningar.

Så här kan du exportera med aktiva filter:

1. Tillämpa filtren på ditt projekt genom att använda de filteralternativ som finns tillgängliga, såsom tilldelningar, taggar eller avancerade filter.
2. När du har den filtrerade vy du vill exportera, gå till projektinställningen och klicka på Importera & Exportera.
3. I exportalternativens meny, välj "Exportera med aktiva filter."
4. När din export är klar, kommer du att få datan i ditt e-postmeddelande.

## Nyckelfördelar

- **Dataportabilitet**: Ladda ner projekt utanför Blue när det behövs för säkerhetskopiering eller extern analys
- [Utlös via API](/docs/integrations/api): Du kan använda tredjepartssystem för att automatiskt utlösa Blue-exporter regelbundet för att uppfylla dina krav på databas.
- **Tillgänglighet för intressenter**: Dela exporterade CSV-filer enkelt med kunder, chefer och andra utanför Blue.

Exporten ger datatillgång och flexibilitet för att matcha olika potentiella integrations- och rapporteringsbehov. Tillsammans med [import](/docs/integrations/csv-import) ger det fullständig kontroll över dina Blue-data.
