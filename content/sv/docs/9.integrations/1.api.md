Blue har ett API (Application Programming Interface), så att du kan bygga dina egna integrationer om det behövs.

**Blue har 100% API-täckning**. Detta betyder att allt som du kan göra i [någon av våra appar](/docs/start-guide/download-apps), kan du göra programmässigt. Vi har ett kraftfullt [GraphQL](https://graphql.org/) API som har en fullständig lekplats där du kan testa dina frågor och som är fullt dokumenterat.
