[Make.com](https://www.make.com) är en integrationsplattform som hjälper till att koppla samman olika webbappar och automatisera arbetsflöden. Blue är **officiellt integrerat** med Make.com, vilket möjliggör sömlös automation över 2 000 appar—ingen kodning krävs!

![Blue Make.com Moduler Översikt](/docs/blue_make_modules_overview.png "Overview of available Blue modules in Make.com")

## Användningsfall

Här är några kraftfulla sätt att utöka Blues kapabiliteter med Make.com-integrationer:
Blues Make.com-integration möjliggör kraftfull automation över olika affärsfunktioner. För kundupplevelse kan du koppla Blue med verktyg som Intercom, HubSpot och Zendesk för att skapa omfattande kundregister, synkronisera interaktioner och upprätthålla realtidskommunikation genom Slack och Microsoft Teams. Inom projektledning integrerar Blue med Jira, GitHub, Trello och Monday.com för att synkronisera uppgifter, spåra framsteg och automatisera meddelanden.

För marknadsföring och försäljning kopplar Blue samman med Mailchimp, Stripe och Calendly för att spåra kampanjer, hantera betalningar och automatisera mötesarbetsflöden. Dokumenthantering effektiviseras genom integrationer med Google Drive, Dropbox, OneDrive och SharePoint, medan kommunikationsverktyg som Slack, Gmail och WhatsApp förenas genom Blues centrala uppgiftshanteringssystem. Dataanalysmöjligheter förbättras genom kopplingar med Google Sheets, Power BI, Airtable och Tableau, och branschspecifika lösningar finns tillgängliga för finansiella tjänster, e-handel och fjärrteamhantering.

## Anslut och autentisera ditt Blue-konto

För att säkert koppla Blue med Make.com:

1. Logga in på ditt Blue-konto och öppna din profil → gå till fliken **API**
2. Klicka på **Generera en token**, ange ett namn (och valfri utgångstid)
3. Kopiera båda:
   - **Token-ID** (`x‑bloo‑token‑id`)
   - **Hemlig nyckel** (`x‑bloo‑token‑secret`)

::callout
 ---
icon: lightbulb
---
**Viktig Notis** <br/>
Den hemliga nyckeln visas endast en gång—lagra den säkert!
::

5. I Make.com, när du skapar en ny anslutning, ange:
   - **Token-ID**
   - **Hemlig nyckel**

## Modulparameterreferens

### Övervaka Händelser
- Händelser (valfritt): Multi-välj händelsetyper
- Företags-ID (valfritt): Multi-välj företags-ID eller slugs
- Projekt-ID (valfritt): Multi-välj projekt-ID eller slugs

### Skapa en Post
- Företags-ID: Företags-ID eller slug
- Projekt-ID: Projekt-ID eller slug
- Lista: Lista-ID
- Postnamn: Namn på posten
- Standardfält: Standardvärde
- Enskilt Datum: Förfallodatum
- Datumintervall: Start- och förfallodatum
- Tilldelade: Multi-välj användare
- Taggar: Multi-välj taggar
- Beskrivning: Textfält, stöder HTML
- Checklistor: Array av checklisttitlar

**Stödda Anpassade Fält:**
| Fälttyp | Beskrivning |
|---------|-------------|
| Enkelradstext | En rad textfält med valfri validering och teckenbegränsning |
| Flera rader text | Flera rader textfält med stöd för rik text och formateringsalternativ |
| Enkelval | Dropdown med enstaka alternativval och anpassade alternativ |
| Flera val | Dropdown med flera alternativval och anpassade alternativ |
| Plats | Bredd- och längdgrad med kartvy och adressökning |
| Kryssruta | Sann/falsk växlare med valfritt standardvärde och anpassade etiketter |
| Land | Tvåbokstavskod för land med landsnamn |
| Valuta | Numeriskt fält med valutasymbol och stöd för växelkurs (t.ex. USD200, 200USD, KHR4000) |
| Datum | Datumväljare med valfri tidskomponent och tidszonstöd |
| Telefon | Telefonnummerfält som stöder E.164, internationellt format eller landskod + nummer med validering |
| E-post | E-postfält med formatvalidering och domänverifiering |
| Stjärnbetyg | Numeriskt betyg från 1-5 med visuella stjärnor och anpassade etiketter |
| Procent | Numeriskt fält med procenttecken (0-100) och decimalprecision |
| URL | Webbadressfält med formatvalidering och förhandsgranskning |
| Nummer | Numeriskt fält med valfria decimaler och enhetsvisning |
| Formel | Beräknat fält baserat på andra fält med matematiska operationer |
| Unik ID | Automatgenererad identifierare med anpassad prefix och format |
| Referens | Länk till en annan post med sök- och filtreringsalternativ |
| Uppsökning | Dynamiskt fält som hämtar data från refererade poster med mappingsalternativ |
| Tidsduration | Tidsintervall i timmar och minuter med anpassat visningsformat |

- Standardfält: Standardvärde
- Enskilt Datum: Förfallodatum
- Datumintervall: Start- och förfallodatum
- Tilldelade: Multi-välj användare
- Taggar: Multi-välj taggar
- Beskrivning: Textfält, stöder HTML
- Checklistor: Array av checklisttitlar

### Sök Poster
- Företags-ID: Företags-ID eller slug
- Projekt-ID: Projekt-ID eller slug
- Lista: Lista-ID
- Sökning/Fråga: Söktext med avancerade filtreringsalternativ

### Hämta en Post
- Företags-ID: Företags-ID eller slug
- Projekt-ID: Projekt-ID eller slug
- Lista: Lista-ID
- Post-ID: ID för posten som ska hämtas med all associerad data

### Bjud in Användare
- Företags-ID: Företags-ID eller slug
- Projekt-ID: Projekt-ID eller slug
- E-postadress: Användarens e-postadress med validering
- Roll-ID: Roll-ID (standard eller anpassad) med behörigheter

### Lista Anpassade Fält
- Företags-ID: Företags-ID eller slug
- Projekt-ID: Projekt-ID eller slug
- Gräns: Maximalt antal fält att returnera med paginering

### Lista Listor
- Företags-ID: Företags-ID eller slug
- Projekt-ID: Projekt-ID eller slug
- Gräns: Maximalt antal listor att returnera med sorteringsalternativ

### Lista Alla Projekt
- Företags-ID: Multi-välj företags-ID eller slugs
- Gräns: Maximalt antal projekt att returnera med filtrering

### Lista Taggar
- Företags-ID: Företags-ID eller slug
- Projekt-ID: Projekt-ID eller slug
- Gräns: Maximalt antal taggar att returnera med användningsstatistik

### Utför GraphQL
- Rubriker: Array av nyckel-värde-par för autentisering och anpassade rubriker
- Metod: GET eller POST med begärningsalternativ
- Fråga: GraphQL-fråga eller mutation med variabelstöd
- Operationsnamn: Namn på operationen för spårning
- Variabler Datakälla: Rå, Form eller Samling med typkontroll
- Variabler: Innehåll av GraphQL-variabler med validering
