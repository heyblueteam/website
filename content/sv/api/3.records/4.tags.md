## Översikt

Taggar är ett kraftfullt sätt att kategorisera och organisera poster i Blue. Taggsystemet erbjuder omfattande funktionalitet för att skapa, hantera och filtrera taggar, med stöd för massoperationer och avancerad filtrering.

## Lista Taggar

Hämta taggar från projekt med flexibla filtrerings- och sorteringsalternativ.

### Grundläggande Exempel

```graphql
query ListTags {
  tagList(
    filter: { 
      projectIds: ["project-123"] 
      excludeArchivedProjects: false 
    }
    first: 50
    orderBy: title_ASC
  ) {
    items {
      id
      uid
      title
      color
      project {
        id
        name
      }
      createdAt
      updatedAt
    }
    pageInfo {
      totalPages
      totalItems
      page
      perPage
      hasNextPage
      hasPreviousPage
    }
    totalCount
  }
}
```

### Avancerat Filtreringsexempel

```graphql
query FilterTags {
  tagList(
    filter: {
      projectIds: ["project-123", "project-456"]
      search: "urgent"
      colors: ["#ff0000", "#ffaa00"]
      excludeArchivedProjects: true
    }
    orderBy: createdAt_DESC
    first: 20
  ) {
    items {
      id
      title
      color
      todos {
        id
        title
      }
    }
    totalCount
  }
}
```

## Skapa Taggar

Skapa nya taggar inom ett projekt.

```graphql
mutation CreateTag {
  createTag(
    input: {
      title: "High Priority"
      color: "#ff0000"
    }
  ) {
    id
    uid
    title
    color
    project {
      id
      name
    }
    createdAt
  }
}
```

## Uppdatera Taggar

Ändra egenskaper för befintliga taggar.

```graphql
mutation UpdateTag {
  editTag(
    input: {
      id: "tag-123"
      title: "Critical Priority"
      color: "#cc0000"
    }
  ) {
    id
    title
    color
    updatedAt
  }
}
```

## Ta Bort Taggar

Ta bort taggar från systemet. Detta kommer också att ta bort taggen från alla associerade poster.

```graphql
mutation DeleteTag {
  deleteTag(id: "tag-123")
}
```

## Tagga Poster

Associera taggar med poster (att göra) med hjälp av befintliga taggar eller skapa nya.

### Använda Befintliga Taggar

```graphql
mutation TagRecord {
  setTodoTags(
    input: {
      todoId: "todo-123"
      tagIds: ["tag-456", "tag-789"]
    }
  )
}
```

### Skapa Taggar Vid Taggning

Du kan skapa nya taggar i farten genom att ange taggnamn:

```graphql
mutation TagRecordWithNewTags {
  setTodoTags(
    input: {
      todoId: "todo-123"
      tagIds: ["existing-tag-id"]
      tagTitles: ["New Tag", "Another New Tag"]
    }
  )
}
```

## AI Taggförslag

Få AI-drivna taggförslag för en post:

```graphql
mutation GetTagSuggestions {
  aiTag(
    input: {
      todoId: "todo-123"
      suggestionsCount: 5
    }
  ) {
    suggestions {
      title
      confidence
      color
    }
  }
}
```

## Inmatningsparametrar

### TagListFilter

| Parameter | Typ | Obligatorisk | Beskrivning |
|-----------|------|--------------|-------------|
| `projectIds` | [String!] | Nej | Filtrera taggar efter projekt-ID eller slugs |  |
| `excludeArchivedProjects` | Boolean | Nej | Exkludera taggar från arkiverade projekt |  |
| `search` | String | Nej | Sök taggnamn (skiftlägeskänslig) |  |
| `titles` | [String!] | Nej | Filtrera efter specifika taggnamn |  |
| `colors` | [String!] | Nej | Filtrera efter specifika färger (hex-format) |  |
| `tagIds` | [String!] | Nej | Filtrera efter specifika tagg-ID |  |

### CreateTagInput

| Parameter | Typ | Obligatorisk | Beskrivning |
|-----------|------|--------------|-------------|
| `title` | String | Nej | Tagg titel (automatiskt genererad om inte angiven) |  |
| `color` | String! | ✅ Ja | Tagg färg i hex-format (t.ex. "#ff0000") |  |

### EditTagInput

| Parameter | Typ | Obligatorisk | Beskrivning |
|-----------|------|--------------|-------------|
| `id` | String! | ✅ Ja | ID för taggen som ska uppdateras |  |
| `title` | String | Nej | Ny tagg titel |  |
| `color` | String | Nej | Ny tagg färg i hex-format |  |

### SetTodoTagsInput

| Parameter | Typ | Obligatorisk | Beskrivning |
|-----------|------|--------------|-------------|
| `todoId` | String! | ✅ Ja | ID för posten som ska taggas |  |
| `tagIds` | [String!] | Nej | ID för befintliga taggar som ska tillämpas |  |
| `tagTitles` | [String!] | Nej | Titlar på nya taggar som ska skapas och tillämpas |  |

### TagOrderByInput Värden

| Värde | Beskrivning |
|-------|-------------|
| `id_ASC` / `id_DESC` | Sortera efter tagg-ID |  |
| `uid_ASC` / `uid_DESC` | Sortera efter unikt identifierare |  |
| `title_ASC` / `title_DESC` | Sortera alfabetiskt efter titel |  |
| `color_ASC` / `color_DESC` | Sortera efter färgvärde |  |
| `createdAt_ASC` / `createdAt_DESC` | Sortera efter skapelsedatum |  |
| `updatedAt_ASC` / `updatedAt_DESC` | Sortera efter senaste uppdateringsdatum |  |

## Svarsfält

### Taggtyp

| Fält | Typ | Beskrivning |
|-------|------|-------------|
| `id` | ID! | Unik identifierare för taggen |  |
| `uid` | String! | Användarvänlig unik identifierare |  |
| `title` | String! | Tagg titel/namn |  |
| `color` | String! | Tagg färg i hex-format |  |
| `project` | Project! | Projekt som denna tagg tillhör |  |
| `todos` | [Todo!]! | Poster associerade med denna tagg |  |
| `createdAt` | DateTime! | När taggen skapades |  |
| `updatedAt` | DateTime! | När taggen senast uppdaterades |  |

### TagPagination Svar

| Fält | Typ | Beskrivning |
|-------|------|-------------|
| `items` | [Tag!]! | Array av taggposter |  |
| `pageInfo` | PageInfo! | Pagineringinformation |  |
| `totalCount` | Int! | Totalt antal taggar som matchar filtren |  |

### PageInfo Fält

| Fält | Typ | Beskrivning |
|-------|------|-------------|
| `totalPages` | Int | Totalt antal sidor |  |
| `totalItems` | Int | Totalt antal objekt över alla sidor |  |
| `page` | Int | Aktuellt sidnummer |  |
| `perPage` | Int | Antal objekt per sida |  |
| `hasNextPage` | Boolean! | Om det finns en nästa sida |  |
| `hasPreviousPage` | Boolean! | Om det finns en föregående sida |  |

## Obligatoriska Behörigheter

### Taggförvaltning

| Åtgärd | Obligatorisk Roll |
|--------|------------------|
| Create tags | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |
| Edit tags | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |
| Delete tags | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |
| Apply tags to records | `OWNER`, `ADMIN`, `MEMBER`, or `CLIENT` |  |

### Tagg Synlighet

Taggar är synliga för alla användare som har tillgång till det projekt där taggen skapades. Rollbaserad filtrering kan tillämpas för vissa åtgärder.

## Bästa Praxis

1. **Använd Konsekventa Färger**: Etablera ett färgkodningssystem för ditt team
2. **Beskrivande Titlar**: Använd tydliga, beskrivande taggnamn som teammedlemmar kommer att förstå
3. **Batchoperationer**: Använd `setTodoTags` för att tillämpa flera taggar på en gång
4. **Sök och Filtrera**: Använd sökfunktionen för att hitta befintliga taggar innan du skapar nya
5. **Taggförvaltning**: Granska och rensa regelbundet oanvända taggar
6. **Färgkonventioner**: Överväg att använda standardfärger för vanliga taggtyper (röd för brådskande, grön för slutförd, etc.)

## Avancerade Funktioner

### Standarder för Taggskapande

- Nya taggar som skapas via `setTodoTags` med `tagTitles` standardiseras till färg `#4a9fff`
- Taggar utan angivna färger faller tillbaka till `#00a0d2`
- Taggtitlar trimmas automatiskt av mellanslag

### Automatisering Integration

Taggar kan:
- Automatiskt tillämpas av automatiseringsregler
- Användas som utlösare för automatiseringsarbetsflöden
- Ändras genom automatiseringsåtgärder

### Realtidsuppdateringar

Taggändringar sänds via GraphQL-prenumerationer, vilket möjliggör realtidsuppdateringar i anslutna klienter.

## Viktiga Noteringar

- Att ta bort en tagg tar bort den från alla associerade poster
- Taggar är projektbundna och kan inte delas över projekt
- Färgvärden måste vara giltiga hexkoder (t.ex. "#ff0000", "#00aa00")
- Taggtitlar är skiftlägeskänsliga
- Maximalt 100 taggar kan tillämpas på en enda post
- Taggar stöder fulltextsökning över titlar
