Blue API ger fullständig åtkomst till Blues processhanteringsplattform genom ett modernt [GraphQL](https://graphql.org/) gränssnitt. Betrodd av **17 000+ kunder** som hanterar **miljarder API-förfrågningar årligen**, driver vår API kritiska arbetsflöden med **[99,99% drifttid](/platform/status)**.

## Varför Blue API?

- **100% Funktionsöverensstämmelse** - Allt du kan göra i Blues UI kan du göra via API
- **Företagsredo** - Testad i stor skala med omfattande hastighetsbegränsning och säkerhet
- **Utvecklarvänlig** - GraphQL ger exakt de data du behöver, inget mer
- **Realtidsuppdateringar** - WebSocket-prenumerationer för omedelbar datasynkronisering

## Snabbstart

Redo att göra ditt första API-anrop? Här är hur enkelt det är:

```graphql
query MyProjects {
  projectList(filter: { companyIds: ["your-company-id"] }) {
    items {
      id
      name
      updatedAt
    }
  }
}
```

**[Prova det i vår GraphQL Playground →](https://api.blue.cc/graphql)**
