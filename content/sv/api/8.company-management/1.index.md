---
title: Skapa ett företag
description: Hur man skapar ett nytt företag med Blue API.
---

För att skapa ett nytt företag kan du använda följande mutation:

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

Observera att skapandet av nya företag kan påverka faktureringen av ditt Blue-konto, eftersom varje företag faktureras separat.

Här är alternativen för `createCompany` mutation och deras beskrivningar:

| Alternativ | Typ | Beskrivning |
|------------|-----|-------------|
| name | String | Namnet på det nya företaget. Detta är ett obligatoriskt fält. |  |
| slug | Sträng | En URL-vänlig version av företagsnamnet. Detta används i företagets URL och bör vara unikt. Detta är ett obligatoriskt fält och kommer också att fungera som den unika identifieraren för företaget. |
