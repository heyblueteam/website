---
title: Créer une entreprise
description: Comment créer une nouvelle entreprise en utilisant l'API Blue.
---

Pour créer une nouvelle entreprise, vous pouvez utiliser la mutation suivante :

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

Notez que la création de nouvelles entreprises peut affecter la facturation de votre compte Blue, car chaque entreprise est facturée séparément.

Voici les options pour la mutation `createCompany` et leurs descriptions :

| Option | Type | Description |
|--------|------|-------------|
| name | String | Le nom de la nouvelle entreprise. C'est un champ requis. |  |
| slug | Chaîne | Une version conviviale pour les URL du nom de l'entreprise. Cela est utilisé dans l'URL de l'entreprise et doit être unique. C'est un champ requis et agira également comme l'identifiant unique de l'entreprise. |
