Blue dispose d'une API (Interface de Programmation d'Applications), vous permettant de créer vos propres intégrations si nécessaire.

**Blue a une couverture API de 100 %**. Cela signifie que tout ce que vous pouvez faire dans [n'importe laquelle de nos applications](/docs/start-guide/download-apps), vous pouvez le faire de manière programmatique. Nous avons une API [GraphQL](https://graphql.org/) puissante qui dispose d'un playground complet où vous pouvez tester vos requêtes et qui est entièrement documentée.
