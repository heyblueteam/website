Blue vous permet d'utiliser Google Maps pour la vue de la carte au lieu de l'OpenStreetMap par défaut. Google Maps a souvent des cartes plus détaillées de certaines zones et des données de localisation plus précises. Cependant, cela nécessite une clé API Google Maps qui doit être saisie dans les paramètres de l'entreprise par le propriétaire de l'entreprise.

Lorsque vous passez à Google Maps, cela affecte toutes les vues de carte dans Blue, y compris la carte de l'entreprise, la [vue de la carte dans les projets](/docs/views/map) et le [champ personnalisé de localisation](/docs/custom-fields/location-map).

Pour utiliser Google Maps dans Blue, les API suivantes doivent être activées dans Google Cloud Console :
- [Maps Tiles API](https://console.cloud.google.com/apis/library/tile.googleapis.com) : Ceci est requis pour rendre les tuiles de la carte.
- [Places API](https://console.cloud.google.com/apis/library/places.googleapis.com) : Ceci est requis pour rechercher des lieux et obtenir leurs coordonnées.
- [Geocoding API](https://console.cloud.google.com/apis/library/geocoding-backend.googleapis.com) : Ceci est requis pour obtenir les coordonnées d'un lieu.

Vous devrez vous inscrire pour un [compte Google Cloud](https://console.cloud.google.com/) et créer un projet pour obtenir les clés API en suivant les instructions [ici](https://developers.google.com/maps/documentation/embed/get-api-key).

## Restriction de l'utilisation de la clé API

Pour des raisons de sécurité, il est crucial de restreindre votre clé API Google Maps pour qu'elle ne fonctionne qu'avec des domaines autorisés. Cela empêche l'utilisation non autorisée de votre clé API si elle est exposée. Vous devez restreindre la clé API pour qu'elle fonctionne uniquement avec :

- app.blue.cc - L'application principale de Blue
- forms.blue.cc - Pour les formulaires publics
- Votre domaine personnalisé si vous utilisez [white label](/docs/start-guide/white-label)

Pour restreindre votre clé API :

1. Allez sur la [page des identifiants de Google Cloud Console](https://console.cloud.google.com/apis/credentials)
2. Trouvez votre clé API et cliquez sur "Modifier" (icône de crayon)
3. Sous "Restrictions d'application", sélectionnez "Référents HTTP (sites web)"
4. Ajoutez les référents suivants :
   - `*.app.blue.cc/*`
   - `*.forms.blue.cc/*`
   - `*.yourdomain.com/*` (si vous utilisez white label)
5. Cliquez sur "Enregistrer"
