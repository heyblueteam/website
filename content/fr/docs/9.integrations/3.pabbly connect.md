[Pabbly Connect](https://payments.pabbly.com/api/affurl/RVYZ07kQyUZ0Z1HUKZ1m/zp2GoChM1JcZ0dQBo?target=nMfdbf0I90K3UdJn) vous permet de connecter Blue avec de nombreuses applications populaires comme Google Calendar, Stripe, MailChimp, et plus encore. Cela permet l'automatisation des flux de travail en envoyant des données entre les outils.

<youtube url="https://www.youtube.com/watch?v=IRxYAe5snzY" />

Il existe d'innombrables possibilités d'intégration de Blue en utilisant Pabbly Connect. Voici les étapes clés :

 ## Accédez à votre tableau de bord Pabbly

1. Connectez-vous à votre [tableau de bord Pabbly](https://connect.pabbly.com/workflows)
2. Cliquez sur le bouton "+ Créer un flux de travail".
3. Nommez votre flux de travail.
4. Sélectionnez Blue comme application de destination.

## Connectez-vous à votre compte Blue

1. Générez un [jeton API](/docs/integrations/api) dans votre profil Blue.
2. Dans Pabbly, entrez votre :
- ID de jeton
- Clé secrète
- URL de l'entreprise (par exemple `https://app.blue.cc/company/mycompany`)
- ID de projet (par exemple `myproject`)

::callout
 ---

icon: lightbulb
---

*Copiez votre clé secrète immédiatement, car elle ne sera plus affichée.* 
::

## Créez une intégration de test
1. Sélectionnez une application de déclenchement et un événement (par exemple, nouvel événement Google Calendar).
2. Envoyez une demande de test.
3. Vérifiez Blue pour vous assurer que les données de test ont été reçues dans votre projet spécifié.
