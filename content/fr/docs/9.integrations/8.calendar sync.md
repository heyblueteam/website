---
title: Synchronisation de Calendrier
description: Synchronisez votre calendrier Blue avec des calendriers externes.
---

Blue vous permet de synchroniser les données de calendrier avec des calendriers externes, y compris :

- Google Calendar
- Outlook
- iCal (Apple)

Cela signifie que les événements de calendrier de Blue peuvent être affichés dans votre calendrier personnel ou professionnel.

::callout
 ---

icon: lightbulb
---

 **Remarque sur la synchronisation unidirectionnelle**

Veuillez noter que la synchronisation de calendrier fonctionne dans un sens. Vous pouvez synchroniser Blue avec votre calendrier, mais pas l'inverse avec cette méthode. 
::

Il existe trois types de calendriers dans Blue qui peuvent être synchronisés dans Blue :

- **Calendrier Personnel** : Tous les enregistrements qui ont une date qui vous sont attribués dans tous vos projets.
- **Calendriers de Projet** : Tous les enregistrements qui ont une date dans un projet spécifique.
- **Calendrier d'Entreprise** : Tous les enregistrements avec une date dans tous les projets auxquels vous appartenez.

Un guide visuel de l'endroit où vous pouvez accéder aux trois types de synchronisations de calendrier dans Blue.

En haut à droite de n'importe quel calendrier, vous trouverez un bouton "Synchroniser le Calendrier" :

![Synchronisation de Calendrier 2](/docs/2.png "Calendar Sync 2")

Le bouton de synchronisation de calendrier

Lorsque vous appuyez sur le bouton de synchronisation de calendrier, vous trouverez un lien sur lequel vous pouvez cliquer pour copier.

![Synchronisation de Calendrier 3](/docs/3.png "Calendar Sync 3")

L'URL de synchronisation de calendrier de Blue
Ajoutez un nouveau calendrier à votre calendrier personnel ou professionnel. Ensuite, ajoutez un nouveau calendrier à votre calendrier personnel ou professionnel. Choisissez l'option "Depuis l'URL" et collez l'URL de synchronisation de Blue.

Vous pouvez trouver des instructions spécifiques pour les principales plateformes :

- [Instructions Google Calendar](https://support.google.com/calendar/answer/37100?hl=en&co=GENIE.Platform%3DDesktop)
- [Instructions Outlook](https://support.microsoft.com/en-au/office/import-or-subscribe-to-a-calendar-in-outlook-com-or-outlook-on-the-web-cff1429c-5af6-41ec-a5b4-74f2c278e98c#:~:text=Sign%20into%20Outlook%20on%20the%20web.,-Sign%20in&text=Sign%20in-,At%20the%20bottom%20of%20the%20navigation%20pane%2C%20click%20the%20Calendar,web%20address%20of%20the%20calendar.)
- [Instructions iCal (Apple)](https://support.apple.com/en-gb/guide/calendar/icl1022/mac#:~:text=In%20the%20Calendar%20app%20on,menu%20and%20choose%20a%20color.)

Le calendrier Blue apparaîtra maintenant comme un calendrier disponible à activer. Activez le bouton pour afficher ce calendrier dans l'interface de votre plateforme de calendrier. Maintenant, chaque fois que vous ajoutez, modifiez ou supprimez des événements dans le calendrier de Blue, ces changements se synchroniseront automatiquement avec votre calendrier connecté.

La synchronisation avec des calendriers externes maintient vos événements Blue intégrés directement dans votre flux de travail quotidien et votre emploi du temps sur plusieurs appareils. Cela garantit que vous avez une visibilité complète sur le travail à venir sans changer de contexte.
