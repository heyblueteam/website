## Exportation des données

1. Dans le projet que vous souhaitez exporter, cliquez sur l'icône des paramètres (engrenage)
2. Sélectionnez "Exporter"
3. Cliquez sur "Exporter" pour déclencher le processus d'exportation sur les serveurs de Blue.
4. Une fois terminé, vous recevrez un e-mail avec un lien pour télécharger le fichier CSV exporté

::callout
 ---

icon: lightbulb
---

**Combien de temps faut-il pour recevoir l'e-mail avec les données ?** <br/> <br/>
En général, cela arrivera dans quelques secondes, mais cela peut dépendre du nombre d'autres processus d'exportation en cours chez d'autres clients et de la taille de vos données. Pour les ensembles de données de plus de 100 000 éléments, cela peut prendre quelques minutes.
::

![Exportation CSV 1](/docs/integration/CSVexport/1.png "CSV Export 1")

::callout
 ---

Notez que la fonction d'exportation n'exporte que les données de vos enregistrements. Les fichiers dans l'onglet Fichiers doivent être téléchargés manuellement en masse.
::

## Exportation avec filtres actifs

Si vous avez déjà appliqué des filtres à votre projet, vous pouvez facilement exporter uniquement les enregistrements filtrés en utilisant l'option "Exporter avec filtres actifs". De cette façon, vous n'aurez pas à vous soucier de filtrer manuellement les données exportées, car elles incluront automatiquement uniquement les enregistrements qui correspondent à vos paramètres de filtre actuels.

Voici comment vous pouvez exporter avec des filtres actifs :

1. Appliquez les filtres à votre projet en utilisant les options de filtre disponibles, telles que les responsables, les étiquettes ou les filtres avancés.
2. Une fois que vous avez la vue filtrée que vous souhaitez exporter, allez dans les paramètres du projet et cliquez sur Importer & Exporter.
3. Dans le menu des options d'exportation, sélectionnez "Exporter avec filtres actifs".
4. Une fois votre exportation prête, vous recevrez les données dans votre e-mail.

## Avantages clés

- **Portabilité des données** : Téléchargez des projets hors de Blue lorsque cela est nécessaire pour une sauvegarde ou une analyse externe
- [Déclenchement via API](/docs/integrations/api) : Vous pouvez utiliser des systèmes tiers pour déclencher automatiquement les exportations de Blue régulièrement afin de répondre à vos besoins de sauvegarde de données.
- **Accessibilité pour les parties prenantes** : Partagez facilement les CSV exportés avec des clients, des gestionnaires et d'autres personnes en dehors de Blue.

L'exportation offre une accessibilité et une flexibilité des données pour répondre à divers besoins d'intégration et de reporting potentiels. Combinée avec [l'importation](/docs/integrations/csv-import), elle fournit un contrôle complet sur vos données Blue.
