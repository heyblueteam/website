Zapier est une plateforme d'intégration qui aide à connecter différentes applications web et à automatiser les flux de travail.

**Cas d'utilisation**

Voici quelques-unes des manières les plus populaires dont Blue est intégré à d'autres applications via Zapier :

**_Synchroniser les enregistrements Blue avec Google Sheets_**

Au fur et à mesure que les enregistrements Blue sont créés ou mis à jour, ajoutez-les automatiquement en tant que nouvelles lignes dans une feuille Google pour un reporting centralisé.

**_Envoyer une confirmation de commande à partir des enregistrements Blue_**

Lorsque vous sélectionnez un certain champ personnalisé sur Blue, envoyez automatiquement un e-mail pour confirmer la commande avec votre client.

**_Ajouter des commentaires Blue à Slack_**

Recevez des notifications Blue en temps réel publiées dans les bons canaux Slack afin que votre équipe soit au courant des mises à jour importantes des enregistrements.

## Avantages clés

**Automatisation entre les outils** : Les Zaps créés dans Zapier peuvent automatiquement envoyer et recevoir des données entre Blue et des milliers d'autres applications. Cela élimine le travail manuel de transfert de données entre les systèmes.

**Intégrations en temps réel** : En utilisant les déclencheurs de webhook de Blue, les Zaps réagissent instantanément aux événements dans Blue comme un nouvel enregistrement, un commentaire ajouté, un changement de statut, etc. Cela permet une synchronisation en temps réel et une automatisation des processus.

**Facile à construire et à maintenir** : Les Zaps offrent un moyen convivial et sans code d'intégrer des outils. Aucun travail de développement n'est nécessaire. Et ils continuent de fonctionner sans maintenance continue.

1. Connectez-vous à votre compte Blue dans les paramètres de profil
2. Allez dans les paramètres de profil et cliquez sur "Webhooks"
3. Cliquez sur "Ajouter un Webhook"
4. Donnez un nom au webhook et entrez l'URL du webhook Zapier
5. Choisissez quels événements doivent déclencher le webhook
6. Cliquez sur "Créer un Webhook"

## Blue vers Zapier

1. Connectez-vous à votre compte Zapier
2. Cliquez sur Créer un Zap  
3. Pour la première étape, recherchez et sélectionnez l'application de déclenchement ["Webhooks by Zapier"](https://zapier.com/apps/webhook)
4. Configurez-le pour accepter les charges utiles de votre URL de webhook Blue
5. Pour la deuxième étape, recherchez et sélectionnez l'application de destination pour envoyer des données
6. Mappez les champs de la charge utile JSON aux champs d'entrée de l'application de destination
7. Activez le Zap

Maintenant, lorsque les événements spécifiés se produisent dans Blue, la charge utile du webhook sera envoyée à Zapier, qui transmet automatiquement les données à votre application intégrée selon les paramètres du Zap.

## Zapier vers Blue

Pour pousser des données d'autres applications vers Blue, suivez ces étapes :

**1. Obtenez les identifiants API de Blue**
- Allez dans Paramètres de Blue > Jetons API
- Créez un nouveau jeton ou utilisez des identifiants existants :
  - ID du jeton
  - Secret du jeton
  - ID de l'entreprise
  - ID du projet

**2. Créez un Zap dans Zapier**
1. Configurez votre application de déclenchement (par exemple, Google Sheets, Slack, etc.)
2. Ajoutez "Webhooks by Zapier" comme application d'action
3. Choisissez le type d'événement "Demande personnalisée"

**3. Configurez le Webhook**
- Méthode : POST
- URL : `https://api.blue.cc/graphql` (ou point de terminaison bêta si nécessaire)
- Passer les données : FAUX
- Dé-platir : NON

**4. Définir les en-têtes :**
```json
{
  "Content-Type": "application/json",
  "X-Bloo-Token-ID": "Your_Token_ID",
  "X-Bloo-Token-Secret": "Your_Token_Secret", 
  "X-Bloo-Company-ID": "Your_Company_ID",
  "X-Bloo-Project-ID": "Your_Project_ID"
}
```

**5. Configurer la requête (Exemple de mutation) :**
```json
{
  "query": "mutation CreateTodo($input: CreateTodoInput!) {\n  createTodo(input: $input) {\n    id\n    title\n    customFields {\n      value\n    }\n  }\n}",
  "variables": {
    "input": {
      "todoListId": "your-todo-list-id",
      "title": "New task from Zapier",
      "customFields": [
        {
          "customFieldId": "priority-field-id",
          "value": "High"
        },
        {
          "customFieldId": "due-date-field-id",
          "value": "2024-03-01"
        }
      ]
    }
  }
}
```

**6. Tester et activer**
- Utilisez le terrain de jeu API de Blue pour tester les mutations d'abord
- Mappez les valeurs dynamiques des applications de déclenchement en utilisant les champs de données de Zapier
- Activez le Zap après des tests réussis
