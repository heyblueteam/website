[Make.com](https://www.make.com) est une plateforme d'intégration qui aide à connecter différentes applications web et à automatiser les workflows. Blue est **officiellement intégré** avec Make.com, permettant une automatisation fluide à travers plus de 2 000 applications—aucun codage requis !

![Aperçu des modules Blue Make.com](/docs/blue_make_modules_overview.png "Overview of available Blue modules in Make.com")

## Cas d'utilisation

Voici quelques façons puissantes d'étendre les capacités de Blue avec les intégrations Make.com :
L'intégration de Blue avec Make.com permet une automatisation puissante à travers diverses fonctions commerciales. Pour l'expérience client, vous pouvez connecter Blue avec des outils comme Intercom, HubSpot et Zendesk pour créer des dossiers clients complets, synchroniser les interactions et maintenir une communication en temps réel via Slack et Microsoft Teams. En gestion de projet, Blue s'intègre avec Jira, GitHub, Trello et Monday.com pour synchroniser les tâches, suivre les progrès et automatiser les notifications.

Pour le marketing et les ventes, Blue se connecte avec Mailchimp, Stripe et Calendly pour suivre les campagnes, gérer les paiements et automatiser les workflows de réunion. La gestion documentaire est rationalisée grâce aux intégrations avec Google Drive, Dropbox, OneDrive et SharePoint, tandis que les outils de communication comme Slack, Gmail et WhatsApp sont unifiés grâce au système central de gestion des tâches de Blue. Les capacités d'analyse de données sont améliorées grâce aux connexions avec Google Sheets, Power BI, Airtable et Tableau, et des solutions spécifiques à l'industrie sont disponibles pour les services financiers, le commerce électronique et la gestion d'équipes à distance.

## Connecter et authentifier votre compte Blue

Pour connecter en toute sécurité Blue avec Make.com :

1. Connectez-vous à votre compte Blue et ouvrez votre profil → allez à l'onglet **API**
2. Cliquez sur **Générer un jeton**, fournissez un nom (et une date d'expiration optionnelle)
3. Copiez les deux :
   - **ID de jeton** (`x‑bloo‑token‑id`)
   - **Clé secrète** (`x‑bloo‑token‑secret`)

::callout
 ---
icon: lightbulb
---
**Remarque importante** <br/>
La clé secrète est affichée une seule fois—conservez-la en toute sécurité !
::

5. Dans Make.com, lors de la création d'une nouvelle connexion, entrez :
   - **ID de jeton**
   - **Clé secrète**

## Référence des paramètres du module

### Surveiller les événements
- Événements (optionnel) : Types d'événements à sélection multiple
- ID d'entreprise (optionnel) : IDs ou slugs d'entreprise à sélection multiple
- ID de projet (optionnel) : IDs ou slugs de projet à sélection multiple

### Créer un enregistrement
- ID d'entreprise : ID ou slug d'entreprise
- ID de projet : ID ou slug de projet
- Liste : ID de liste
- Nom de l'enregistrement : Nom de l'enregistrement
- Champ par défaut : Valeur par défaut
- Date unique : Date d'échéance
- Plage de dates : Dates de début et d'échéance
- Assignés : Utilisateurs à sélection multiple
- Étiquettes : Étiquettes à sélection multiple
- Description : Champ de texte, prend en charge HTML
- Listes de contrôle : Tableau de titres de listes de contrôle

**Champs personnalisés pris en charge :**
| Type de champ | Description |
|---------------|-------------|
| Texte sur une ligne | Champ de texte sur une ligne avec validation et limite de caractères optionnelles |
| Texte sur plusieurs lignes | Champ de texte sur plusieurs lignes avec prise en charge du texte enrichi et options de formatage |
| Sélection unique | Menu déroulant avec sélection d'une seule option et options personnalisées |
| Sélection multiple | Menu déroulant avec sélection de plusieurs options et options personnalisées |
| Localisation | Latitude et longitude avec vue carte et recherche d'adresse |
| Case à cocher | Bascule vrai/faux avec valeur par défaut optionnelle et étiquettes personnalisées |
| Pays | Sélecteur de code pays ISO à deux lettres avec noms de pays |
| Devise | Champ numérique avec symbole de devise et prise en charge des taux de change (ex. : USD200, 200USD, KHR4000) |
| Date | Sélecteur de date avec composant horaire optionnel et prise en charge des fuseaux horaires |
| Téléphone | Champ de numéro de téléphone prenant en charge E.164, format international ou code pays + numéro avec validation |
| Email | Champ d'email avec validation de format et vérification de domaine |
| Évaluation par étoiles | Évaluation numérique de 1 à 5 avec étoiles visuelles et étiquettes personnalisées |
| Pourcentage | Champ numérique avec symbole de pourcentage (0-100) et précision décimale |
| URL | Champ d'adresse web avec validation de format et aperçu |
| Nombre | Champ numérique avec décimales optionnelles et affichage d'unité |
| Formule | Champ calculé basé sur d'autres champs avec opérations mathématiques |
| ID unique | Identifiant généré automatiquement avec préfixe et format personnalisés |
| Référence | Lien vers un autre enregistrement avec options de recherche et de filtrage |
| Recherche | Champ dynamique qui extrait des données d'enregistrements référencés avec options de mappage |
| Durée | Durée en heures et minutes avec format d'affichage personnalisé |
