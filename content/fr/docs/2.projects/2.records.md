Les enregistrements dans Blue sont flexibles et peuvent être utilisés à de nombreuses fins, comme le suivi des interactions avec les clients, des ventes, des jalons de projet ou de l'inventaire. Cette adaptabilité vous permet de personnaliser Blue pour répondre aux besoins de votre entreprise, en faisant un outil précieux pour organiser les informations importantes.

## Renommer les Enregistrements

Vous pouvez facilement renommer les enregistrements dans Blue pour répondre à vos besoins. Il vous suffit d'aller dans les Paramètres du Projet et de changer les noms comme vous le souhaitez, par exemple en tâches ou en affectations. Ce changement s'appliquera dans tout le projet.

Voici quelques exemples de noms que vous pouvez utiliser pour les enregistrements :

- Tâche / Tâches
- Affectation / Affectations
- Étape / Étapes
- Problème / Problèmes
- Ticket / Tickets
- Prospect / Prospects
- Affaire / Affaires
- Dossier / Dossiers
- Opportunité / Opportunités
- Client / Clients
- Produit / Produits
- Commande / Commandes

N'oubliez pas d'inclure les formes singulière et plurielle des noms que vous avez choisis. Cela aide le système à afficher le terme correct selon le contexte.

Personnaliser les noms des enregistrements vous permet d'utiliser une terminologie adaptée à votre secteur ou à vos préférences personnelles, rendant la plateforme plus facile et plus intuitive pour votre équipe.

![Records 1](/docs/Record_1.png "Records 1")

Les enregistrements peuvent représenter une large gamme d'entités, telles que :

- Interactions avec les clients
- Enregistrements de ventes
- Jalons de projet
- Articles d'inventaire

Cette flexibilité garantit que Blue peut être adapté pour s'adapter à divers processus d'affaires et besoins opérationnels, en faisant un outil puissant pour organiser et gérer des données critiques.
