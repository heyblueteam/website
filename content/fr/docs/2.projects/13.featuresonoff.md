---
title: Activer/Désactiver des fonctionnalités
description: Les fonctionnalités dans Blue peuvent être activées ou désactivées pour un projet.
---

Les administrateurs de projet peuvent activer ou désactiver des fonctionnalités pour un projet pour tous les utilisateurs. Cela est utile si vous souhaitez limiter les fonctionnalités disponibles pour un projet. Par exemple, tous les projets n'ont pas besoin d'avoir le Wiki activé. Les administrateurs peuvent également réorganiser l'ordre des onglets dans le projet.

  <source src="/product/features-toggle.mp4" type="video/mp4">
</video>

Limiter les fonctionnalités dans un projet peut également être fait au niveau des groupes d'utilisateurs en utilisant [des rôles d'utilisateur personnalisés](/docs/user-management/roles/custom-user-roles).
