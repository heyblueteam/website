Stockez les informations financières dans le champ devise

Lors de l'ajout de valeurs monétaires, le champ formatera le nombre selon les conventions locales et affichera le symbole de devise approprié. Par exemple, 1 000,50 $ pour les dollars américains ou 1.500,25 € pour les euros.

Le champ devise valide les entrées pour s'assurer que seuls les montants de devise valides sont capturés. Une fois ajouté à un projet, le champ stockera les montants dans le type de devise unique désigné. Il ne fournit pas de conversion automatique de devise entre plusieurs dénominations.

Ce champ offre une expérience optimisée pour le suivi des données monétaires associées aux enregistrements. Le formatage automatisé supprime la complexité de la saisie manuelle des valeurs de devise dans les formats locaux, tandis que la validation protège l'intégrité des données.

Les utilisations potentielles du champ devise incluent :

- Capturer les montants des transactions
- Stocker les budgets de projet
- Suivre les totaux de factures
- Enregistrer les montants de paiement
- Calculer les revenus

Avec un support robuste pour les formats de devise mondiaux et une saisie de données simplifiée, le champ personnalisé Devise vous permet de centraliser les informations financières clés dans vos enregistrements dans Blue.
