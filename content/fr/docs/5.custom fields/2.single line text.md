Le champ Texte sur une seule ligne vous permet d'ajouter des chaînes de texte non formatées à vos enregistrements dans Blue. Il n'existe aucune limite de caractères, vous pouvez donc saisir du texte de n'importe quelle longueur dans ce champ. Cependant, le texte est limité à une seule ligne et ne prend pas en charge les options de formatage comme le gras, l'italique ou les liens.

![Single Line Text](/docs/SingleLineText.png "Single Line Text")

Une façon flexible de capturer des données textuelles structurées

Le champ de texte est très flexible, permettant la saisie de lettres et de chiffres, y compris des caractères spéciaux. Il n'y a aucune restriction sur les caractères qui peuvent être saisis. Le système n'applique aucune validation ou masquage au texte saisi par l'utilisateur.

Le champ Texte sur une seule ligne est idéal pour capturer toutes sortes d'informations, notamment des noms, des descriptions, des numéros de modèle, des SKU, des adresses, des notes, des ID, et plus encore. Vous pouvez utiliser ce champ flexible pour stocker des noms de clients, fournisseurs ou contacts, indiquer des codes d'inventaire et des identifiants, prendre des notes courtes générales sur les enregistrements, ou stocker des ID uniques pour les relations et recherches de base de données. Le champ de texte sur une seule ligne vous permet de saisir efficacement toute information textuelle importante liée à vos enregistrements dans Blue.

Vous pouvez exploiter les [capacités de filtrage avancées des enregistrements](/fr/docs/views/filters) de Blue pour trouver et filtrer les enregistrements contenant du texte spécifique dans le champ Texte sur une seule ligne afin de localiser les enregistrements correspondants.

Avec sa conception simple sur une seule ligne, sa capacité de texte illimitée et son intégration avec le filtrage, le champ Texte sur une seule ligne offre une expérience optimisée pour capturer des données non structurées. Il fournit la flexibilité nécessaire pour stocker toute information selon les besoins par enregistrement.
