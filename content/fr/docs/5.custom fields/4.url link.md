Le champ personnalisé URL/Lien vous permet de stocker des liens et des adresses web dans vos enregistrements dans Blue. Lorsqu'une URL est saisie dans ce champ, le système validera qu'il s'agit d'un lien correctement structuré.

![URL-Link 1](/docs/URLlink_1.png "URL-Link 1")

Le champ personnalisé URL/Lien dans Blue

Chaque enregistrement peut contenir un lien, stocké dans un seul champ personnalisé URL/Lien. Le lien est cliquable, vous permettant de naviguer directement vers la page web ou la ressource en cliquant dessus. Pour plus de commodité, il s'ouvrira automatiquement dans un nouvel onglet du navigateur.

![URL-Link 2](/docs/URLlink_2.png "URL-Link 2")

Vous pouvez ouvrir les URLs dans un nouvel onglet en un clic

Ce champ est extrêmement utile pour créer des liens vers des documents basés sur le cloud comme Google Docs, des fichiers Word, des feuilles de calcul Excel, etc. Stocker ces liens permet un accès en un clic aux fichiers pertinents associés à chaque enregistrement.

Le champ URL/Lien est parfait pour des cas d'usage supplémentaires tels que :

- Stocker les sites web de clients ou partenaires
- Capturer les liens associés aux enregistrements
- Créer des références vers des pages web

Avec son interface simplifiée et ses capacités de lien intégrées, le champ personnalisé URL/Lien vous permet d'enrichir les enregistrements avec des ressources web contextuelles et d'y accéder instantanément d'un clic.
