Les cartes statistiques affichent une valeur calculée en temps réel provenant d'un ou plusieurs projets. C'est idéal pour garder un œil sur les statistiques clés.

## Création de Cartes Statistiques

### Définir le Titre

Lors de la création d'une carte statistique, la première chose à faire est de définir un titre. Donner un titre à vos cartes statistiques est une étape essentielle pour la clarté et l'utilisabilité. Le titre doit encapsuler la métrique ou l'insight critique que la carte statistique représente, comme "Ventes Mensuelles", "Problèmes Ouverts" ou "Utilisateurs Actifs Hebdomadaires".

![Carte Statistique 2](/docs/dashboard/statCard/StatCard_2.png "Stat Card 2")

Un titre bien défini guide les utilisateurs d'un coup d'œil, leur permettant de comprendre rapidement le contexte et la pertinence des données. Le titre que vous définissez sera ensuite affiché en haut des cartes statistiques :

![Carte Statistique 3](/docs/dashboard/statCard/StatCard_3.png "Stat Card 3")

## Créer des Valeurs

Vous pouvez créer plusieurs valeurs par carte statistique, chacune représentant une dimension ou une métrique importante pour votre projet. Cette fonctionnalité multi-valeurs permet d'avoir un tableau de bord complet qui reflète la performance du projet en temps réel.

Les valeurs sur les cartes statistiques peuvent être dérivées d'un projet spécifique, vous permettant de faire ce qui suit :
- **Comptes d'Enregistrements** : Calculez le nombre total d'enregistrements dans un projet, vous donnant un aperçu rapide du volume de tâches ou d'entrées.
- **Calculs de Champs Personnalisés** : Effectuez des calculs basés sur les données contenues dans des champs personnalisés. Par exemple, vous pourriez additionner les valeurs d'un champ "Prix du Projet" pour évaluer les chiffres budgétaires globaux.

Pour garantir que les valeurs sur vos cartes statistiques soient aussi pertinentes et ciblées que possible, Blue offre de robustes options de filtrage — [tout filtre](/docs/views/filters) dans n'importe quelle vue de projet est disponible à l'utilisation !

## Ajouter de la Logique

Dans Blue, la fonctionnalité de logique au sein des cartes statistiques est le moteur pour calculer et présenter des insights basés sur les données. La logique vous permet d'exploiter la puissance des données grâce à des formules et opérations personnalisées, transformant les données brutes du projet en informations exploitables.

Commencez par référencer des valeurs spécifiques dans votre projet en tapant '{'. Cette action fera apparaître une liste de valeurs disponibles que vous pouvez inclure dans votre formule logique.

Vous pouvez améliorer votre logique avec diverses fonctions en tapant '='. Blue fournit une suite de fonctions telles que :

- **MOYENNE** : Calculez la moyenne d'un ensemble de nombres.
- **MOYENNEA** : Déterminez la moyenne, en considérant le texte comme zéro.
- **COMPTE** : Comptez le nombre d'entrées dans un champ numérique.
- **COMPTEA** : Comptez le nombre de valeurs non vides.
- **MAX** : Trouvez la valeur maximale dans un ensemble.
- **MIN** : Identifiez la valeur minimale dans un ensemble.
- **MOINS** : Soustrayez une valeur d'une autre.
- **MULTIPLIER** : Multipliez deux valeurs.
- **SOMME** : Additionnez une série de nombres.

Ces fonctions sont des outils essentiels dans l'analyse des données, vous permettant de résumer et d'analyser efficacement les métriques du projet. ## Configurer l'Affichage

La dernière étape pour créer une carte statistique consiste à décider comment afficher les données.

Les cartes statistiques proposent trois options d'affichage :

1. **Nombre** : Optez pour un affichage numérique simple, avec la fonctionnalité supplémentaire de définir le niveau de précision. Cela signifie que vous pouvez dicter combien de décimales sont affichées, permettant une précision qui correspond au niveau de détail requis pour votre analyse de données.
2. **Monnaie** : Sélectionnez cette option pour représenter les valeurs en termes monétaires. Elle est idéale pour les métriques financières telles que le suivi des budgets, les revenus, les coûts ou d'autres statistiques financières. Le format de monnaie appliquera le formatage local approprié et les symboles monétaires.
3. **Pourcentage** : Utilisez ce format pour exprimer les valeurs en pourcentages, parfait pour montrer les taux d'achèvement, les métriques de croissance ou d'autres données proportionnelles. Cette option convertit automatiquement la valeur calculée en format pourcentage, fournissant une compréhension instantanée des ratios et des parts.
