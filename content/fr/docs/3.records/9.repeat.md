Vous pouvez utiliser les enregistrements répétitifs dans Blue pour automatiser les tâches répétitives afin de ne pas avoir à recréer manuellement le même enregistrement à chaque fois. Ceci est utile pour les travaux planifiés tels que les contrôles de maintenance ou les travaux récurrents. Les enregistrements récurrents suivent la même structure en transférant automatiquement les listes de contrôle et les données des champs personnalisés.


## Création d'enregistrements répétitifs

Pour configurer un enregistrement répétitif, ouvrez l'enregistrement que vous souhaitez répéter et cliquez sur le menu ••• puis sur répéter :

![Repeat 1](/docs/Repeat_1.png "Repeat 1")


Les options suivantes vous sont alors présentées :

![Repeat 2](/docs/Repeat_2.png "Repeat 2")


## Intervalles de répétition

Vous pouvez choisir parmi :
- Quotidien
- Hebdomadaire
- Mensuel
- Annuel

Ou créez un calendrier de répétition personnalisé qui vous offre beaucoup plus de flexibilité et d'options de personnalisation :

![Repeat 3](/docs/Repeat_3.png "Repeat 3")

Ici, vous pouvez choisir :

- **À partir du :** Choisissez la date précise à partir de laquelle l'enregistrement doit commencer à se répéter.
- **Répéter tous les :** Choisissez si l'enregistrement doit se répéter quotidiennement, hebdomadairement, mensuellement ou annuellement.
- **Condition de fin :** Choisissez si l'enregistrement doit se répéter indéfiniment, ou se terminer après une certaine date ou un certain nombre de répétitions.

## Options de copie

Lorsqu'une tâche répétitive est recréée, vous pouvez choisir quelles données sont copiées de la tâche originale :

- Assignés
- Étiquettes
- Champs personnalisés
- Listes de contrôle
- Commentaires
- Description

![Repeat 4](/docs/Repeat_4.png "Repeat 4")


Vous pouvez également choisir dans quelle liste l'enregistrement doit se répéter. Actuellement, nous ne prenons en charge que la répétition d'enregistrements au sein du même projet, pas entre les projets.

::callout
---

icon: lightbulb
---

Sachez que si vous supprimez l'enregistrement original, cela empêchera la création de nouveaux enregistrements avec l'option de répétition.
::

## Enregistrements répétitifs actifs

Pour tous les enregistrements qui se répètent, ceux-ci afficheront le symbole de répétition sur la carte d'enregistrement dans la [Vue tableau](/fr/docs/views/board) :

![Repeat 5](/docs/Repeat_5.png "Repeat 5")

Et également dans l'enregistrement individuel :

![Repeat 6](/docs/Repeat_6.png "Repeat 6")


## Arrêter la répétition

Pour arrêter la répétition d'un enregistrement :
1. Ouvrez l'enregistrement récurrent original
2. Cliquez sur l'icône de répétition
3. Sélectionnez l'option « Arrêter la répétition »

![Repeat 7](/docs/Repeat_7.png "Repeat 7")


Cela empêchera la création de toute autre tâche récurrente.
